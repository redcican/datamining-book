# 10.4 图分类与聚类方法

前几节分别讨论了图的表示（§10.1）、节点特征（§10.2）和社区结构（§10.3）。本节将图挖掘与经典的**分类和聚类**任务（§3.1–§3.7，§5.1–§5.7）结合，回答两类问题：

1. **节点分类**：在一个大图中，已知少量节点的标签，如何预测其余节点的标签？例如，社交网络中已知部分用户的兴趣，预测其他用户的兴趣。
2. **图分类**：给定一组图（如分子结构），如何将它们分为不同类别？例如，判断一个化合物是否具有药理活性。

这两个问题的共同特征是**利用图的拓扑信息**来辅助分类——与传统分类方法仅使用节点/样本的特征向量不同，图分类方法还利用了**节点之间的关系**。

## 10.4.1 半监督节点分类

### 10.4.1.1 问题设定

{% block type="definition" title="定义 10.18　半监督节点分类" %}
给定图 $$G = (V, E)$$，节点集 $$V = V_L \cup V_U$$（$$V_L$$ 为已标注节点，$$V_U$$ 为未标注节点，$$|V_L| \ll |V_U|$$）。已标注节点的标签为 $$y_i \in \{1, \ldots, K\}$$（$$v_i \in V_L$$）。

**半监督节点分类**的目标是利用图的拓扑结构和已标注节点的标签，预测所有未标注节点的标签。其核心假设是**同质性**（Homophily）：

$$
\text{相邻节点倾向于具有相同的标签：} \quad (v_i, v_j) \in E \implies P(y_i = y_j) \text{ 较高} \tag{10.41}
$$

该假设在社交网络（朋友有相似兴趣）、引文网络（相关论文属于同领域）等场景中广泛成立。
{% endblock %}

### 10.4.1.2 图正则化框架

同质性假设可以形式化为一个优化问题——预测标签应同时满足两个目标：(1) 在已标注节点上与真实标签一致；(2) 在相邻节点上尽可能平滑。

设 $$\mathbf{f} \in \mathbb{R}^{n \times K}$$ 为所有节点的标签预测（soft label），则**图正则化损失**为：

$$
\mathcal{L}(\mathbf{f}) = \underbrace{\sum_{i \in V_L} \|f_i - y_i\|^2}_{\text{监督项}} + \underbrace{\alpha \sum_{(v_i, v_j) \in E} A_{ij} \|f_i - f_j\|^2}_{\text{图平滑项}} \tag{10.42}
$$

其中 $$\alpha > 0$$ 控制平滑程度。图平滑项正是 §10.1 定理 10.1 中的拉普拉斯二次型 $$\mathbf{f}^T \mathbf{L} \mathbf{f}$$。

利用拉普拉斯矩阵，方程 (10.42) 可以写成矩阵形式：

$$
\mathcal{L}(\mathbf{f}) = (\mathbf{f} - \mathbf{y})^T \mathbf{S} (\mathbf{f} - \mathbf{y}) + \alpha \, \mathbf{f}^T \mathbf{L} \mathbf{f} \tag{10.43}
$$

其中 $$\mathbf{S}$$ 为对角选择矩阵（$$S_{ii} = 1$$ 若 $$v_i \in V_L$$，否则为 0）。对 $$\mathbf{f}$$ 求导并令其为零，得到闭式解：

$$
\mathbf{f}^* = (\mathbf{S} + \alpha \mathbf{L})^{-1} \mathbf{S} \mathbf{y} \tag{10.44}
$$

### 10.4.1.3 标签传播算法

闭式解需要矩阵求逆，对大规模图不实际。**标签传播**（Label Propagation）提供了高效的迭代替代方案。

{% block type="theorem" title="定理 10.4　标签传播的收敛性" %}
设归一化邻接矩阵 $$\mathbf{P} = \mathbf{D}^{-1}\mathbf{A}$$（随机游走转移矩阵），标签传播迭代：

$$
\mathbf{f}^{(t+1)} = \beta \, \mathbf{P} \mathbf{f}^{(t)} + (1 - \beta) \, \mathbf{y}_0 \tag{10.45}
$$

其中 $$\beta \in (0, 1)$$ 为传播系数，$$\mathbf{y}_0$$ 为初始标签（已标注节点的真实标签，未标注节点为零向量）。则：

1. 迭代序列 $$\{\mathbf{f}^{(t)}\}$$ 收敛到唯一的不动点 $$\mathbf{f}^* = (1 - \beta)(\mathbf{I} - \beta \mathbf{P})^{-1} \mathbf{y}_0$$；
2. 收敛速率为 $$O(\beta^t)$$。

**分析**　由于 $$\mathbf{P}$$ 的谱半径为 1（它是随机矩阵），且 $$\beta < 1$$，矩阵 $$\beta \mathbf{P}$$ 的谱半径 $$< 1$$，因此 $$(\mathbf{I} - \beta \mathbf{P})$$ 可逆，Neumann 级数 $$\sum_{t=0}^{\infty} (\beta \mathbf{P})^t$$ 收敛。收敛速率由 $$\beta \lambda_2(\mathbf{P})$$ 决定，其中 $$\lambda_2$$ 是 $$\mathbf{P}$$ 的次大特征值。$$\blacksquare$$
{% endblock %}

{% block type="algorithm" title="算法 10.5　标签传播节点分类" %}
**输入**：图 $$G = (V, E)$$，已标注节点 $$V_L$$ 及其标签 $$\{y_i\}_{i \in V_L}$$，传播系数 $$\beta = 0.8$$，收敛阈值 $$\varepsilon = 10^{-4}$$

**输出**：所有节点的预测标签

**步骤 1（初始化）**：$$f_i^{(0)} \leftarrow y_i$$（$$v_i \in V_L$$），$$f_i^{(0)} \leftarrow 0$$（$$v_i \in V_U$$）

**步骤 2（迭代传播）**：
$$\quad$$ **repeat**
$$\quad\quad$$ **for each** $$v_i \in V$$ **do**
$$\quad\quad\quad$$ $$f_i^{(t+1)} \leftarrow \beta \sum_{j \in N(i)} \frac{A_{ij}}{d(v_j)} f_j^{(t)} + (1 - \beta) \, y_{0,i}$$
$$\quad$$ **until** $$\|\mathbf{f}^{(t+1)} - \mathbf{f}^{(t)}\| < \varepsilon$$

**步骤 3（标签赋值）**：$$\hat{y}_i \leftarrow \arg\max_k f_{ik}^*$$

**复杂度**：每次迭代 $$O(m)$$，通常 10–50 次迭代收敛。
{% endblock %}

{% hint style="warning" %}
⚠️ **易错点：标签传播的同质性假设**

标签传播的前提是**同质性**——相邻节点有相似标签。在**异质性网络**（Heterophily）中（如约会网络中男女交替、对抗网络中敌对阵营相邻），这个假设不成立，标签传播的效果会急剧下降。对于此类网络，需要使用能够捕获高阶邻域模式的方法（如 §10.7 中的图神经网络）。
{% endhint %}

## 10.4.2 图分类

### 10.4.2.1 问题设定与特征提取

{% block type="definition" title="定义 10.19　图分类" %}
给定训练集 $$\{(G_1, y_1), (G_2, y_2), \ldots, (G_N, y_N)\}$$，其中 $$G_i$$ 为图、$$y_i$$ 为类别标签，**图分类**的目标是学习映射 $$f: \mathcal{G} \rightarrow \mathcal{Y}$$，使得对新图 $$G_{\text{new}}$$ 能正确预测其标签。

图分类的核心挑战是：不同图的**节点数和边数不同**，无法直接用固定长度的特征向量表示。
{% endblock %}

{% block type="definition" title="定义 10.20　图级别特征提取" %}
将图转化为固定长度特征向量的常用策略包括：

**统计特征向量**：将图的统计量组合为特征向量：

$$
\boldsymbol{\phi}(G) = \left[ n, m, \bar{d}, \bar{C}, \bar{L}, \rho, \lambda_2, \ldots \right]^T \tag{10.46}
$$

其中包含节点数、边数、平均度、平均聚集系数、平均路径长度、密度、Fiedler 值等（§10.1–§10.2 定义的各种指标）。

**图核方法**：使用图核（§10.2 定义 10.13）隐式计算图对之间的相似度，结合核 SVM 分类：

$$
\hat{y} = \text{sign}\left( \sum_{i=1}^{N} \alpha_i y_i K(G_i, G_{\text{new}}) + b \right) \tag{10.47}
$$

其中 $$K(G_i, G_j)$$ 为图核函数（如 WL 子树核、随机游走核）。

**图嵌入方法**：通过学习将图映射为低维向量（如 Graph2Vec），然后用标准分类器。
{% endblock %}

图分类的核心挑战在于：不同图的节点数和边数各异，无法直接输入传统分类器。上述三种方法——统计特征、图核和图嵌入——提供了不同的解决思路，它们在**可解释性、表达力和计算效率**之间做出了不同的权衡。统计特征最为简单直观，适合结构差异明显的图；图核通过隐式特征空间捕获更精细的子结构信息，但核矩阵的计算开销与图数量的平方成正比；基于学习的方法（Graph2Vec 和 §10.7 中的 GNN）具有最强的表达力，但依赖大量训练数据。下表系统比较了四类方法的关键特性。

| 方法 | 特征维度 | 是否需要训练 | 优势 | 局限 |
|------|---------|-------------|------|------|
| 统计特征 | 固定（手工设计） | 否 | 可解释、快速 | 丢失结构信息 |
| 图核 + SVM | 核矩阵 $$N \times N$$ | 是 | 捕获子结构 | 核矩阵计算 $$O(N^2)$$ |
| Graph2Vec | 低维嵌入 | 是 | 端到端学习 | 需要大量训练数据 |
| GNN（§10.7） | 自动学习 | 是 | 最强表达力 | 计算量大 |

## 10.4.3 图聚类

{% block type="definition" title="定义 10.21　图聚类" %}
给定一组图 $$\{G_1, G_2, \ldots, G_N\}$$（无标签），**图聚类**的目标是将它们分为 $$k$$ 组，使得同组内的图结构相似、不同组的图结构差异大。

图聚类需要定义**图距离**或**图相似度**。常用方法包括：

1. **基于特征的距离**：先提取统计特征 $$\boldsymbol{\phi}(G)$$，再计算欧氏距离或余弦距离；
2. **基于核的距离**：利用图核 $$K(G_i, G_j)$$（§10.2 定义 10.13）定义相似度矩阵，然后用谱聚类；
3. **基于编辑距离**：图编辑距离（Graph Edit Distance）计算将一个图变换为另一个图所需的最少操作（增删节点/边），但计算是 NP-hard 的。
{% endblock %}

{% hint style="info" %}
🤔 **思考：节点分类 vs 图分类——两种不同的"图+标签"任务**

节点分类和图分类虽然名称相似，但问题设定截然不同：(1) 节点分类在**一个图**上进行——利用图结构传播少量节点标签到整个图，是**半监督学习**问题；(2) 图分类在**多个图**上进行——每个图是一个样本，需要提取图级别特征，是**标准监督学习**问题。前者利用的是"同质性"（邻居有相同标签），后者利用的是"结构相似性"（相似结构的图有相同标签）。
{% endhint %}

## 10.4.4 案例分析

{% block type="case" title="案例 10.4　分子图分类" %}

**问题背景**：在药物发现中，化合物可以表示为分子图——原子是节点，化学键是边。判断化合物是否具有某种生物活性（如抗菌、抗癌）是图分类的典型应用。我们使用合成的分子图数据演示图分类流程。

**数据说明**：生成三类结构不同的图（每类 50 个）：(1) 环形图（模拟环状化合物）；(2) 树状图（模拟支链结构）；(3) 随机图（模拟复杂结构），共 150 个图。

**方法与步骤**：

```python
# --- 1. 生成三类图 ---
import networkx as nx
import numpy as np
np.random.seed(42)
graphs, labels = [], []
for _ in range(50):
    n = np.random.randint(10, 20)
    graphs.append(nx.cycle_graph(n))       # 环形
    labels.append(0)
for _ in range(50):
    graphs.append(nx.random_tree(np.random.randint(10, 20)))
    labels.append(1)
for _ in range(50):
    n = np.random.randint(10, 20)
    graphs.append(nx.erdos_renyi_graph(n, 0.3, seed=None))
    labels.append(2)
# --- 2. 提取统计特征 ---
def graph_features(G):
    n = G.number_of_nodes()
    m = G.number_of_edges()
    if n == 0: return np.zeros(6)
    degrees = [d for _, d in G.degree()]
    cc = nx.average_clustering(G)
    density = nx.density(G)
    components = nx.number_connected_components(G)
    return np.array([n, m, np.mean(degrees), np.std(degrees), cc, density])
X = np.array([graph_features(G) for G in graphs])
y = np.array(labels)
# --- 3. 分类 ---
from sklearn.model_selection import cross_val_score
from sklearn.ensemble import RandomForestClassifier
clf = RandomForestClassifier(n_estimators=100, random_state=42)
scores = cross_val_score(clf, X, y, cv=5)
print(f"5-fold CV accuracy: {scores.mean():.4f} ± {scores.std():.4f}")
```

**结果分析**：

1. **特征区分性**：环形图具有高聚集系数（$$C = 0$$，因为环没有三角形）但固定度 $$d = 2$$；树状图连通但无环，密度最低；随机图度分布宽、聚集系数中等。仅凭 6 个统计特征就能有效区分三类图。

2. **分类准确率**：随机森林在 5 折交叉验证上达到 $$95\%$$ 以上的准确率——说明简单的统计特征对结构差异明显的图已经足够。

3. **特征重要性**：平均度和密度是最重要的两个特征——环形图度恒为 2，树状图度接近 2 但结构不同（通过密度区分），随机图度和密度都更高。

4. **局限性**：对结构相似但类别不同的图（如同构图的不同标签），统计特征的区分力不足——此时需要图核或图神经网络（§10.7）来捕获更精细的子结构差异。

{% endblock %}

**D3.** 实现标签传播算法（不使用 sklearn 的 LabelPropagation），在 Cora 引文网络上测试。比较 $$\beta = 0.5, 0.7, 0.9$$ 的效果。

**D4.** 比较 WL 子树核（使用 `grakel` 库，如果可用）和统计特征在分子图数据集上的分类准确率。

### E 级（案例分析）

**E1.** 在 Cora 引文网络上：(1) 使用标签传播进行节点分类（标注比例 5%）；(2) 与 NetworkX 的 `label_propagation_communities` 社区发现结果对比；(3) 分析哪些节点最容易被正确/错误分类，它们在网络中的位置有什么特征？

**E2.** 从 TUDataset（`https://chrsmrrs.github.io/datasets/`）选择一个分子图数据集（如 MUTAG），构建图分类流程：(1) 提取统计特征；(2) 用 5 折交叉验证评估随机森林分类器；(3) 分析哪些特征最重要。

### F 级（开放探讨）

**F1.** **异质性图上的节点分类**：当网络不满足同质性假设时（如相邻节点倾向于有不同标签），标签传播会失败。讨论：(1) 如何检测网络是否具有同质性？(2) 对于异质性网络，有哪些替代方法？(3) 图神经网络（§10.7）如何在同质和异质网络上都有效？

**F2.** **图分类的可解释性**：统计特征提供了一定的可解释性（如"密度高→随机图"），但图核和 GNN 的决策过程不透明。讨论：(1) 如何解释图分类器的决策？(2) 图的哪些子结构对分类最关键（子图重要性、注意力机制）？(3) 可解释图分类在药物发现等高风险领域的价值。

**F3.** **多标签与层级分类**：现实中的节点或图可能同时属于多个类别（如一个蛋白质参与多个生物过程），或类别之间存在层级关系（如"碳水化合物→单糖→葡萄糖"）。讨论标签传播和图分类如何扩展到多标签和层级分类设定。
