# 第七章 异常检测

异常检测是数据挖掘中最具实践价值却也最具挑战性的任务之一：在海量正常数据中识别少数偏离正常模式的异常点。与分类任务不同，异常检测面临极端类不平衡（异常样本通常不足 1%）、异常模式多变且缺乏标注等特殊困难。本章建立统一的异常评分框架，系统介绍五大类方法：§7.1 形式化定义异常的三种类型（点异常、上下文异常、集体异常）和评估指标体系；§7.2 从 Z-score 到核密度估计，介绍基于分布假设的统计方法；§7.3 以 KNN 距离为核心构建全局异常评分；§7.4 引入 LOF 解决密度不均匀场景下的局部异常检测；§7.5 利用聚类结果间接识别异常；§7.6 介绍 Isolation Forest 和 One-Class SVM 两种分类型方法；§7.7 将所有方法融合到金融欺诈检测的多层防御体系中。

## 本章学习目标

1. 掌握异常的形式化定义，区分点异常、上下文异常和集体异常三种类型；
2. 理解基于评分函数的统一异常检测框架及 ROC/PR 曲线评估方法；
3. 掌握 Z-score、Grubbs 检验、Mahalanobis 距离和核密度估计的统计检测方法；
4. 理解 KNN 异常评分的全局策略及其在密度不均匀场景下的局限；
5. 掌握 LOF 的局部可达密度思想，理解其对全局方法的改进机制；
6. 掌握 Isolation Forest 的随机隔离原理和 One-Class SVM 的决策边界构造；
7. 能设计多层异常检测融合方案，理解金融欺诈检测的工程实践。
