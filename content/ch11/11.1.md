# 11.1 深度学习基础

前面各章介绍了从分类、聚类到关联规则的经典数据挖掘方法，这些方法大多依赖人工设计的特征——分析者需要凭借领域知识将原始数据转化为有效的特征表示，然后在此基础上训练浅层模型。这一范式在低维结构化数据上表现优异，但面对图像、语音、自然语言等高维非结构化数据时，特征工程的瓶颈日益突出：不同任务需要截然不同的特征设计，而特征的质量直接决定了模型的性能上限。

**深度学习**（Deep Learning）通过堆叠多层非线性变换，让模型**自动**从原始数据中学习层次化的特征表示——低层提取边缘、纹理等局部模式，中层组合出部件和结构，高层形成语义概念。这种"端到端"的学习方式不仅降低了人工干预的成本，更在图像识别、机器翻译、蛋白质结构预测等任务上取得了突破性成果，彻底改变了数据挖掘的实践方式。

本节作为深度学习的入门，将从最简单的感知机出发，逐步构建多层感知机（MLP）的完整技术栈——激活函数、前向传播、损失函数、反向传播、优化算法和正则化技术，为后续章节（§11.2 卷积神经网络、§11.3 循环神经网络等）奠定理论基础。

## 11.1.1 从浅层学习到深度学习

传统机器学习遵循"特征工程 + 浅层模型"的两阶段范式。以图像分类为例：分析者首先提取 HOG（方向梯度直方图）、SIFT（尺度不变特征变换）等手工特征，再将这些特征向量输入 SVM 或随机森林进行分类。这一流程的核心瓶颈在于**特征提取**——不同任务需要不同的特征设计，且特征的质量直接决定了模型的性能上限。一个识别猫狗的特征集通常无法直接用于医学影像诊断，每换一个任务都需要领域专家从头设计。

深度学习的革命性贡献在于将特征提取与模型训练**统一**为一个端到端的优化问题。模型不再依赖人工设计的特征，而是通过多层非线性变换自动从原始像素、音频波形或文本序列中发现有效的表示。

{% block type="definition" title="定义 11.1　深度学习" %}
**深度学习**（Deep Learning）是一类基于多层非线性变换的表示学习方法。给定输入 $$\mathbf{x} \in \mathbb{R}^d$$，深度模型通过 $$L$$ 层变换逐步构建层次化特征表示：

$$
\mathbf{h}^{(l)} = f^{(l)}\!\big(W^{(l)} \mathbf{h}^{(l-1)} + \mathbf{b}^{(l)}\big), \quad l = 1, 2, \ldots, L \tag{11.1}
$$

其中 $$\mathbf{h}^{(0)} = \mathbf{x}$$ 为输入，$$W^{(l)} \in \mathbb{R}^{n_l \times n_{l-1}}$$ 和 $$\mathbf{b}^{(l)} \in \mathbb{R}^{n_l}$$ 分别为第 $$l$$ 层的**权重矩阵**和**偏置向量**，$$f^{(l)}$$ 为非线性**激活函数**。层数 $$L$$ 称为网络的**深度**（Depth），各层的宽度 $$n_l$$ 组成网络的**架构**（Architecture）。
{% endblock %}

深度学习之所以有效，关键在于**层次化特征的自动构建**。以人脸识别为例：第一层滤波器学习检测边缘和色块；第二层组合边缘形成眼睛、鼻子等部件；第三层将部件组装为完整的人脸模式。每一层在前一层表示的基础上构建更高级的抽象——这种从简单到复杂的特征层次，正是深度学习名称中"深度"的含义。与之对比，浅层模型（如 SVM、逻辑回归）只能学习输入到输出的单层映射，缺乏逐步抽象的能力。

从历史角度看，深度学习的发展经历了三次浪潮。**第一次浪潮**（1958–1969）以 Rosenblatt 的**感知机**（Perceptron）为代表，但 Minsky 和 Papert 证明了单层感知机无法解决非线性问题（如 XOR），导致神经网络研究陷入长达十余年的低谷。**第二次浪潮**（1986–1995）以 Rumelhart 等人提出的**反向传播**算法为标志，多层网络在理论上可以有效训练，但受限于计算能力和数据规模，深层网络往往陷入局部极值而难以优化，加上 SVM 和核方法的兴起，神经网络再度被边缘化。**第三次浪潮**（2006 至今）由 Hinton 的深度信念网络引领序幕，随后 GPU 并行计算的普及、大规模标注数据集的出现和 ReLU 等新型激活函数的应用，使得真正"深层"的网络得以训练——2012 年 AlexNet 在 ImageNet 竞赛中以碾压性优势夺冠，拉开了深度学习席卷工业界和学术界的大幕。

## 11.1.2 感知机与多层感知机

### 11.1.2.1 感知机

深度学习的起点是最简单的神经网络模型——感知机。它模拟生物神经元的工作方式：接收多个输入信号，加权求和后通过阈值判断是否"激活"输出。

{% block type="definition" title="定义 11.2　感知机" %}
**感知机**（Perceptron）是最简单的线性二分类模型，由输入层和单个输出神经元组成。给定输入向量 $$\mathbf{x} = (x_1, x_2, \ldots, x_d)^\top$$，感知机的输出为：

$$
\hat{y} = \operatorname{sign}(\mathbf{w}^\top \mathbf{x} + b) = \operatorname{sign}\!\Big(\sum_{i=1}^{d} w_i x_i + b\Big) \tag{11.2}
$$

其中 $$\mathbf{w} \in \mathbb{R}^d$$ 为权重向量，$$b \in \mathbb{R}$$ 为偏置项，符号函数为：

$$
\operatorname{sign}(z) = \begin{cases} +1, & z \geq 0 \\ -1, & z < 0 \end{cases} \tag{11.3}
$$
{% endblock %}

感知机的几何解释十分直观：权重 $$\mathbf{w}$$ 和偏置 $$b$$ 定义了输入空间中的一个**超平面** $$\mathbf{w}^\top \mathbf{x} + b = 0$$，将空间分为正负两个半空间。感知机的训练目标是找到一个超平面，使所有训练样本被正确分类。其学习规则基于错误驱动更新——对于误分类样本 $$(\mathbf{x}_i, y_i)$$，按以下规则调整参数：

$$
\mathbf{w} \leftarrow \mathbf{w} + \eta \, y_i \, \mathbf{x}_i, \quad b \leftarrow b + \eta \, y_i \tag{11.4}
$$

其中 $$\eta > 0$$ 为学习率。Novikoff 定理保证了当数据线性可分时，感知机学习算法在有限步内必然收敛。这一朴素而优美的结果曾在 20 世纪 50 年代末引发了对神经网络的巨大热情。

{% hint style="warning" %}
⚠️ **感知机的局限性**

感知机只能解决**线性可分**问题。经典反例是 **XOR（异或）**问题：对于输入 $$(0,0) \to 0$$, $$(0,1) \to 1$$, $$(1,0) \to 1$$, $$(1,1) \to 0$$，不存在任何直线能将两类正确分开。Minsky 和 Papert (1969) 对此的严格证明曾导致神经网络研究停滞近二十年。解决方案是引入**隐藏层**，形成多层感知机。
{% endhint %}

### 11.1.2.2 多层感知机

突破线性可分限制的关键在于引入**隐藏层**（Hidden Layer）和**非线性激活函数**，将多个线性变换与非线性变换交替堆叠。直觉上，隐藏层的作用是将原始输入空间"扭曲"到一个新的特征空间，使得原本线性不可分的数据在新空间中变得可分。

{% block type="definition" title="定义 11.3　多层感知机" %}
**多层感知机**（Multi-Layer Perceptron, MLP）是由输入层、一个或多个隐藏层和输出层组成的全连接前馈神经网络。对于 $$L$$ 层 MLP，前向计算过程为：

$$
\mathbf{h}^{(1)} = f\!\big(W^{(1)} \mathbf{x} + \mathbf{b}^{(1)}\big) \tag{11.5}
$$

$$
\mathbf{h}^{(l)} = f\!\big(W^{(l)} \mathbf{h}^{(l-1)} + \mathbf{b}^{(l)}\big), \quad l = 2, \ldots, L-1 \tag{11.6}
$$

$$
\hat{\mathbf{y}} = g\!\big(W^{(L)} \mathbf{h}^{(L-1)} + \mathbf{b}^{(L)}\big) \tag{11.7}
$$

其中 $$f(\cdot)$$ 为隐藏层激活函数（如 ReLU），$$g(\cdot)$$ 为输出层激活函数——回归任务用恒等映射 $$g(z)=z$$，二分类用 Sigmoid，多分类用 Softmax。网络的总参数量为 $$p = \sum_{l=1}^{L} n_l(n_{l-1} + 1)$$，其中 $$n_0 = d$$ 为输入维度。
{% endblock %}

多层感知机的每一层都在前一层表示的基础上进行线性变换和非线性映射，从而逐步将原始输入空间变换为更易于分类的特征空间。以 XOR 问题为例：一个包含 2 个隐藏神经元的两层网络可以首先将四个输入点映射到一个新的二维空间，在该空间中两类变得线性可分，然后输出层的线性分类器即可完成分类。第一个隐藏神经元可以学到"至少一个输入为 1"（近似 OR 函数），第二个学到"两个输入都为 1"（AND 函数），输出层通过 $$\text{OR} - \text{AND}$$ 的组合实现 XOR 逻辑。

## 11.1.3 激活函数

如果去掉激活函数中的非线性，多层网络的复合仍然是线性映射——$$W^{(2)} W^{(1)} \mathbf{x}$$ 等价于单个矩阵 $$W \mathbf{x}$$，层数的增加不会带来表达能力的提升。因此，**非线性激活函数是深度学习的关键组件**，它使得每一层都能引入非线性变换，从而让深层网络拥有远超线性模型的表达能力。

{% block type="definition" title="定义 11.4　激活函数" %}
**激活函数**（Activation Function）$$f: \mathbb{R} \to \mathbb{R}$$ 是作用于神经元输出的非线性映射。常用的激活函数包括：

**Sigmoid 函数**：

$$
\sigma(z) = \frac{1}{1 + e^{-z}} \tag{11.8}
$$

值域为 $$(0, 1)$$，输出可解释为概率，常用于二分类输出层。导数 $$\sigma'(z) = \sigma(z)(1 - \sigma(z))$$，最大值仅为 0.25（当 $$z=0$$ 时），在 $$|z|$$ 较大时趋近于 0，导致**梯度饱和**。

**Tanh 函数**：

$$
\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}} = 2\sigma(2z) - 1 \tag{11.9}
$$

值域为 $$(-1, 1)$$，输出**零中心化**，收敛通常快于 Sigmoid。但同样存在梯度饱和问题，导数最大值为 1.0。

**ReLU 函数**（Rectified Linear Unit）：

$$
\operatorname{ReLU}(z) = \max(0, z) \tag{11.10}
$$

正半轴梯度恒为 1，有效缓解梯度消失；计算仅需一次比较，非常高效；约 50% 的神经元输出为零，引入稀疏激活。但负半轴梯度严格为 0，可能导致神经元永久"死亡"——一旦权重更新使某神经元的输入始终为负，该神经元将永远不再被激活。

**Leaky ReLU 函数**：

$$
\operatorname{LeakyReLU}(z) = \begin{cases} z, & z > 0 \\ \alpha z, & z \leq 0 \end{cases} \quad (\alpha = 0.01) \tag{11.11}
$$

在负半轴保留微小梯度 $$\alpha$$，避免 ReLU 的神经元死亡问题，同时保持了 ReLU 的计算效率。
{% endblock %}

图 11.1.2 直观展示了四种激活函数的曲线及其导数。Sigmoid 和 Tanh 的导数在远离原点时迅速衰减到接近零，这意味着梯度在反向传播过程中会逐层衰减——这就是**梯度消失**（Vanishing Gradient）问题的根源。在 $$L$$ 层网络中，底层的梯度大约按 $$(\max |f'|)^L$$ 的比例衰减：对于 Sigmoid，$$(\frac{1}{4})^L$$ 在层数稍多时就趋近于零。ReLU 的出现正是为了解决这一问题：正半轴的恒定梯度使信号可以无损地传播到深层网络，这也是 ReLU 成为现代深度学习默认激活函数的主要原因。

对于输出层，多分类任务通常使用 **Softmax** 函数将网络输出转化为概率分布：

$$
\operatorname{softmax}(z_j) = \frac{e^{z_j}}{\sum_{k=1}^{K} e^{z_k}}, \quad j = 1, 2, \ldots, K \tag{11.12}
$$

Softmax 保证所有输出为正且和为 1，满足概率分布的要求。它是逻辑回归（§3.2）从二分类到多分类的自然推广——当 $$K=2$$ 时 Softmax 退化为 Sigmoid。实践中为避免数值溢出，通常先减去输入的最大值：$$\operatorname{softmax}(z_j - \max_k z_k)$$，这不改变结果（平移不变性）但确保了数值稳定。

多层感知机的理论能力由以下经典定理保证：

{% block type="theorem" title="定理 11.1　万能近似定理" %}
（Hornik et al., 1989; Cybenko, 1989）设 $$f: \mathbb{R}^d \to \mathbb{R}$$ 为任意连续函数，$$K \subset \mathbb{R}^d$$ 为任意紧集。对任意 $$\epsilon > 0$$，存在一个单隐藏层 MLP：

$$
F(\mathbf{x}) = \sum_{j=1}^{m} \alpha_j \, \sigma\!\big(\mathbf{w}_j^\top \mathbf{x} + b_j\big) \tag{11.13}
$$

使得 $$\sup_{\mathbf{x} \in K} |F(\mathbf{x}) - f(\mathbf{x})| < \epsilon$$，其中 $$\sigma$$ 为任意非常值连续有界激活函数（如 Sigmoid）。
{% endblock %}

**证明思路**：该定理可通过 Stone-Weierstrass 逼近定理或概率论方法证明。直觉上，每个隐藏神经元 $$\sigma(\mathbf{w}_j^\top \mathbf{x} + b_j)$$ 在输入空间中定义了一个平滑的"台阶"，通过足够多台阶的线性组合可以逼近任意连续函数——类似于用足够多的阶梯函数逼近一条曲线。

{% hint style="tip" %}
🤔 **万能近似 ≠ 万能学习**

万能近似定理保证的是**存在性**——理论上存在一个足够宽的单隐藏层网络可以逼近任意函数。但它没有回答两个关键问题：(1) 需要多少个隐藏神经元？理论表明某些函数需要**指数级**宽度的单层网络才能逼近，而多层网络用**多项式**级参数即可；(2) 梯度下降能否找到这组参数？实践中，**深度**网络（多个窄隐藏层）通常比**宽度**网络（单个宽隐藏层）更高效——深度网络可以利用特征的层次结构，用指数级更少的参数达到相同的近似精度。这也是"深度学习"优于"宽度学习"的理论依据之一。
{% endhint %}

## 11.1.4 前向传播与损失函数

### 11.1.4.1 前向传播

训练神经网络的第一步是**前向传播**——将输入数据从输入层逐层传递到输出层，计算模型的预测结果。这一过程如同信号在电路中单向流动，每经过一层都被变换和放大。

{% block type="definition" title="定义 11.5　前向传播" %}
**前向传播**（Forward Propagation）是神经网络从输入到输出的逐层计算过程。对于第 $$l$$ 层：

$$
\mathbf{z}^{(l)} = W^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)} \tag{11.14}
$$

$$
\mathbf{a}^{(l)} = f\!\big(\mathbf{z}^{(l)}\big) \tag{11.15}
$$

其中 $$\mathbf{z}^{(l)}$$ 为第 $$l$$ 层的**加权输入**（Pre-activation），$$\mathbf{a}^{(l)}$$ 为第 $$l$$ 层的**激活输出**（Activation），$$\mathbf{a}^{(0)} = \mathbf{x}$$ 为网络输入。前向传播的总计算复杂度为 $$O\!\big(\sum_{l=1}^{L} n_l \cdot n_{l-1}\big)$$。
{% endblock %}

前向传播实质上是一系列**矩阵乘法**和**逐元素非线性变换**的交替执行。区分 $$\mathbf{z}^{(l)}$$（线性变换后、激活前的中间量）和 $$\mathbf{a}^{(l)}$$（激活后的输出）对于后续推导反向传播至关重要——反向传播需要利用这些中间量计算梯度，因此在前向传播阶段必须将它们缓存下来。

当处理批量数据时，设 $$X \in \mathbb{R}^{n \times d}$$ 为 $$n$$ 个样本的输入矩阵，前向传播的矩阵形式为 $$Z^{(l)} = A^{(l-1)} (W^{(l)})^\top + \mathbf{1}_n (\mathbf{b}^{(l)})^\top$$，$$A^{(l)} = f(Z^{(l)})$$。矩阵化计算充分利用了 GPU 的并行能力——GPU 拥有数千个计算核心，天然适合大规模矩阵乘法，这也是深度学习在 2012 年后爆发的硬件基础之一。

### 11.1.4.2 损失函数

有了前向传播的预测输出 $$\hat{\mathbf{y}}$$，接下来需要度量它与真实标签 $$\mathbf{y}$$ 之间的差距——这就是**损失函数**的作用。损失函数不仅衡量当前模型的好坏，更为参数优化提供了方向：梯度下降沿着损失函数下降最快的方向调整参数。

{% block type="definition" title="定义 11.6　损失函数" %}
**损失函数**（Loss Function）$$\mathcal{L}(\hat{\mathbf{y}}, \mathbf{y})$$ 度量模型预测与真实标签之间的差异。常用的损失函数包括：

**均方误差**（Mean Squared Error, MSE）——适用于回归任务：

$$
\mathcal{L}_{\text{MSE}} = \frac{1}{n} \sum_{i=1}^{n} \|\mathbf{y}_i - \hat{\mathbf{y}}_i\|^2 \tag{11.16}
$$

MSE 对预测误差施加二次惩罚，因此对异常值（大误差样本）特别敏感。

**交叉熵损失**（Cross-Entropy Loss）——适用于分类任务：

$$
\mathcal{L}_{\text{CE}} = -\frac{1}{n} \sum_{i=1}^{n} \sum_{k=1}^{K} y_{ik} \log \hat{y}_{ik} \tag{11.17}
$$

其中 $$y_{ik} \in \{0, 1\}$$ 为 one-hot 编码的真实标签，$$\hat{y}_{ik}$$ 为 Softmax 输出的预测概率。当 $$K=2$$ 时退化为二分类交叉熵 $$\mathcal{L} = -[y \log \hat{y} + (1-y) \log(1-\hat{y})]$$。
{% endblock %}

交叉熵损失有深刻的**信息论含义**。信息论中，两个分布 $$p$$ 和 $$q$$ 的交叉熵 $$H(p, q) = -\sum_k p_k \log q_k$$ 衡量用分布 $$q$$ 编码分布 $$p$$ 的信息所需的平均比特数。当 $$q = p$$ 时交叉熵达到最小值（等于 $$p$$ 本身的熵 $$H(p)$$），因此最小化交叉熵等价于让预测分布 $$\hat{y}$$ 尽可能逼近真实分布 $$y$$。从另一个角度看，$$H(p, q) = H(p) + D_{\text{KL}}(p \| q)$$，其中 $$D_{\text{KL}}$$ 为 KL 散度——最小化交叉熵等价于最小化 KL 散度，即让两个分布尽可能"接近"。

与 MSE 相比，交叉熵在分类任务中的梯度更有利于学习：当预测严重偏离真实标签时（如真实类别的预测概率 $$\hat{y}_{ik} \to 0$$），$$-\log \hat{y}_{ik}$$ 的梯度趋向无穷大，提供了强烈的纠错信号；而 MSE 的梯度仅与误差成正比，纠错力度要弱得多。这就是为什么分类任务几乎总是选择交叉熵而非 MSE 作为损失函数。

## 11.1.5 反向传播算法

前向传播给出了预测值和损失值，接下来的核心问题是：**如何高效地计算损失函数对每个参数的梯度？** 对于拥有百万乃至十亿参数的现代深度网络，逐参数计算数值梯度（有限差分法）的代价是不可接受的。反向传播算法通过巧妙地复用中间计算结果，将梯度计算的复杂度降至与前向传播相同的量级。

{% block type="definition" title="定义 11.7　反向传播" %}
**反向传播**（Backpropagation, BP）是一种基于**链式法则**（Chain Rule）的高效梯度计算方法。定义第 $$l$$ 层的**误差项**（Error Signal）为损失对加权输入的梯度：

$$
\boldsymbol{\delta}^{(l)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(l)}} \tag{11.18}
$$

则输出层的误差项为：

$$
\boldsymbol{\delta}^{(L)} = \frac{\partial \mathcal{L}}{\partial \mathbf{a}^{(L)}} \odot f'\!\big(\mathbf{z}^{(L)}\big) \tag{11.19}
$$

隐藏层的误差项通过**逐层回传**计算：

$$
\boldsymbol{\delta}^{(l)} = \big((W^{(l+1)})^\top \boldsymbol{\delta}^{(l+1)}\big) \odot f'\!\big(\mathbf{z}^{(l)}\big) \tag{11.20}
$$

其中 $$\odot$$ 表示逐元素乘法（Hadamard 积），$$f'(\mathbf{z}^{(l)})$$ 为激活函数的导数。由误差项可直接计算参数梯度：

$$
\frac{\partial \mathcal{L}}{\partial W^{(l)}} = \boldsymbol{\delta}^{(l)} \big(\mathbf{a}^{(l-1)}\big)^\top, \quad \frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(l)}} = \boldsymbol{\delta}^{(l)} \tag{11.21}
$$
{% endblock %}

反向传播的核心思想是**复用中间结果**。观察公式 (11.20)：第 $$l$$ 层的误差项 $$\boldsymbol{\delta}^{(l)}$$ 仅依赖于第 $$l+1$$ 层的误差项 $$\boldsymbol{\delta}^{(l+1)}$$、连接两层的权重 $$W^{(l+1)}$$ 和当前层的激活导数 $$f'(\mathbf{z}^{(l)})$$。因此，从输出层开始逐层向后计算，每层只需一次矩阵-向量乘法和一次逐元素乘法即可得到该层的误差项和参数梯度。这种"先前向、再反向"的两遍扫描策略，是深度学习能够训练超大规模网络的计算基础。

从**计算图**（Computational Graph）的视角看，反向传播等价于在有向无环图上进行拓扑排序的逆序遍历，每个节点根据链式法则将上游梯度乘以局部导数后向下游传递。现代深度学习框架（PyTorch、TensorFlow）正是基于这一抽象实现了**自动微分**（Automatic Differentiation）——用户只需定义前向计算，框架自动构建计算图并完成反向梯度计算。

{% block type="theorem" title="定理 11.2　反向传播的计算复杂度" %}
设网络有 $$L$$ 层，第 $$l$$ 层宽度为 $$n_l$$，总参数量为 $$p = \sum_{l=1}^{L} n_l(n_{l-1} + 1)$$。反向传播一次完整的梯度计算（包含前向传播和反向传播）的时间复杂度为：

$$
T_{\text{BP}} = O\!\Big(\sum_{l=1}^{L} n_l \cdot n_{l-1}\Big) = O(p) \tag{11.22}
$$

即与参数量**线性**相关。而朴素的数值差分法需要 $$O(p)$$ 次前向传播，总复杂度为 $$O(p \cdot \sum_l n_l n_{l-1}) = O(p^2)$$。
{% endblock %}

**证明**：前向传播每层需 $$O(n_l \cdot n_{l-1})$$ 次乘加运算（矩阵-向量乘法），总计 $$O(\sum_l n_l n_{l-1})$$。反向传播每层需要转置矩阵乘法 $$(W^{(l+1)})^\top \boldsymbol{\delta}^{(l+1)}$$ 复杂度 $$O(n_l \cdot n_{l+1})$$ 和梯度外积 $$\boldsymbol{\delta}^{(l)} (\mathbf{a}^{(l-1)})^\top$$ 复杂度 $$O(n_l \cdot n_{l-1})$$，总计仍为 $$O(\sum_l n_l n_{l-1})$$。两步合计 $$O(\sum_l n_l n_{l-1}) = O(p)$$。∎

{% block type="algorithm" title="算法 11.1　反向传播训练算法" %}
**输入**：训练集 $$\{(\mathbf{x}_i, \mathbf{y}_i)\}_{i=1}^{n}$$，网络结构 $$(n_0, n_1, \ldots, n_L)$$，学习率 $$\eta$$，迭代轮数 $$T$$

**输出**：训练好的网络参数 $$\{W^{(l)}, \mathbf{b}^{(l)}\}_{l=1}^{L}$$

```
1.  随机初始化所有权重 W⁽ˡ⁾ 和偏置 b⁽ˡ⁾
2.  FOR epoch = 1 TO T:
3.      FOR 每个训练样本 (x, y):
4.          // ---- 前向传播 ----
5.          a⁽⁰⁾ ← x
6.          FOR l = 1 TO L:
7.              z⁽ˡ⁾ ← W⁽ˡ⁾ a⁽ˡ⁻¹⁾ + b⁽ˡ⁾
8.              a⁽ˡ⁾ ← f(z⁽ˡ⁾)
9.          计算损失 L(a⁽ᴸ⁾, y)
10.         // ---- 反向传播 ----
11.         δ⁽ᴸ⁾ ← ∂L/∂z⁽ᴸ⁾
12.         FOR l = L-1 DOWN TO 1:
13.             δ⁽ˡ⁾ ← (W⁽ˡ⁺¹⁾)ᵀ δ⁽ˡ⁺¹⁾ ⊙ f'(z⁽ˡ⁾)
14.         // ---- 参数更新 ----
15.         FOR l = 1 TO L:
16.             W⁽ˡ⁾ ← W⁽ˡ⁾ - η · δ⁽ˡ⁾ (a⁽ˡ⁻¹⁾)ᵀ
17.             b⁽ˡ⁾ ← b⁽ˡ⁾ - η · δ⁽ˡ⁾
18. RETURN {W⁽ˡ⁾, b⁽ˡ⁾}
```

**复杂度**：每轮训练 $$O(n \cdot p)$$，$$T$$ 轮总计 $$O(T \cdot n \cdot p)$$。
{% endblock %}

{% hint style="warning" %}
⚠️ **梯度消失与梯度爆炸**

反向传播公式 (11.20) 中，误差项逐层乘以权重矩阵和激活导数。当网络较深时，梯度的累积乘积可能失控：

- 若 $$\|W^{(l)}\| < 1$$ 且激活导数 $$|f'| < 1$$（如 Sigmoid 的最大导数为 0.25），梯度将**指数衰减**——**梯度消失**使底层参数几乎不更新，网络无法学习长距离依赖；
- 若 $$\|W^{(l)}\| > 1$$，梯度将**指数增长**——**梯度爆炸**使参数更新过大，训练数值溢出。

应对策略包括：使用 ReLU 激活函数（正半轴导数恒为 1）、残差连接（§11.2）、梯度裁剪（限制梯度范数）、合理的权重初始化（Xavier 初始化使各层方差保持稳定、He 初始化专为 ReLU 设计）。
{% endhint %}

## 11.1.6 优化方法

反向传播计算出梯度后，需要选择合适的**优化算法**来更新参数。最基本的方法是梯度下降，但全批量梯度下降在大数据集上计算代价过高——每更新一次参数都需要遍历全部数据。因此实践中广泛采用随机梯度下降及其改进变体，它们在收敛速度和稳定性之间做出了不同的权衡。

{% block type="definition" title="定义 11.8　随机梯度下降与改进算法" %}
**随机梯度下降**（Stochastic Gradient Descent, SGD）每次使用单个样本或小批量（Mini-batch）的梯度近似全数据集的梯度：

$$
\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta \, \nabla_{\boldsymbol{\theta}} \mathcal{L}(\boldsymbol{\theta}; \mathbf{x}_i, y_i) \tag{11.23}
$$

**动量法**（Momentum）引入速度变量 $$\mathbf{v}$$，积累历史梯度方向以加速收敛并抑制振荡：

$$
\mathbf{v}_t = \gamma \mathbf{v}_{t-1} + \eta \nabla \mathcal{L}_t, \quad \boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \mathbf{v}_t \tag{11.24}
$$

其中 $$\gamma \in [0, 1)$$ 为动量系数（通常取 0.9）。物理直觉是：$$\mathbf{v}_t$$ 相当于参数空间中的"速度"，梯度是"加速度"，$$\gamma$$ 是"摩擦系数"——球在光滑的峡谷中会自然加速滚向谷底。

**Adam**（Adaptive Moment Estimation）结合动量和自适应学习率，同时维护梯度的一阶矩（均值）和二阶矩（未中心化方差）的指数移动平均：

$$
\mathbf{m}_t = \beta_1 \mathbf{m}_{t-1} + (1-\beta_1) \mathbf{g}_t, \quad \mathbf{v}_t = \beta_2 \mathbf{v}_{t-1} + (1-\beta_2) \mathbf{g}_t^2 \tag{11.25}
$$

$$
\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1 - \beta_1^t}, \quad \hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1 - \beta_2^t}, \quad \boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t} + \epsilon} \hat{\mathbf{m}}_t \tag{11.26}
$$

其中 $$\hat{\mathbf{m}}_t$$ 和 $$\hat{\mathbf{v}}_t$$ 是偏差校正估计（消除初始化偏差），默认超参数 $$\beta_1 = 0.9$$, $$\beta_2 = 0.999$$, $$\epsilon = 10^{-8}$$。
{% endblock %}

三种优化器在损失曲面上的行为差异显著。考虑一个窄长的椭圆形等高线（高条件数问题）：SGD 的梯度方向垂直于等高线，在窄方向上来回振荡、在长方向上缓慢前进。Momentum 通过积累历史梯度抑制横向振荡、加速纵向前进——振荡方向的梯度正负交替相消，前进方向的梯度持续累积。Adam 的自适应学习率进一步为每个参数独立调整步长——梯度波动大的方向（振荡方向）自动缩小步长、梯度稳定的方向自动增大步长，在实践中通常能快速找到一个较好的解。

{% hint style="tip" %}
🤔 **SGD 还是 Adam？**

经验规律是：**Adam 收敛快，SGD 泛化好**。Adam 对学习率不太敏感，适合快速原型开发和复杂模型（如 Transformer、GAN）。但在计算机视觉任务中，精心调参的 SGD+Momentum 常常在最终测试精度上优于 Adam，因为 SGD 的梯度噪声有助于逃离尖锐极小值（Sharp Minima），而尖锐极小值通常泛化能力较差。实际研究中常见的策略是：用 Adam 快速预热，待损失趋于平稳后切换到 SGD 进行微调。
{% endhint %}

## 11.1.7 正则化技术

深度网络的参数量通常远超训练样本数——一个隐藏层为 256 维的三层 MLP 处理 784 维输入就有超过 26 万个参数，而 MNIST 训练集仅有 6 万个样本。这种高度过参数化使模型极易过拟合训练数据。正则化技术通过**约束模型复杂度**来缩小训练误差与测试误差之间的差距，提升模型的泛化能力。

**L2 正则化（权重衰减）** 在损失函数中添加参数范数惩罚项：

$$
\mathcal{L}_{\text{reg}} = \mathcal{L} + \frac{\lambda}{2} \sum_{l=1}^{L} \|W^{(l)}\|_F^2 \tag{11.27}
$$

其中 $$\lambda > 0$$ 为正则化强度，$$\|\cdot\|_F$$ 为 Frobenius 范数。从贝叶斯角度看，L2 正则化等价于对权重施加零均值高斯先验 $$W \sim \mathcal{N}(0, \frac{1}{\lambda}I)$$，倾向于将权重推向较小的值，防止任何单个特征主导输出。在梯度更新中，L2 正则化使权重每步额外乘以缩减因子 $$(1 - \eta\lambda)$$，因此也称"权重衰减"（Weight Decay）。

**Dropout** 在训练时随机"丢弃"（置零）一部分神经元的输出：

$$
\tilde{\mathbf{a}}^{(l)} = \mathbf{a}^{(l)} \odot \mathbf{m}, \quad m_j \sim \operatorname{Bernoulli}(p) \tag{11.28}
$$

其中 $$p$$ 为保留概率（通常取 0.5–0.8）。Dropout 的效果等价于同时训练 $$2^{n_l}$$ 个共享参数的子网络，并在推理时做模型平均——这是一种极其高效的集成学习。它强制网络学习冗余表示：每个神经元不能依赖特定的其他神经元协同工作，必须独立地学到有用的特征。推理时不丢弃，但需将激活值乘以 $$p$$ 以保持期望不变（或训练时除以 $$p$$，即"逆 Dropout"，这是 PyTorch 的默认实现）。

**批归一化**（Batch Normalization, BN）对每层的加权输入做归一化，减少**内部协变量偏移**（Internal Covariate Shift）——即训练过程中每层输入分布的持续变化：

$$
\hat{z}_j = \frac{z_j - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}, \quad \tilde{z}_j = \gamma \hat{z}_j + \beta \tag{11.29}
$$

其中 $$\mu_B$$ 和 $$\sigma_B^2$$ 为当前小批量的均值和方差，$$\gamma$$ 和 $$\beta$$ 为可学习的缩放和平移参数——它们使归一化层能够在需要时恢复原始分布。BN 使每层的输入分布更稳定，允许使用更大的学习率（因为不必担心输入分布剧变导致的参数更新过大），显著加速训练收敛。

## 11.1.8 方法对比

从感知机到深度网络，模型的表达能力和训练复杂度随之递增。在选择模型架构和训练策略时，需要在表达能力、计算成本和泛化风险之间做出权衡。下表总结了本节介绍的核心组件及其特性，帮助读者建立深度学习基础模块的整体认知框架。激活函数的选择决定了网络的梯度流动特性，损失函数的选择应匹配任务类型，优化器的选择影响收敛速度和最终性能，而正则化策略则控制着模型复杂度与泛化能力之间的平衡。

| 组件 | 核心思想 | 优势 | 局限 | 典型选择 |
|------|---------|------|------|---------|
| Sigmoid | 输出概率化 | 可解释性强 | 梯度消失、非零中心 | 二分类输出层 |
| Tanh | 零中心化输出 | 收敛快于 Sigmoid | 梯度消失 | RNN 隐藏层 |
| ReLU | 分段线性 | 梯度不消失、计算快 | 神经元死亡 | 默认隐藏层激活 |
| MSE 损失 | 最小二乘 | 直观、处处可导 | 分类梯度弱 | 回归任务 |
| 交叉熵 | 信息论最优 | 分类梯度强、收敛快 | 需概率输出 | 分类任务 |
| SGD | 随机近似 | 实现简单、泛化好 | 收敛慢、振荡 | 基准方法 |
| Momentum | 积累惯性 | 抑制振荡、加速收敛 | 需调动量系数 | 计算机视觉 |
| Adam | 自适应步长 | 快速收敛、超参鲁棒 | 泛化可能弱 | 默认首选 |
| L2 正则化 | 参数范数约束 | 简单有效 | 需调正则化强度 | 所有网络 |
| Dropout | 随机子网络 | 强正则效果 | 训练变慢 | 全连接层 |
| 批归一化 | 分布稳定化 | 加速训练、允许大学习率 | 依赖批大小 | 卷积/全连接层 |

---

{% block type="case" title="案例 11.1　XOR 问题与多层感知机" %}
**背景**：XOR（异或）问题是检验神经网络非线性建模能力的经典基准。感知机无法解决 XOR（§11.1.2.1），而包含一个隐藏层的 MLP 可以完美分类。本案例使用 NumPy 从零实现两层 MLP，演示前向传播、反向传播和梯度下降的完整训练流程。
```python
# --- 1. 准备 XOR 数据 ---
import numpy as np
np.random.seed(42)
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])   # 4 个样本
y = np.array([[0], [1], [1], [0]])                 # XOR 标签
# --- 2. 初始化网络参数（2→4→1） ---
W1 = np.random.randn(2, 4) * 0.5    # 输入层 → 隐藏层
b1 = np.zeros((1, 4))
W2 = np.random.randn(4, 1) * 0.5    # 隐藏层 → 输出层
b2 = np.zeros((1, 1))
lr = 0.5
# --- 3. 定义 Sigmoid 及其导数 ---
sigmoid = lambda z: 1 / (1 + np.exp(-z))
# --- 4. 训练（5000 轮梯度下降） ---
losses = []
for epoch in range(5000):
    # 前向传播
    z1 = X @ W1 + b1;    a1 = sigmoid(z1)
    z2 = a1 @ W2 + b2;   a2 = sigmoid(z2)
    # 交叉熵损失
    loss = -np.mean(y * np.log(a2 + 1e-8)
                    + (1 - y) * np.log(1 - a2 + 1e-8))
    losses.append(loss)
    # 反向传播
    dz2 = a2 - y                               # 输出层误差
    dW2 = a1.T @ dz2 / 4
    db2 = dz2.mean(axis=0, keepdims=True)
    dz1 = (dz2 @ W2.T) * a1 * (1 - a1)        # 隐藏层误差
    dW1 = X.T @ dz1 / 4
    db1 = dz1.mean(axis=0, keepdims=True)
    # 参数更新
    W2 -= lr * dW2;  b2 -= lr * db2
    W1 -= lr * dW1;  b1 -= lr * db1
# --- 5. 验证预测结果 ---
z1 = X @ W1 + b1;  a1 = sigmoid(z1)
z2 = a1 @ W2 + b2; a2 = sigmoid(z2)
preds = (a2 > 0.5).astype(int)
print("预测:", preds.flatten(), " 标签:", y.flatten())
print(f"最终损失: {losses[-1]:.6f}")
```

**运行结果**：

```
预测: [0 1 1 0]  标签: [0 1 1 0]
最终损失: 0.008234
```

**结果分析**：仅含 4 个隐藏神经元的两层 MLP 在约 3000 轮迭代后即可完美解决 XOR 问题，验证了隐藏层非线性变换突破线性可分限制的能力。从隐藏层的激活值可以观察到：两个关键神经元分别近似学到了 OR 函数（"至少一个输入为 1"）和 NAND 函数（"并非两个都为 1"），输出层通过 AND 组合这两个特征实现 XOR = OR AND NAND。这个例子清楚地展示了隐藏层的角色——将原始空间中线性不可分的问题**重新编码**为新空间中线性可分的问题。

## 本节小结

| 关键概念 | 核心要点 |
|---------|---------|
| 深度学习 | 通过多层非线性变换自动学习层次化特征表示，取代人工特征工程 |
| 感知机 | 最简单的线性分类器，无法解决 XOR 等非线性问题 |
| 多层感知机 | 引入隐藏层和非线性激活，突破线性可分限制，理论上可逼近任意连续函数 |
| 激活函数 | ReLU 是现代默认选择——正半轴恒定梯度解决了 Sigmoid/Tanh 的梯度消失问题 |
| 前向传播 | 从输入到输出的逐层矩阵计算，需缓存中间结果供反向传播使用 |
| 损失函数 | 分类任务用交叉熵（梯度强、收敛快），回归任务用 MSE |
| 反向传播 | 基于链式法则的高效梯度计算，复杂度 $$O(p)$$，是深度学习训练的计算基石 |
| 优化器 | Adam 快速收敛适合原型开发，SGD+Momentum 泛化更好适合精细调优 |
| 正则化 | L2 约束参数范数、Dropout 随机丢弃神经元、BN 稳定层输入分布，通常组合使用 |
