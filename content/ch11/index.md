# 第十一章 深度学习在数据挖掘中的应用

> 🚧 本章内容正在编写中。

## 本章概要

本章学习目标：

1. 掌握多层感知机的前向传播、反向传播和常用优化算法；
2. 理解卷积神经网络的局部连接、权值共享与池化机制；
3. 掌握循环神经网络及其变体（LSTM、GRU）处理序列数据的原理；
4. 理解自编码器的表示学习思想与变分自编码器的生成机制；
5. 掌握生成对抗网络的对抗训练框架与常见变体；
6. 了解深度强化学习的基本概念与 DQN 算法；
7. 理解迁移学习与领域适应的核心方法及适用场景。
