# 第二章 数据预处理

数据科学家超过 60% 的时间花在数据预处理上，而非建模。脏数据、缺失值、量纲不一致、异常值等数据质量问题直接制约模型性能的上限——"Garbage in, garbage out" 是数据挖掘领域最深刻的教训。本章系统介绍数据预处理的七大核心技术：§2.1 讨论四类数据质量问题及清洗策略；§2.2 介绍多源数据集成、类型变换与离散化方法；§2.3 从 PCA 降维到分层抽样，讲解数据规约的数学原理；§2.4 系统介绍过滤式、包裹式、嵌入式三类特征选择方法与特征工程实践；§2.5 对比 Min-Max、Z-score、Robust Scaler 等标准化方法；§2.6 深入分析三种缺失机制（MCAR/MAR/MNAR）与对应的处理策略；§2.7 从统计检验到 LOF 和 Isolation Forest，建立系统的异常值检测框架。

## 本章学习目标

1. 识别四类数据质量问题（缺失、噪声、不一致、重复），掌握相应的清洗方法；
2. 理解数据集成中的模式冲突与语义冲突，掌握 Box-Cox 变换和数值离散化技术；
3. 掌握 PCA 的数学原理（方差最大化、特征分解），能选择合适的主成分数量；
4. 区分过滤式、包裹式、嵌入式三类特征选择方法，理解互信息与 Lasso 的原理；
5. 根据数据分布选择合适的标准化方法，理解 Mahalanobis 距离的几何意义；
6. 判断缺失机制并选择对应的处理策略（删除、统计插补、KNN 插补、多重插补）；
7. 掌握 Z-score、Grubbs 检验、LOF 和 Isolation Forest 四种异常值检测方法。
