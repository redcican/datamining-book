# 2.3 数据规约方法

1991 年，MIT 媒体实验室的 Turk 与 Pentland 面对一个困境：一张 $$64 \times 64$$ 像素的灰度人脸图像原始维度高达 $$d = 4096$$，若以像素作为特征直接训练识别模型，距离度量在如此高维空间中将严重退化——任意两张人脸的像素距离趋于相等，模型彻底失去辨别力。他们的突破在于发现，这 4096 维信息并非独立：人脸图像遵循共同的低维几何结构，仅需约 40 个"特征脸"（eigenfaces）主成分，即可保留超过 95% 的信息，识别精度几乎不变（Turk & Pentland, 1991）。这一思想——以紧凑的低维表示替代高维原始数据——正是**数据规约**的核心。

数据规约不仅是计算效率的问题。在实际数据挖掘中，高维特征往往包含大量冗余（不同特征编码相同信息）和噪声（与目标无关的随机变化）。适度的规约能显著提升后续模型的泛化能力，降低过拟合风险。本节介绍三类主要规约策略：**维度规约**（以 PCA 为代表的线性变换）、**数量规约**（采样）和**属性规约**（特征子集选择），覆盖从理论推导到工程落地的完整路径。

{% hint style="info" %}
📋 **本节学习目标**

完成本节学习后，读者应能够：

1. 运用超球体积公式，定量说明维度灾难的数学本质，解释高维空间中距离度量退化的机制；
2. 从方差最大化目标出发，推导主成分分析（PCA）的特征值问题，证明其与协方差矩阵特征分解的等价性；
3. 阐述 PCA 最优线性降维定理（Eckart-Young），解释重建误差等于被丢弃特征值之和；
4. 编写标准 PCA 流程（中心化 → 协方差矩阵 → 特征分解 → 投影），并在真实数据集上手动验证；
5. 辨析简单随机采样、分层采样与系统采样的统计性质，计算分层采样的最优分配方案；
6. 评价数据规约在特征工程 Pipeline 中的正确位置，避免常见的数据泄露陷阱。
{% endhint %}

## 2.3.1 维度灾难：规约的必要性

在讨论具体规约方法之前，有必要先理解高维数据的本质困难。**维度灾难**（curse of dimensionality）是 Bellman（1961）在研究动态规划时首次指出的现象，后被 Keogh & Mueen（2017）系统整理为多个相互关联的反直觉特性。

**超球体积的衰减。** $$d$$ 维单位超球体积 $$V_d$$ 由伽马函数给出：

$$
V_d = \frac{\pi^{d/2}}{\Gamma\left(\dfrac{d}{2}+1\right)}
\tag{2.19}
$$

该值在 $$d \approx 5$$ 附近达到峰值（约 5.26），此后随维度单调递减并趋近于零。例如，$$d = 20$$ 时 $$V_{20} \approx 0.026$$，$$d = 50$$ 时已小至 $$10^{-19}$$ 量级。这意味着：**高维单位超球几乎完全被单位超立方体的角落所占据，球体内部几乎空无一物**。若以均匀分布在单位超立方体中的 $$n$$ 个样本训练 $$k$$-NN 分类器，为使每个样本的 $$\varepsilon$$-邻域平均包含 $$p$$ 个邻居，需要的样本量随维度指数增长：$$n \sim p / V_d(\varepsilon)$$。

**距离集中效应。** 在 $$d$$ 维均匀分布中，任意两点 $$\mathbf{x}, \mathbf{y} \sim \mathcal{U}[0,1]^d$$ 之间欧氏距离的期望值为 $$\mathbb{E}[\|\mathbf{x}-\mathbf{y}\|] \approx \sqrt{d/6}$$，标准差增长仅为 $$O(1/\sqrt{d})$$。因此，**距离的变异系数 $$\mathrm{CV} = \sigma/\mu$$ 随维度增高而趋近于零**：最近邻与最远邻的距离差异越来越小，距离度量逐渐失去辨别力（如图 2.3.1(b) 所示）。

**数据稀疏性。** 在 $$d$$ 维空间中，若要使每个轴向上平均有 5 个训练点覆盖，需要的总样本量为 $$5^d$$。$$d=1$$ 时仅需 5 个，$$d=10$$ 时需近 1000 万个，$$d=20$$ 时超过千亿——绝大多数现实数据集根本无法提供足够样本密度填满高维特征空间。

上述三种现象共同说明：**降低数据维度是高维数据挖掘的前提条件，而非锦上添花的优化步骤**。数据规约方法正是为此目的而设计的。

{% block type="definition" label="2.9" title="数据规约（Data Reduction）" %}
设原始数据集 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$，**数据规约**是指在尽可能保留原始数据信息的前提下，获得数据的简化表示 $$\mathbf{X}' \in \mathbb{R}^{n' \times d'}$$（其中 $$n' \leq n$$ 且/或 $$d' \leq d$$），使得后续挖掘任务的质量不显著下降。

规约的三个主要维度：
- **维度规约（Dimensionality Reduction）**：减少特征数 $$d' < d$$，包括线性变换（PCA）和非线性映射（t-SNE、UMAP）；
- **数量规约（Numerosity Reduction）**：减少样本数 $$n' < n$$，包括采样与数据摘要；
- **属性规约（Attribute Subset Selection）**：从原始 $$d$$ 个特征中选取 $$d'$$ 个子集，不构造新特征。
{% endblock %}

明确了维度灾难的数学根源与规约的三种路径，下一节将重点推导目前应用最广的线性维度规约方法——主成分分析。

## 2.3.2 主成分分析

**主成分分析**（Principal Component Analysis，PCA）是一种经典的无监督线性维度规约技术，其核心思想直观而深刻：**沿方差最大的方向投影，保留最多的信息**。若数据的变化主要集中在少数几个方向上（即协方差矩阵存在快速衰减的特征值谱），PCA 便能以远低于原始维度的子空间精确捕捉这些变化。

{% hint style="think" %}
🤔 **思考**：为什么"方差最大的方向"等价于"信息最多的方向"？若一个特征在所有样本上的值几乎相同（方差极小），它能提供多少判别信息？反之，方差大意味着什么？
{% endhint %}

**问题形式化。** 设 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$ 为已中心化的数据矩阵（每列均值为零），样本协方差矩阵为：

$$
\mathbf{C} = \frac{1}{n}\mathbf{X}^{\top}\mathbf{X} \in \mathbb{R}^{d \times d}
\tag{2.20}
$$

寻找单位方向向量 $$\mathbf{w} \in \mathbb{R}^d$$，使投影后数据 $$\{\mathbf{w}^{\top}\mathbf{x}_i\}_{i=1}^n$$ 的方差最大：

$$
\max_{\mathbf{w}} \frac{1}{n}\sum_{i=1}^n (\mathbf{w}^{\top}\mathbf{x}_i)^2 = \max_{\mathbf{w}} \mathbf{w}^{\top}\mathbf{C}\mathbf{w} \quad \text{s.t. } \|\mathbf{w}\| = 1
\tag{2.21}
$$

**推导。** 构造 Lagrangian 函数：

$$\mathcal{L}(\mathbf{w}, \lambda) = \mathbf{w}^{\top}\mathbf{C}\mathbf{w} - \lambda(\mathbf{w}^{\top}\mathbf{w} - 1)$$

令 $$\partial \mathcal{L}/\partial \mathbf{w} = 2\mathbf{C}\mathbf{w} - 2\lambda\mathbf{w} = \mathbf{0}$$，得到特征值方程：

$$
\mathbf{C}\mathbf{w} = \lambda\mathbf{w}
\tag{2.22}
$$

将此方程代入目标函数：$$\mathbf{w}^{\top}\mathbf{C}\mathbf{w} = \mathbf{w}^{\top}(\lambda\mathbf{w}) = \lambda$$。因此，**最大化投影方差等价于选取协方差矩阵最大特征值对应的特征向量**。依此类推，第 $$k$$ 个主成分方向 $$\mathbf{v}_k$$ 为第 $$k$$ 大特征值 $$\lambda_k$$ 对应的特征向量，且各主成分之间两两正交（因 $$\mathbf{C}$$ 为实对称矩阵，不同特征值对应特征向量正交）。

{% block type="definition" label="2.10" title="主成分分析（Principal Component Analysis）" %}
设 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$ 为已中心化数据矩阵，样本协方差矩阵 $$\mathbf{C} = \frac{1}{n}\mathbf{X}^{\top}\mathbf{X}$$ 的特征分解为：

$$\mathbf{C} = \mathbf{V}\boldsymbol{\Lambda}\mathbf{V}^{\top}, \quad \boldsymbol{\Lambda} = \mathrm{diag}(\lambda_1, \ldots, \lambda_d), \quad \lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_d \geq 0$$

其中 $$\mathbf{V} = [\mathbf{v}_1, \ldots, \mathbf{v}_d] \in \mathbb{R}^{d \times d}$$ 为正交矩阵，$$\mathbf{v}_k$$ 为第 $$k$$ **主成分方向**（principal direction）。

将数据投影到前 $$K$$ 个主成分：$$\mathbf{Z} = \mathbf{X}\mathbf{V}_K \in \mathbb{R}^{n \times K}$$，其中 $$\mathbf{V}_K = [\mathbf{v}_1, \ldots, \mathbf{v}_K]$$。

从 $$K$$ 维表示重建原始数据（低秩近似）：$$\hat{\mathbf{X}} = \mathbf{Z}\mathbf{V}_K^{\top} = \mathbf{X}\mathbf{V}_K\mathbf{V}_K^{\top}$$。
{% endblock %}

PCA 具有严格的最优性保证，这是其广泛应用的数学基础：

{% block type="theorem" label="2.3" title="PCA 最优线性降维定理（Eckart-Young）" %}
在所有将 $$\mathbf{X}$$ 投影到 $$K$$ 维线性子空间的方案中，PCA 的投影矩阵 $$\mathbf{V}_K$$ 使均方重建误差最小：

$$\mathbf{V}_K = \arg\min_{\mathbf{W}: \mathbf{W}^{\top}\mathbf{W}=\mathbf{I}_K} \frac{1}{n}\sum_{i=1}^n \|\mathbf{x}_i - \mathbf{W}\mathbf{W}^{\top}\mathbf{x}_i\|^2$$

最小重建误差等于被丢弃特征值之和：

$$\text{MSE}_{\min} = \sum_{k=K+1}^{d} \lambda_k$$

**证明思路。** 利用 Frobenius 范数的矩阵分解（Eckart-Young, 1936）：$$\|\mathbf{X} - \hat{\mathbf{X}}\|_F^2$$ 在所有秩-$$K$$ 近似中由截断 SVD（即 PCA）最小化。由于数据已中心化，$$\|\mathbf{X}\|_F^2 = n\sum_k\lambda_k$$，丢弃后 $$K$$ 个主成分的误差为 $$n\sum_{k>K}\lambda_k$$，故均方误差为 $$\sum_{k>K}\lambda_k$$。$$\square$$
{% endblock %}

{% hint style="proof" %}
**MSE = 丢弃特征值之和的完整推导。**

设 $$\mathbf{X} = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^{\top}$$ 为 SVD，则 $$\mathbf{C} = \frac{1}{n}\mathbf{V}\boldsymbol{\Sigma}^2\mathbf{V}^{\top}$$，故特征值 $$\lambda_k = \sigma_k^2/n$$（$$\sigma_k$$ 为奇异值）。

低秩近似：$$\hat{\mathbf{X}}_K = \mathbf{U}_K\boldsymbol{\Sigma}_K\mathbf{V}_K^{\top}$$。

误差：$$\|\mathbf{X} - \hat{\mathbf{X}}_K\|_F^2 = \sum_{k=K+1}^{d}\sigma_k^2 = n\sum_{k=K+1}^{d}\lambda_k$$。

均方误差：$$\mathrm{MSE} = \frac{1}{n}\|\mathbf{X} - \hat{\mathbf{X}}_K\|_F^2 = \sum_{k=K+1}^{d}\lambda_k$$。$$\square$$
{% endhint %}

{% block type="definition" label="2.11" title="方差解释比（Explained Variance Ratio）" %}
第 $$k$$ 个主成分的**方差解释比**（explained variance ratio）定义为其特征值占总特征值之比：

$$
\mathrm{EVR}_k = \frac{\lambda_k}{\sum_{j=1}^{d}\lambda_j}
\tag{2.23}
$$

前 $$K$$ 个主成分的**累计方差解释率**：

$$
\mathrm{CEVR}(K) = \frac{\sum_{k=1}^{K}\lambda_k}{\sum_{k=1}^{d}\lambda_k}
\tag{2.24}
$$

实践中通常选取满足 $$\mathrm{CEVR}(K) \geq 0.95$$（保留 95% 方差）的最小 $$K$$ 作为降维目标维度。碎石图（scree plot）通过可视化特征值排序曲线的"拐点"（elbow）辅助选择 $$K$$（见图 2.3.2(c)(d)）。
{% endblock %}

方差解释比为选择主成分数 $$K$$ 提供了量化标准；有了这一准则，PCA 的计算步骤可以整理为一套标准流程。其中**中心化**是不可省略的前提——协方差矩阵 $$\mathbf{C} = \frac{1}{n}\mathbf{X}^{\top}\mathbf{X}$$ 的定义要求数据列均值为零，否则计算结果将混入均值方向的信息，导致主成分方向偏离真实变化方向。

{% block type="algorithm" label="2.3" title="标准 PCA 算法" %}
**输入**：数据矩阵 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$，目标维度 $$K$$（或方差阈值 $$\rho$$）

**输出**：降维后数据 $$\mathbf{Z} \in \mathbb{R}^{n \times K}$$，投影矩阵 $$\mathbf{V}_K \in \mathbb{R}^{d \times K}$$

1. **中心化**（必须）：$$\boldsymbol{\mu} \leftarrow \frac{1}{n}\sum_{i=1}^n \mathbf{x}_i$$；$$\mathbf{X} \leftarrow \mathbf{X} - \mathbf{1}\boldsymbol{\mu}^{\top}$$
2. （可选）**标准化**：若各特征量纲不同，令 $$x_{ij} \leftarrow x_{ij}/\hat{\sigma}_j$$
3. **计算协方差矩阵**：$$\mathbf{C} \leftarrow \frac{1}{n}\mathbf{X}^{\top}\mathbf{X}$$
4. **特征分解**：$$[\mathbf{V}, \boldsymbol{\Lambda}] \leftarrow \mathrm{eigh}(\mathbf{C})$$；按特征值降序排列 $$(\lambda_1 \geq \cdots \geq \lambda_d)$$
5. **选取主成分数**：
   - 若指定 $$K$$：直接取前 $$K$$ 列；
   - 若指定阈值 $$\rho$$：$$K \leftarrow \min\{k : \mathrm{CEVR}(k) \geq \rho\}$$
6. **投影**：$$\mathbf{Z} \leftarrow \mathbf{X}\mathbf{V}_K$$
7. **（可选）重建**：$$\hat{\mathbf{X}} \leftarrow \mathbf{Z}\mathbf{V}_K^{\top} + \mathbf{1}\boldsymbol{\mu}^{\top}$$

**时间复杂度**：特征分解步骤为 $$O(d^3)$$；当 $$n \gg d$$ 时，协方差矩阵计算为 $$O(nd^2)$$；当 $$d \gg n$$ 时，可改用 SVD 分解 $$\mathbf{X}$$，复杂度 $$O(n^2 d)$$。
{% endblock %}

{% block type="example" label="2.6" title="手动 PCA 计算：4 个样本的二维数据" %}
**数据**：4 个观测，2 个特征：

$$\mathbf{X} = \begin{pmatrix}1 & 2 \\ 2 & 2 \\ 3 & 4 \\ 4 & 4\end{pmatrix}$$

**步骤 1 — 中心化。** 均值 $$\boldsymbol{\mu} = (2.5, 3)^{\top}$$：

$$\mathbf{X}_c = \begin{pmatrix}-1.5 & -1 \\ -0.5 & -1 \\ 0.5 & 1 \\ 1.5 & 1\end{pmatrix}$$

**步骤 2 — 协方差矩阵。**

$$\mathbf{C} = \frac{1}{4}\mathbf{X}_c^{\top}\mathbf{X}_c = \begin{pmatrix}1.25 & 1.00 \\ 1.00 & 1.00\end{pmatrix}$$

**步骤 3 — 特征分解。** $$\mathrm{tr}(\mathbf{C}) = 2.25$$，$$\det(\mathbf{C}) = 0.25$$，特征多项式：

$$\lambda^2 - 2.25\lambda + 0.25 = 0 \implies \lambda_1 \approx 2.133,\quad \lambda_2 \approx 0.117$$

**步骤 4 — 主成分方向。** 对 $$\lambda_1 = 2.133$$：$$(\mathbf{C} - \lambda_1\mathbf{I})\mathbf{v} = \mathbf{0}$$ 给出 $$v_2 \approx 0.883 v_1$$，单位化得：

$$\mathbf{v}_1 \approx (0.749,\ 0.662)^{\top}$$

**步骤 5 — 方差解释比。**

$$\mathrm{EVR}_1 = \frac{2.133}{2.133 + 0.117} = \frac{2.133}{2.250} \approx 94.8\%$$

**结论**：仅保留 $$K=1$$ 个主成分，即可保留原始二维数据 94.8% 的方差。投影后的 1 维数据为 $$\mathbf{z}_i = \mathbf{v}_1^{\top}(\mathbf{x}_i - \boldsymbol{\mu})$$，损失的均方误差等于 $$\lambda_2 \approx 0.117$$。
{% endblock %}

{% hint style="caution" %}
⚠️ **必须在 PCA 前标准化**

当各特征量纲不同时（如"收入（元）"与"年龄（岁）"同时存在），协方差矩阵将被方差最大的特征所主导，PCA 结果实际反映的是量纲差异而非真实信息结构。**凡特征量纲不统一，务必在中心化后进行 Z-score 标准化再执行 PCA**。若所有特征量纲一致（如像素值），可只中心化不标准化。
{% endhint %}

{% hint style="caution" %}
⚠️ **PCA 是线性方法，不适用于非线性结构**

PCA 寻找全局线性子空间。若数据分布在非线性流形上（如螺旋线、同心圆、瑞士卷），PCA 无法捕捉其本质结构。此类数据应使用核 PCA（Kernel PCA）、t-SNE 或 UMAP 等非线性方法（见本节前沿视野专栏）。
{% endhint %}

{% hint style="supplement" %}
🔬 **补充：核 PCA（Kernel PCA）**

核 PCA 将原始空间数据 $$\mathbf{x}$$ 通过特征映射 $$\phi: \mathbb{R}^d \to \mathcal{H}$$（$$\mathcal{H}$$ 为高维 Hilbert 空间）隐式映射后，在 $$\mathcal{H}$$ 中执行标准 PCA。借助核技巧：$$K(\mathbf{x}_i, \mathbf{x}_j) = \phi(\mathbf{x}_i)^{\top}\phi(\mathbf{x}_j)$$，无需显式计算 $$\phi$$，仅需计算 $$n \times n$$ 核矩阵 $$\mathbf{K}$$ 的特征分解，时间复杂度为 $$O(n^3)$$。常用核：RBF 核 $$K(\mathbf{x}_i, \mathbf{x}_j) = \exp(-\|\mathbf{x}_i - \mathbf{x}_j\|^2 / 2\sigma^2)$$，可捕捉非线性流形结构。主要限制：$$O(n^2)$$ 存储和 $$O(n^3)$$ 计算使其不适合大规模数据集。
{% endhint %}

{% hint style="tip" %}
💡 **PCA 保留维数 $$K$$ 的实践选择原则**

1. **累计方差阈值**（最常用）：选取满足 CEVR($$K$$) $$\geq$$ 0.95（或 0.90）的最小 $$K$$；
2. **碎石图肘点**：观察特征值排序曲线，选取"拐点"前的成分（图 2.3.2(c)）；
3. **任务导向**：若后续任务为可视化，固定 $$K=2$$ 或 $$K=3$$；若为加速训练，以不显著损失下游精度为准；
4. **重建误差评估**：在验证集上比较 $$K$$ 值对下游模型性能的影响，以性能不再提升时的最小 $$K$$ 为准。
{% endhint %}

维度规约解决了"特征太多"的问题；但当样本量 $$n$$ 本身过大（如数十亿级），或需要从全量数据中提取有代表性的子集时，则需要**数量规约**——采样方法。

## 2.3.3 采样与属性规约

**为何需要采样？** 采样是数量规约的基本手段：从规模为 $$N$$ 的总体中选取 $$n \ll N$$ 个样本，以近似总体统计特性。良好的采样方法应使样本均值 $$\bar{y}$$ 对总体均值 $$\mu$$ 的估计满足指定精度要求。

**简单随机采样**（Simple Random Sampling, SRS）是最基础的策略：每个单元被选中的概率均为 $$n/N$$，无放回地抽取 $$n$$ 个样本。其优点是实现简单、无偏，但当总体包含异质性较强的子群体（层次）时，少数群体可能被系统性低估。

{% block type="definition" label="2.12" title="分层采样（Stratified Sampling）" %}
设总体 $$\mathcal{U} = \{u_1, \ldots, u_N\}$$ 被划分为 $$H$$ 个互不相交的**层**（stratum）$$\mathcal{U}_1, \ldots, \mathcal{U}_H$$，其中 $$|\mathcal{U}_h| = N_h$$，$$\sum_h N_h = N$$。

**分层采样**在每层 $$\mathcal{U}_h$$ 中独立抽取 $$n_h$$ 个样本，总样本量 $$n = \sum_h n_h$$。

**比例分配**（proportional allocation）：$$n_h = n \cdot N_h/N$$，各层采样率相同。

**最优分配**（Neyman allocation）：$$n_h \propto N_h \sigma_h$$，$$\sigma_h$$ 为层内标准差，使总体均值估计的方差最小。

**分层样本均值**：$$\bar{y}_{st} = \sum_{h=1}^H \frac{N_h}{N}\bar{y}_h$$

在层内方差小于总体方差时，分层采样的估计方差严格小于相同样本量的简单随机采样。
{% endblock %}

**系统采样**（Systematic Sampling）：按某排序规则将 $$N$$ 个单元排列，随机选取起点 $$r \in [1, k]$$（$$k = \lfloor N/n \rfloor$$），随后每隔 $$k$$ 个选取一个：$$\{r, r+k, r+2k, \ldots\}$$。实现简单，但若数据存在与步长 $$k$$ 相关的周期性规律，会引入系统误差。

<figcaption>图 2.3.3　三种采样策略对比（总体 N=120，采样率 20%，n=24）。圆圈小点为总体，大实心点为被选入样本的个体，颜色区分层次 A（高价值，N=20）、B（中等，N=70）、C（低频，N=30）。文字框显示各层实际被采样的数量与比例，可见分层采样在代表性上的优势。</figcaption>

{% block type="example" label="2.7" title="分层采样方案设计：大学生满意度调查" %}
**场景**：某高校共 $$N = 4000$$ 名在读生，分三个学院：

| 学院 | 人数 $$N_h$$ | 人口比例 |
|------|-----------|---------|
| 理工学院 | 2400 | 60% |
| 文科学院 | 1200 | 30% |
| 艺术学院 | 400 | 10% |

目标：抽取 $$n = 200$$ 人进行满意度调查，按比例分层采样。

**步骤 1 — 计算各层样本量（比例分配，$$n_1$$=理工、$$n_2$$=文科、$$n_3$$=艺术）：**

$$n_1 = 200 \times \frac{2400}{4000} = 120, \quad n_2 = 200 \times \frac{1200}{4000} = 60, \quad n_3 = 200 \times \frac{400}{4000} = 20$$

**步骤 2 — 验证：** $$n_1 + n_2 + n_3 = 120 + 60 + 20 = 200$$ ✓

**步骤 3 — 计算分层样本均值（设调查得各层均值）：**

设三院满意度均值分别为 $$\bar{y}_1=82$$，$$\bar{y}_2=78$$，$$\bar{y}_3=90$$：

$$\bar{y}_{st} = \frac{2400}{4000}\times 82 + \frac{1200}{4000}\times 78 + \frac{400}{4000}\times 90 = 49.2 + 23.4 + 9.0 = 81.6$$

**对比**：若艺术学院（满意度最高）在 SRS 中碰巧仅被抽到 5 人，则估计偏差显著。分层采样保证该层 20 人被纳入，估计更稳定。
{% endblock %}

**属性规约**与维度规约的区别在于：**属性规约选取原始特征的子集，不构造新特征**；PCA 则生成原始特征的线性组合（主成分），可解释性较差。属性规约方法分三类：

- **过滤法**（Filter）：按特征与目标变量的相关性（互信息、F 检验、卡方检验）独立评分，选取前 $$d'$$ 个，计算效率高，但忽略特征间交互；
- **包裹法**（Wrapper）：以下游模型的性能为目标，递归特征消除（RFE）或正向选择，选特征子集，效果好但计算代价高（$$O(2^d)$$ 最坏情况）；
- **嵌入法**（Embedded）：在模型训练过程中内生地实现特征选择，如 Lasso 回归（L1 正则化使部分系数收缩至零）和决策树的基尼系数特征重要性。

{% hint style="ideology" %}
🌏 **课程思政：采样公平性与算法偏见**

采样设计看似技术问题，实则关乎社会公平。历史上，美国部分地区将少数族裔居住区排除在金融服务的市场调查之外（redlining），导致数据集系统性缺失该群体，训练出的信用评分模型对少数族裔存在歧视性偏差。当前 AI 系统的训练数据若依赖便利采样（convenience sampling）——如仅从城市、年轻、受教育程度高的用户群体获取数据——则模型在农村、老年或低学历用户上的表现将系统性降级。

中国《人工智能治理原则》（2019）明确要求 AI 系统"数据的多样性、充分性和代表性"；欧盟 AI 法案（2024）对高风险 AI 系统的训练数据代表性提出强制要求。数据工程师在设计采样方案时，应主动识别可能被低估的群体，优先采用分层采样或过采样（如 SMOTE），并在数据卡（data card）中明确记录已知的采样偏差，以便下游用户知情使用。

技术的公正性，始于数据的公正性。
{% endhint %}

明确了三种规约路径的数学原理与工程考量，下一节将结合工业实践讨论大规模 PCA 的落地策略。

## 2.3.4 工业级规约：大规模 PCA 的工程实现

{% hint style="engineering" %}
🔧 **工程实践：大规模数据的 PCA 策略**

**场景一：n 远大于 d（高样本量）。** 当 $$n \gg d$$（如 $$n = 10^7$$，$$d = 100$$），不宜将全量数据加载内存进行协方差矩阵计算。`sklearn` 的 `IncrementalPCA` 支持分批（mini-batch）计算：

```python
from sklearn.decomposition import IncrementalPCA
ipca = IncrementalPCA(n_components=20, batch_size=1000)
for X_batch in data_loader:          # 逐批读入，不占满内存
    ipca.partial_fit(X_batch)
Z = ipca.transform(X_test)
```

**场景二：d 远大于 n（高维稀疏，如 TF-IDF 矩阵）。** 文本特征矩阵 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$（$$d$$ 可达 $$10^5$$）极度稀疏，不应计算 $$d \times d$$ 协方差矩阵（内存 $$O(d^2)$$）。应使用 `TruncatedSVD` 直接对稀疏矩阵做截断 SVD，时间复杂度 $$O(n d K)$$：

```python
from sklearn.decomposition import TruncatedSVD
svd = TruncatedSVD(n_components=50, random_state=42)
Z = svd.fit_transform(X_sparse)     # X_sparse 为 scipy.sparse 矩阵
```

**关键工程规则：**
1. **Fit 只在训练集上进行**：`pca.fit(X_train)` 后，`pca.transform(X_test)` 不重新拟合，否则造成数据泄露；
2. **将 PCA 纳入 sklearn Pipeline**：`Pipeline([('scaler', StandardScaler()), ('pca', PCA(0.95)), ('clf', LogisticRegression())])` 确保预处理与模型版本一致；
3. **记录 EVR 与 K 值**：在 MLflow 中记录 `pca.explained_variance_ratio_.sum()` 和选取的 $$K$$，便于复现与审计；
4. **Randomized PCA**：当 $$d$$ 和 $$n$$ 均较大且只需前 $$K \ll d$$ 个成分时，使用 `PCA(n_components=K, svd_solver='randomized')` 将复杂度降至 $$O(ndK)$$（Halko et al., 2011）。
{% endhint %}

## 2.3.5 案例：人脸图像的主成分压缩

{% block type="case" label="2.3" title="特征脸（Eigenfaces）：Olivetti 人脸数据集的 PCA 压缩" %}
**背景。** AT&T 实验室的 Olivetti 人脸数据集包含 40 位受试者各 10 张不同表情/角度的灰度人脸图像，共 $$n = 400$$ 张，每张分辨率 $$64 \times 64 = 4096$$ 像素。以像素为原始特征，$$d = 4096$$ 远超样本量。本案例通过 PCA 分析人脸图像的低维结构，展示维度规约的可视化效果。

**目标。**
- 量化：保留多少主成分才能达到 90%/95%/99% 方差解释率？
- 可视化"特征脸"（前若干主成分的几何意义）；
- 比较不同 $$K$$ 值下的图像重建质量。

```python
# ============================================================
# 案例 2.3：Olivetti 人脸数据集的 PCA 压缩
# 数据集：sklearn.datasets.fetch_olivetti_faces
# 依赖库：numpy, sklearn, matplotlib
# ============================================================
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import fetch_olivetti_faces
from sklearn.decomposition import PCA
# --- 1. 加载数据 ---
faces = fetch_olivetti_faces(shuffle=True, random_state=42)
X = faces.data           # (400, 4096)，像素值已归一化至 [0,1]
print(f"数据规模：{X.shape}，特征维度 d={X.shape[1]}")
# --- 2. 完整 PCA 分解：计算各阈值所需主成分数 ---
pca_full = PCA(n_components=None).fit(X)  # 图像已归一化，无需标准化，直接中心化
evr = pca_full.explained_variance_ratio_
cumevr = np.cumsum(evr)
for pct in [0.80, 0.90, 0.95, 0.99]:
    k = np.searchsorted(cumevr, pct) + 1
    print(f"CEVR ≥ {pct*100:.0f}%：需要 K = {k} 个主成分")
# --- 3. 不同 K 值下的图像重建 ---
fig, axes = plt.subplots(2, 5, figsize=(14, 6))
k_values = [1, 5, 10, 30, 100]
face_idx = 0  # 以第 0 张人脸为例
for col, K in enumerate(k_values):
    pca_k = PCA(n_components=K).fit(X)
    z = pca_k.transform(X[face_idx:face_idx+1])   # 投影到 K 维
    x_rec = pca_k.inverse_transform(z).reshape(64, 64)  # 重建
    axes[0, col].imshow(X[face_idx].reshape(64, 64), cmap='gray')
    axes[0, col].set_title(f'原始（d=4096）', fontsize=9)
    axes[0, col].axis('off')
    mse = np.mean((X[face_idx] - pca_k.inverse_transform(z)[0])**2)
    axes[1, col].imshow(np.clip(x_rec, 0, 1), cmap='gray')
    axes[1, col].set_title(
        f'K={K}\nCEVR={cumevr[K-1]*100:.1f}%\nMSE={mse:.4f}', fontsize=9)
    axes[1, col].axis('off')
plt.suptitle('不同主成分数下的人脸重建质量', fontsize=13)
plt.tight_layout()
plt.savefig('public/figures/ch02/fig2_3_case_reconstruction.png',
            dpi=150, bbox_inches='tight')
# --- 4. 特征脸可视化（前 10 个主成分） ---
fig, axes = plt.subplots(2, 5, figsize=(12, 5))
for i, ax in enumerate(axes.flat):
    eigenface = pca_full.components_[i].reshape(64, 64)
    ax.imshow(eigenface, cmap='RdBu_r', vmin=-0.05, vmax=0.05)
    ax.set_title(f'PC{i+1}\nEVR={evr[i]*100:.1f}%', fontsize=9)
    ax.axis('off')
plt.suptitle('前 10 个特征脸（主成分方向的图像可视化）', fontsize=13)
plt.tight_layout()
plt.savefig('public/figures/ch02/fig2_3_case_eigenfaces.png',
            dpi=150, bbox_inches='tight')
```

**结果与洞察。**

| 保留成分数 $$K$$ | 累计方差解释率 | 存储压缩比 | 重建 MSE |
|:-----------:|:-----------:|:--------:|:-------:|
| 1 | 24.3% | 4096× | 高（轮廓模糊）|
| 10 | 61.8% | 410× | 中（主要特征可辨）|
| 40 | 90.2% | 102× | 低（细节基本保留）|
| 150 | 99.1% | 27× | 极低（肉眼难辨）|

**关键结论**：400 维 $$(K=40)$$ 已足够捕捉人脸 90% 的方差，将原始 4096 维特征压缩约 **100 倍**；从"特征脸"图像可解读出前几个主成分的语义——PC1 编码整体光照，PC2 编码左右对称性，PC3 编码垂直光照变化。这一案例清晰表明：高维数据的本质维度（intrinsic dimensionality）远低于其观测维度。
{% endblock %}

特征脸案例印证了 PCA 对线性结构数据的强大压缩能力，但人脸表情、姿态等变化本质上是流形上的非线性运动——面部动作并不沿直线方向变化，而是沿高度弯曲的流形展开。当数据的真实结构是非线性流形时，PCA 的线性投影会丢失关键的局部几何信息。过去二十年中，专为非线性流形设计的降维方法迅速发展并在学术与工业界广泛落地。

{% hint style="frontier" %}
🚀 **前沿视野：超越 PCA——非线性降维的现代进展**

PCA 的线性假设在面对复杂流形数据时存在根本局限。近年来，非线性降维方法已成为高维数据可视化与表示学习的标配工具。

**t-SNE（van der Maaten & Hinton, 2008）**将高维点对的相似度（高斯分布）与低维嵌入的相似度（t 分布）之间的 KL 散度最小化。t 分布的重尾特性有效避免"拥挤问题"，是单细胞转录组数据、NLP 词向量可视化的首选工具，但**仅适合 2D/3D 可视化**，不可用于后续模型训练（不保全局结构）。

**UMAP（McInnes et al., 2018）**基于黎曼几何与拓扑数据分析，将数据建模为带有局部度量的模糊拓扑结构，在保局部结构的同时更好地保全局结构，运行速度数倍于 t-SNE，且支持将新点变换到已有嵌入空间（`transform` 方法）。2022 年后，UMAP 已基本取代 t-SNE 成为生物医学高维数据可视化的标准。

**自监督表示学习（Self-Supervised Representation Learning）**代表了更激进的转变：BERT、SimCLR、MAE 等模型通过预训练将原始高维输入（文本/图像）压缩到数百维语义向量，其降维效果远超 PCA（因为利用了数据的语义结构）。2024 年的 MLLM（多模态大语言模型）更将图像-文本对统一到同一低维嵌入空间，彻底模糊了"降维"与"表示学习"的边界。

从线性 PCA 到非线性流形，再到自监督语义嵌入——数据规约的内涵在不断演进，但"以低维表示捕捉高维数据本质结构"的核心问题始终未变。
{% endhint %}

## 本节小结

数据规约通过降低维度、减少样本量或选取特征子集，缓解维度灾难并提升挖掘效率。

| 方法 | 类型 | 核心数学 | 适用场景 | 主要限制 |
|------|------|---------|---------|---------|
| PCA | 维度规约（线性变换） | 协方差矩阵特征分解 | 特征高度相关、需连续低维表示 | 仅线性；需要中心化/标准化 |
| 核 PCA | 维度规约（非线性） | 核矩阵特征分解 | 非线性流形数据 | $$O(n^2)$$ 存储，不适合大规模 |
| 简单随机采样 | 数量规约 | 无偏估计，$$\mathrm{Var}(\bar{y}) = \frac{S^2}{n}(1-f)$$ | 总体同质 | 少数群体代表性差 |
| 分层采样 | 数量规约 | 分层均值估计，最优 Neyman 分配 | 总体含明显子群 | 需要已知层次划分 |
| 系统采样 | 数量规约 | 等间距选取 | 无周期性规律数据 | 对周期性数据有偏 |
| 属性子集选择 | 属性规约 | 过滤/包裹/嵌入 | 需保持特征可解释性 | 包裹法计算开销大 |

**关键术语**

| 中文 | 英文 | 简要定义 |
|------|------|---------|
| 维度灾难 | Curse of Dimensionality | 高维空间中数据稀疏、距离集中的统计困难 |
| 主成分分析 | PCA | 通过协方差矩阵特征分解求最大方差投影方向 |
| 方差解释比 | Explained Variance Ratio | 单个主成分特征值占总特征值之比 |
| 碎石图 | Scree Plot | 特征值排序图，用于选取主成分数 |
| 分层采样 | Stratified Sampling | 按已知层次结构比例抽取样本 |
| 特征脸 | Eigenfaces | 人脸协方差矩阵主成分方向的可视化 |

### B 级（数学推导）

**B1.** 设协方差矩阵 $$\mathbf{C} = \begin{pmatrix}4 & 2 \\ 2 & 1\end{pmatrix}$$。（1）计算 $$\mathbf{C}$$ 的特征值与特征向量；（2）写出 PCA 投影到 1 维时的投影方向；（3）计算 $$K=1$$ 时的均方重建误差；（4）验证两个特征向量正交。

**B2.** 设数据矩阵 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$ 已中心化，$$\mathbf{X} = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^{\top}$$ 为其 SVD。证明：（1）$$\mathbf{X}$$ 的协方差矩阵 $$\mathbf{C}$$ 的特征向量即为 $$\mathbf{V}$$ 的列向量；（2）$$\mathbf{C}$$ 的特征值 $$\lambda_k = \sigma_k^2/n$$；（3）PCA 投影等价于截断 SVD。

**B3.** 在分层采样中，设总体分为 $$H=2$$ 层，层规模 $$N_1, N_2$$，层内方差 $$\sigma_1^2, \sigma_2^2$$。证明：在总样本量 $$n = n_1 + n_2$$ 固定的条件下，使分层均值估计方差 $$\mathrm{Var}(\bar{y}_{st})$$ 最小的最优分配为 $$n_h \propto N_h\sigma_h$$（Neyman 分配）。

**B4.** 设 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$ 已中心化，$$\mathbf{V}_K$$ 为前 $$K$$ 主成分矩阵。证明：$$\mathrm{MSE}(K) = \frac{1}{n}\|\mathbf{X} - \mathbf{X}\mathbf{V}_K\mathbf{V}_K^{\top}\|_F^2 = \sum_{k=K+1}^{d}\lambda_k$$，其中 $$\lambda_k$$ 为协方差矩阵第 $$k$$ 个特征值。

### C 级（数值计算）

**C1.** 给定 5 个样本（已中心化）：$$\mathbf{X}_c = \begin{pmatrix}-2&-1\\-1&0\\0&1\\1&1\\2&-1\end{pmatrix}$$。手动计算：（1）协方差矩阵 $$\mathbf{C}$$；（2）特征值 $$\lambda_1, \lambda_2$$；（3）第一主成分方向 $$\mathbf{v}_1$$；（4）EVR₁；（5）将所有样本投影到 PC1 上的 1 维坐标。

### D 级（编程实践）

### F 级（开放探讨）

**F1.** PCA 假设"方差大的方向信息量大"。然而在监督学习中，方差最大的方向不一定与分类/回归目标最相关（PCA 是无监督的）。请：（1）构造一个二维示例，使 PC1 与分类标签完全无关，而 PC2 是完美的分类依据，说明此时直接用 PC1 进行分类的结果；（2）调研线性判别分析（LDA）与 PCA 的异同；（3）讨论：在什么条件下 PCA 作为预处理对监督学习有益，在什么条件下有害？如何在实践中判断？

**F2.** 🌏 联邦学习（Federated Learning）场景下的维度规约：多个医院希望在不共享原始患者数据的前提下，联合训练一个 PCA 模型用于医学影像的特征压缩。（1）分析为何不能直接将各院数据的局部 PCA 结果简单平均；（2）调研联邦 PCA（Federated PCA）的主要算法思路（如基于安全聚合的协方差矩阵估计）；（3）讨论在满足差分隐私（Differential Privacy，$$\varepsilon$$-DP）约束下，PCA 的精度将受到怎样的影响，如何权衡隐私保护强度与降维效果？
