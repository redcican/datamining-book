# 第一章 数据挖掘概述

数据挖掘是从大规模数据中自动发现有价值模式的交叉学科，融合了统计学、机器学习和数据库技术。本章作为全书的起点，回答三个根本性问题：数据挖掘**是什么**（定义与边界）、**能做什么**（基本任务与应用领域）、**怎么做**（流程与方法论）。§1.1 从 Fayyad 的 KDD 框架出发给出数据挖掘的形式化定义，梳理其 40 年发展历程；§1.2 系统介绍分类、回归、聚类、关联规则挖掘和异常检测五大基本任务，并展示金融、医疗、零售等行业的典型应用；§1.3 对比 KDD、CRISP-DM 和 SEMMA 三种主流方法论，建立贯穿全书的项目实践框架。

## 本章学习目标

1. 理解数据挖掘与 KDD 的关系，掌握数据挖掘的形式化定义；
2. 区分数据挖掘的五大基本任务——分类、回归、聚类、关联规则挖掘与异常检测；
3. 了解数据挖掘在金融、医疗、零售、工业等领域的典型应用场景；
4. 掌握 KDD 九步流程和 CRISP-DM 六阶段方法论的核心思想；
5. 理解数据质量五维度与数据集划分策略对项目成功的关键作用。
