# 第六章 关联规则挖掘

关联规则挖掘旨在从事务数据中发现项目之间的有趣关联模式——最经典的应用是"购物篮分析"。与分类和回归不同，关联规则不预设目标变量，而是在所有项目组合中搜索频繁共现模式。本章从数学基础出发系统介绍关联规则挖掘的完整技术体系：§6.1 形式化定义事务数据库、支持度、置信度和提升度；§6.2 详解 Apriori 算法的逐层搜索策略和候选剪枝原理；§6.3 介绍 FP-Growth 的压缩存储与无候选挖掘方案；§6.4 分析支持度-置信度框架的局限，引入全置信度、余弦、Kulczynski 等兴趣度量及零不变性概念；§6.5 将关联规则扩展到概念层次和量化属性；§6.6 介绍有向图、共现网络、热力图等可视化方法；§6.7 通过零售、医疗和 Web 点击流三个案例展示端到端应用。

## 本章学习目标

1. 掌握事务数据库、支持度、置信度、提升度的形式化定义与 Apriori 性质；
2. 理解 Apriori 算法的逐层候选生成-剪枝策略及其复杂度分析；
3. 掌握 FP-Tree 的构造过程和 FP-Growth 的条件模式基递归挖掘方法；
4. 能辨析提升度、全置信度、余弦、Kulczynski 等兴趣度量的适用场景；
5. 理解多层次关联规则的概念层次与支持度策略；
6. 能使用有向图、共现网络和热力图可视化和解释关联规则；
7. 能在真实数据集上完成从数据预处理到规则发现的端到端关联分析。
