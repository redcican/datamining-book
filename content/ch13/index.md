# 第十三章 数据挖掘实践案例

前十二章系统介绍了数据挖掘的核心方法（第 3–11 章）和工具平台（第 12 章）。本章将这些知识融会贯通，通过七个来自不同行业领域的真实案例，展示数据挖掘从问题定义到结果分析的完整实践流程。每个案例均使用公开数据集和 Python 代码，读者可完整复现。七个案例覆盖了关联规则、分类、聚类、图挖掘、推荐系统、异常检测和时序预测等核心方法，涉及零售、金融、医疗、社交、娱乐、工业和环境等多个行业领域。

## 本章学习目标

1. 能在零售数据上完成从数据预处理到关联规则挖掘的端到端分析；
2. 掌握金融欺诈检测中的类不平衡处理策略和多模型对比方法；
3. 理解医疗数据挖掘中模型可解释性的重要性，掌握 SHAP 解释方法；
4. 能对社交网络数据进行中心性分析和社区发现；
5. 掌握协同过滤推荐系统的用户基/物品基/矩阵分解三种实现方法；
6. 理解工业预测性维护中异常检测与分类的结合策略；
7. 能设计时间序列回归的特征工程方案，完成环境数据的预测建模。
