# 第九章 文本挖掘

企业数据存储中超过 80% 是非结构化文本。机器学习算法处理的是数值向量，因此文本挖掘的核心挑战是如何将自然语言转化为数学对象并保留语义信息。本章系统介绍文本数据挖掘的完整技术体系：§9.1 介绍分词、去停用词、词干提取/词形还原等预处理流程，特别讨论中文分词的挑战；§9.2 梳理文本表示模型的演进——从词袋模型、TF-IDF 到 Word2Vec 和 BERT；§9.3 介绍朴素贝叶斯、线性 SVM 和 TextCNN 三种文本分类方法；§9.4 讨论 LSA 和 LDA 主题模型的数学原理；§9.5 从词典方法到 BiLSTM+Attention，介绍情感分析的三个层次（文档/句子/方面级）；§9.6 介绍 TextRank 摘要算法和 CRF 命名实体识别；§9.7 讨论跨语言词嵌入对齐和零样本跨语言迁移。

## 本章学习目标

1. 掌握文本预处理的完整流程——分词、去停用词、词干提取/词形还原与文本规范化；
2. 理解文本表示模型的演进——从词袋模型、TF-IDF 到 Word2Vec 和 BERT；
3. 掌握朴素贝叶斯、线性 SVM 和 TextCNN 三种文本分类方法；
4. 理解 LSA 的 SVD 降维思想和 LDA 的概率生成模型；
5. 掌握情感分析的词典方法与深度学习方法，理解方面级情感分析；
6. 理解 TextRank 图排序算法和 CRF 序列标注方法；
7. 了解跨语言词嵌入对齐（Procrustes 方法）和零样本跨语言迁移。
