# 9.4 文本聚类与主题模型

上一节（§9.3）讨论的文本分类是**有监督**任务——需要人工标注的类别标签来训练分类器。但在许多场景中，我们面对的是**无标注**的文本集合：一批客户反馈、一组学术论文、数年的新闻存档——我们不知道其中有哪些主题，更不知道每个文档属于什么类别。此时需要的是**无监督**方法：文本聚类和主题模型。

文本聚类回答"哪些文档是相似的"——将文档按内容相似性分组。主题模型回答"这些文档在讨论什么"——自动发现语料库中潜在的主题结构。两者的目标不同但互补：聚类将每个文档分配到**一个**组，主题模型认为每个文档可以**混合多个主题**（例如一篇新闻可能 60% 讨论科技、30% 讨论财经、10% 讨论政策）。

## 9.4.1 文本聚类

### 9.4.1.1 问题定义

{% block type="definition" title="定义 9.17　文本聚类" %}
给定无标注的文档集合 $$\mathcal{D} = \{d_1, d_2, \ldots, d_N\}$$ 及其向量表示 $$\{\mathbf{x}_1, \ldots, \mathbf{x}_N\}$$，**文本聚类**是将文档划分为 $$K$$ 个不相交的组 $$\{\mathcal{G}_1, \ldots, \mathcal{G}_K\}$$，使得同组文档相似度高、不同组文档相似度低。

在 TF-IDF 空间中，文档相似度通常使用余弦相似度（§9.2 方程 9.14）。对应的聚类目标函数为**球面 K-means**：

$$
\max_{\{\mathcal{G}_k\}, \{\boldsymbol{\mu}_k\}} \sum_{k=1}^{K} \sum_{\mathbf{x}_i \in \mathcal{G}_k} \frac{\mathbf{x}_i^\top \boldsymbol{\mu}_k}{\|\mathbf{x}_i\| \cdot \|\boldsymbol{\mu}_k\|} \tag{9.37}
$$

其中 $$\boldsymbol{\mu}_k$$ 为第 $$k$$ 个簇的中心向量。
{% endblock %}

标准 K-means 使用欧氏距离，但在高维稀疏的文本空间中，**余弦相似度比欧氏距离更合理**。原因在于：TF-IDF 向量的模（$$\|\mathbf{x}\|$$）主要反映文档长度，而非内容——一篇 1,000 词的文章和一篇 100 词的摘要如果主题相同，余弦相似度接近 1，但欧氏距离可能很大。球面 K-means 通过对向量做 L2 归一化（投影到单位超球面上）消除了长度的影响。

### 9.4.1.2 文本聚类的挑战

直接对 TF-IDF 向量做 K-means 面临两个主要挑战：

- **维度灾难**：词汇表大小 $$|V|$$ 通常为 $$10^4$$–$$10^5$$，在如此高维的空间中，点之间的距离趋于均匀化——"距离集中"现象使得聚类效果下降。
- **同义词问题**：两篇主题相同但使用不同措辞的文档可能在 TF-IDF 空间中距离很远（例如"机器学习"和"人工智能算法"可能没有共享词汇）。

这两个问题的共同解决方案是**降维**——将文档从高维稀疏的词汇空间映射到低维稠密的**语义空间**。LSA 正是为此目的设计的。

## 9.4.2 潜在语义分析

### 9.4.2.1 SVD 分解

{% block type="definition" title="定义 9.18　潜在语义分析（LSA / LSI）" %}
**潜在语义分析**（Latent Semantic Analysis，又称 Latent Semantic Indexing, LSI）通过对文档-词项矩阵进行**奇异值分解**（SVD），发现词项和文档的潜在语义结构。

设 TF-IDF 矩阵为 $$\mathbf{X} \in \mathbb{R}^{N \times |V|}$$，其 SVD 分解为：

$$
\mathbf{X} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^\top \tag{9.38}
$$

其中 $$\mathbf{U} \in \mathbb{R}^{N \times r}$$ 的列为左奇异向量（文档在语义空间中的坐标），$$\mathbf{V} \in \mathbb{R}^{|V| \times r}$$ 的列为右奇异向量（词项在语义空间中的坐标），$$\boldsymbol{\Sigma} = \text{diag}(\sigma_1, \ldots, \sigma_r)$$ 为奇异值（$$\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0$$），$$r = \text{rank}(\mathbf{X})$$。

**截断 SVD**：保留前 $$k$$ 个最大奇异值（$$k \ll r$$），得到低秩近似：

$$
\mathbf{X}_k = \mathbf{U}_k \boldsymbol{\Sigma}_k \mathbf{V}_k^\top \tag{9.39}
$$

文档 $$d_i$$ 在 $$k$$ 维语义空间中的表示为 $$\mathbf{U}_k$$ 的第 $$i$$ 行乘以 $$\boldsymbol{\Sigma}_k$$。
{% endblock %}

LSA 的核心洞察是：**词汇空间的维度远大于语义空间的维度**。成千上万个词汇可能只表达几十到几百个潜在概念——"机器学习""深度学习""神经网络"都指向"AI 技术"这一潜在语义维度。SVD 将这些共现模式压缩到少数奇异向量中，实现了从词汇空间到语义空间的降维。

### 9.4.2.2 最优低秩近似

{% block type="theorem" title="定理 9.5　Eckart-Young 定理" %}
在所有秩不超过 $$k$$ 的矩阵中，截断 SVD 给出的 $$\mathbf{X}_k$$（方程 9.39）是 $$\mathbf{X}$$ 在 Frobenius 范数下的**最优近似**：

$$
\mathbf{X}_k = \arg\min_{\text{rank}(\mathbf{A}) \leq k} \|\mathbf{X} - \mathbf{A}\|_F^2 \tag{9.40}
$$

其中 $$\|\mathbf{X} - \mathbf{X}_k\|_F^2 = \sum_{i=k+1}^{r} \sigma_i^2$$。

**证明概要**　设 $$\mathbf{A}$$ 是任意秩不超过 $$k$$ 的矩阵，则 $$\mathbf{A}$$ 的零空间维度至少为 $$|V| - k$$。利用 Courant-Fischer 极小极大定理，可以证明 $$\|\mathbf{X} - \mathbf{A}\|_F^2 \geq \sum_{i=k+1}^{r} \sigma_i^2$$，而截断 SVD 恰好达到这一下界。$$\blacksquare$$
{% endblock %}

这一定理保证了 LSA 的最优性：在所有将文档映射到 $$k$$ 维空间的线性方法中，SVD 保留了**最多的原始信息**（以重构误差衡量）。LSA 的效果高度依赖于 $$k$$ 的选择——$$k$$ 太小会丢失重要语义，$$k$$ 太大会保留噪声。典型的 $$k$$ 值为 100–300。

{% hint style="tip" %}
🔧 **工程实践：LSA 的高效实现**

对于大规模文档-词项矩阵（$$N > 10^5$$，$$|V| > 10^5$$），完整 SVD 的计算代价为 $$O(\min(N, |V|)^2 \cdot \max(N, |V|))$$，通常不可接受。实践中使用**截断 SVD**的高效算法——随机化 SVD（Halko et al., 2011）可以在 $$O(N |V| k)$$ 时间内计算前 $$k$$ 个奇异值，且利用 TF-IDF 矩阵的稀疏性进一步加速。scikit-learn 的 `TruncatedSVD` 正是基于此实现。

```python
# --- 1. TF-IDF 向量化 ---
from sklearn.decomposition import TruncatedSVD
from sklearn.feature_extraction.text import TfidfVectorizer
tfidf = TfidfVectorizer(max_features=10000)
X_tfidf = tfidf.fit_transform(documents)
# --- 2. LSA 降维 ---
lsa = TruncatedSVD(n_components=100, random_state=42)
X_lsa = lsa.fit_transform(X_tfidf)
# X_lsa: (N, 100) 的稠密文档表示
```

## 9.4.3 潜在狄利克雷分配

### 9.4.3.1 从聚类到混合成员

LSA 和 K-means 都假设每个文档属于**一个**语义维度或一个簇。但真实文档通常涉及**多个主题**——一篇关于"AI 在医疗中的应用"的论文同时涉及"人工智能"和"医学"两个主题。**主题模型**放弃了"硬分配"的假设，允许每个文档是多个主题的混合体。

{% block type="definition" title="定义 9.19　主题模型（Topic Model）" %}
**主题模型**假设语料库中存在 $$K$$ 个潜在**主题**（Topic），每个主题是词汇表上的一个概率分布。文档的生成过程为：

1. 每个文档 $$d$$ 有一个**主题分布** $$\boldsymbol{\theta}_d = (\theta_{d1}, \ldots, \theta_{dK})$$，表示该文档对各主题的偏好程度
2. 每个主题 $$k$$ 有一个**词分布** $$\boldsymbol{\phi}_k = (\phi_{k1}, \ldots, \phi_{k|V|})$$，表示该主题下各词项的出现概率
3. 文档中的每个词先从主题分布中抽取一个主题，再从该主题的词分布中抽取一个词
{% endblock %}

这个框架的关键洞察是**双重概率结构**：文档到主题的映射（$$\boldsymbol{\theta}$$）和主题到词的映射（$$\boldsymbol{\phi}$$）。通过推断这两组参数，我们可以同时获得(1) 每个文档的主题构成和 (2) 每个主题的代表性词汇。

### 9.4.3.2 LDA 生成模型

{% block type="definition" title="定义 9.20　潜在狄利克雷分配（LDA）" %}
**LDA**（Latent Dirichlet Allocation, Blei et al., 2003）是最经典的主题模型。它为文档-主题分布和主题-词分布分别施加 **Dirichlet 先验**：

**文档-主题分布**：

$$
\boldsymbol{\theta}_d \sim \text{Dir}(\boldsymbol{\alpha}), \quad d = 1, \ldots, N \tag{9.41}
$$

**主题-词分布**：

$$
\boldsymbol{\phi}_k \sim \text{Dir}(\boldsymbol{\beta}), \quad k = 1, \ldots, K \tag{9.42}
$$

**词的生成过程**：对文档 $$d$$ 中的第 $$n$$ 个词（$$n = 1, \ldots, L_d$$）：

$$
z_{dn} \sim \text{Cat}(\boldsymbol{\theta}_d), \quad w_{dn} \sim \text{Cat}(\boldsymbol{\phi}_{z_{dn}}) \tag{9.43}
$$

其中 $$z_{dn} \in \{1, \ldots, K\}$$ 是潜在主题指示变量，$$w_{dn}$$ 是观测到的词项。

语料库的**联合概率**为：

$$
P(\mathbf{W}, \mathbf{Z}, \boldsymbol{\Theta}, \boldsymbol{\Phi} \mid \boldsymbol{\alpha}, \boldsymbol{\beta}) = \prod_{k=1}^{K} P(\boldsymbol{\phi}_k \mid \boldsymbol{\beta}) \prod_{d=1}^{N} P(\boldsymbol{\theta}_d \mid \boldsymbol{\alpha}) \prod_{n=1}^{L_d} P(z_{dn} \mid \boldsymbol{\theta}_d) P(w_{dn} \mid \boldsymbol{\phi}_{z_{dn}}) \tag{9.44}
$$
{% endblock %}

Dirichlet 先验的超参数控制分布的"稀疏性"：$$\alpha < 1$$ 鼓励每个文档只关注少数主题（稀疏的主题分布），$$\beta < 1$$ 鼓励每个主题只使用少数词汇（稀疏的词分布）。这与真实文档的特点一致——一篇新闻通常只涉及 2–3 个主题，一个主题的核心词汇通常只有 20–50 个。

### 9.4.3.3 Gibbs 采样推断

LDA 的推断目标是计算后验分布 $$P(\mathbf{Z} \mid \mathbf{W}, \boldsymbol{\alpha}, \boldsymbol{\beta})$$——给定观测到的词，推断每个词的潜在主题分配。这一后验没有解析解，需要近似推断。**Gibbs 采样**是最常用的推断方法。

其核心思想是：每次固定其他所有词的主题分配，只更新一个词 $$w_{dn}$$ 的主题分配 $$z_{dn}$$：

$$
P(z_{dn} = k \mid \mathbf{Z}_{-dn}, \mathbf{W}, \boldsymbol{\alpha}, \boldsymbol{\beta}) \propto (n_{dk}^{-dn} + \alpha_k) \cdot \frac{n_{kw}^{-dn} + \beta_w}{\sum_{w'} n_{kw'}^{-dn} + \beta_{w'}} \tag{9.45}
$$

其中 $$n_{dk}^{-dn}$$ 是文档 $$d$$ 中（排除位置 $$dn$$）被分配到主题 $$k$$ 的词数，$$n_{kw}^{-dn}$$ 是主题 $$k$$ 中（排除位置 $$dn$$）词项 $$w$$ 出现的次数。

方程 (9.45) 有直观的解释：一个词被分配到主题 $$k$$ 的概率正比于两个因子的乘积——(1) **文档中已有多少词属于主题 $$k$$**（$$n_{dk}^{-dn} + \alpha_k$$，即"近朱者赤"）；(2) **主题 $$k$$ 中这个词有多常见**（$$(n_{kw}^{-dn} + \beta_w) / \sum_{w'} (n_{kw'}^{-dn} + \beta_{w'})$$，即"词的主题亲和力"）。

{% block type="algorithm" title="算法 9.4　LDA 的 Gibbs 采样" %}
**输入**：词-文档数据 $$\{w_{dn}\}$$，主题数 $$K$$，超参数 $$\boldsymbol{\alpha}, \boldsymbol{\beta}$$，迭代次数 $$T$$

**输出**：主题分配 $$\{z_{dn}\}$$，文档-主题计数 $$n_{dk}$$，主题-词计数 $$n_{kw}$$

**步骤 1（随机初始化）**：
$$\quad$$ **for** 每个文档 $$d$$，每个词位置 $$n$$ **do**
$$\quad\quad$$ $$z_{dn} \sim \text{Uniform}(1, K)$$
$$\quad\quad$$ 更新计数矩阵 $$n_{dk}$$ 和 $$n_{kw}$$
$$\quad$$ **end for**

**步骤 2（迭代采样）**：
$$\quad$$ **for** $$t = 1$$ **to** $$T$$ **do**
$$\quad\quad$$ **for** 每个文档 $$d$$，每个词位置 $$n$$ **do**
$$\quad\quad\quad$$ 将当前分配 $$z_{dn}^{\text{old}}$$ 从计数中减去
$$\quad\quad\quad$$ 按方程 (9.45) 计算 $$K$$ 个主题的条件概率
$$\quad\quad\quad$$ 采样新的主题：$$z_{dn}^{\text{new}} \sim \text{Cat}(p_1, \ldots, p_K)$$
$$\quad\quad\quad$$ 将新分配加入计数
$$\quad\quad$$ **end for**
$$\quad$$ **end for**

**步骤 3（估计参数）**：
$$\quad$$ $$\hat{\theta}_{dk} = (n_{dk} + \alpha_k) / (\sum_{k'} n_{dk'} + \alpha_{k'})$$
$$\quad$$ $$\hat{\phi}_{kw} = (n_{kw} + \beta_w) / (\sum_{w'} n_{kw'} + \beta_{w'})$$

**时间复杂度**：每次迭代 $$O(N_{\text{total}} \cdot K)$$（$$N_{\text{total}} = \sum_d L_d$$ 为语料总词数）
{% endblock %}

{% hint style="warning" %}
⚠️ **易错点：LDA 的主题数 K 需要预设**

LDA 最大的局限是需要**预先指定主题数 $$K$$**。$$K$$ 太小会将不同主题合并，$$K$$ 太大会将一个主题拆分为多个碎片。常用的选择方法包括：(1) **困惑度**——在不同 $$K$$ 下计算测试集困惑度，选择拐点；(2) **主题一致性**——衡量每个主题内高概率词的语义一致性；(3) **非参数方法**——Hierarchical Dirichlet Process（HDP）自动从数据中推断主题数。实践中，建议尝试多个 $$K$$ 值并结合人工检查主题质量来决定。
{% endhint %}

## 9.4.4 主题模型评价

### 9.4.4.1 困惑度

**困惑度**（Perplexity）衡量模型对未见文本的预测能力——困惑度越低，模型对新文档的"惊讶程度"越小：

$$
\text{Perplexity}(\mathcal{D}_{\text{test}}) = \exp\left(-\frac{\sum_{d} \sum_{n} \log P(w_{dn})}{\sum_{d} L_d}\right) \tag{9.46}
$$

其中 $$P(w_{dn}) = \sum_{k=1}^{K} \theta_{dk} \phi_{k, w_{dn}}$$ 是模型对词 $$w_{dn}$$ 的预测概率。困惑度的直觉是：如果模型认为测试文本中每个词平均有 $$p$$ 种等可能的选择，则困惑度为 $$p$$。

### 9.4.4.2 主题一致性

困惑度衡量的是统计拟合度，但高统计拟合度不一定意味着"好的主题"。**主题一致性**（Topic Coherence）直接衡量每个主题内的高概率词是否语义相关：

$$
C(k) = \frac{2}{M(M-1)} \sum_{i=1}^{M} \sum_{j=i+1}^{M} \text{sim}(w_i^{(k)}, w_j^{(k)}) \tag{9.47}
$$

其中 $$w_1^{(k)}, \ldots, w_M^{(k)}$$ 是主题 $$k$$ 概率最高的 $$M$$ 个词，$$\text{sim}$$ 可以是 PMI 或归一化 PMI。一致性高的主题中，高概率词之间具有语义关联（如"学习""训练""模型""算法"）；一致性低的主题则包含语义不相关的词（如"学习""天气""足球""银行"）。

{% hint style="info" %}
🤔 **思考：LSA vs LDA——选择哪个？**

LSA 和 LDA 都能从文本中发现潜在结构，但适用场景不同：(1) **LSA** 是确定性方法，计算高效，适合**信息检索**（降维后的余弦相似度比原始 TF-IDF 更好）和**可视化**（降到 2–3 维）；(2) **LDA** 是概率生成模型，输出可解释的主题-词分布，适合**主题发现**和**文档理解**（"这篇文章 40% 是科技、35% 是财经"）。如果目标是"将文档分组"，LSA + K-means 足够；如果目标是"发现每篇文档讨论了什么主题"，LDA 是更自然的选择。
{% endhint %}

## 9.4.5 案例分析

{% block type="case" title="案例 9.4　新闻语料主题发现" %}

**问题背景**：给定一组新闻文档，使用 LSA 降维进行文档聚类，并用主题模型发现潜在的主题结构。比较 LSA 降维后的 K-means 聚类与原始 TF-IDF 空间中 K-means 的效果。

**数据说明**：模拟 200 条新闻，覆盖 5 个主题（体育、科技、财经、娱乐、教育），每个文档是若干关键词的组合。

**方法与步骤**：

```python
# --- 1. 导入工具 ---
import numpy as np
from collections import Counter
# --- 2. 模拟主题-词分布 ---
np.random.seed(42)
K = 5  # 主题数
V = 100  # 词汇表大小
N = 200  # 文档数
topic_names = ["体育", "科技", "财经", "娱乐", "教育"]
# 每个主题有 20 个高概率词
phi = np.random.dirichlet(np.ones(V) * 0.01, K)
for k in range(K):
    high_prob_idx = range(k * 20, (k + 1) * 20)
    phi[k, high_prob_idx] += 0.05
    phi[k] /= phi[k].sum()
# --- 3. 生成文档 ---
alpha = np.ones(K) * 0.3  # 稀疏的主题分布
doc_word_matrix = np.zeros((N, V))
true_topics = np.zeros(N, dtype=int)
for d in range(N):
    theta_d = np.random.dirichlet(alpha)
    true_topics[d] = np.argmax(theta_d)
    n_words = np.random.randint(30, 80)
    for _ in range(n_words):
        z = np.random.choice(K, p=theta_d)
        w = np.random.choice(V, p=phi[z])
        doc_word_matrix[d, w] += 1
# --- 4. TF-IDF 变换 ---
df = np.sum(doc_word_matrix > 0, axis=0)
idf = np.log(N / (df + 1))
tfidf = doc_word_matrix * idf
# --- 5. LSA 降维 ---
from numpy.linalg import svd
U, S, Vt = svd(tfidf, full_matrices=False)
k_lsa = 20
X_lsa = U[:, :k_lsa] * S[:k_lsa]
# --- 6. K-means 聚类 ---
from sklearn.cluster import KMeans
km_tfidf = KMeans(n_clusters=K, random_state=42, n_init=10)
km_tfidf.fit(tfidf)
km_lsa = KMeans(n_clusters=K, random_state=42, n_init=10)
km_lsa.fit(X_lsa)
# --- 7. 评估 (NMI) ---
from sklearn.metrics import normalized_mutual_info_score as nmi
print(f"TF-IDF K-means NMI: {nmi(true_topics, km_tfidf.labels_):.3f}")
print(f"LSA K-means NMI:    {nmi(true_topics, km_lsa.labels_):.3f}")
```

**结果分析**：

1. **LSA 提升聚类效果**：在 LSA 降维后的空间中做 K-means，NMI（归一化互信息）通常比直接在 TF-IDF 空间中聚类高 5%–15%。这是因为 LSA 消除了词汇空间中的噪声维度，使得不同主题的文档在语义空间中分离更清晰。

2. **主题可解释性**：通过检查 $$\mathbf{V}_k$$ 的每一列中绝对值最大的词项，可以为每个潜在语义维度赋予解释——例如，第一个维度可能由"比赛""冠军""进球"主导，对应"体育"主题。

3. **维度选择**：$$k = 20$$ 已经能保留原始信息的 85% 以上（由前 20 个奇异值的平方和占总奇异值平方和的比例衡量）。$$k$$ 从 5 增加到 20 时聚类效果显著提升，$$k > 50$$ 后提升趋于平缓。

4. **与 LDA 对比**：LDA 的输出更加直观——每个主题的 Top 词直接展示为词云或柱状图，文档-主题分布直接给出每篇文档的主题构成。LSA 的语义维度需要人工解读，但计算效率更高。

|--------|---------|

**B2.** 推导 LDA 的 Gibbs 采样条件概率（方程 9.45）。从联合分布 $$P(\mathbf{Z}, \mathbf{W} \mid \boldsymbol{\alpha}, \boldsymbol{\beta})$$ 出发，利用 Dirichlet-多项式共轭关系，得到积分掉 $$\boldsymbol{\theta}$$ 和 $$\boldsymbol{\phi}$$ 后的条件分布。

**B3.** 推导困惑度（方程 9.46）与交叉熵之间的关系。说明为什么最小化困惑度等价于最大化模型对测试数据的似然。

### E 级（案例分析）

**E1.** 在一个学术论文摘要数据集（如 arXiv 摘要）上：(1) 使用 LDA 发现 10 个主题；(2) 分析每个主题的 Top 词是否对应可识别的学术领域；(3) 追踪不同主题在年份上的变化趋势——哪些领域在增长，哪些在萎缩？

**E2.** 在社交媒体数据（如 Twitter/微博）上进行主题分析：(1) 预处理短文本（去噪、合并 hashtag）；(2) 使用 LDA 和 LSA 分别提取主题；(3) 比较两种方法在短文本上的表现差异——短文本的稀疏性对 LDA 有什么影响？

### F 级（开放探讨）

**F1.** **神经主题模型**：传统 LDA 使用 Gibbs 采样或变分推断，训练速度慢且难以扩展。近年来，基于变分自编码器（VAE）的神经主题模型（如 NVDM、ProdLDA、ETM）将主题模型的推断过程替换为神经网络。讨论：(1) 神经主题模型与 LDA 在建模假设上的异同；(2) 神经主题模型的优势（速度、可扩展性）和劣势（可解释性、稳定性）；(3) 预训练语言模型时代，主题模型是否仍有价值？

**F2.** **动态主题模型**：标准 LDA 假设主题分布不随时间变化，但现实中主题会演化——"人工智能"的含义在 2010 年和 2025 年大不相同。讨论动态主题模型（Blei & Lafferty, 2006）如何建模主题的时间演化，以及它在学术文献分析、新闻追踪中的应用。
