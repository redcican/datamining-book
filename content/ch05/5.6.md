# 5.6 聚类有效性评估

前五节介绍了五大类聚类方法——划分式（K-means）、密度（DBSCAN）、层次（AGNES/DIANA）、网格（STING/CLIQUE）和模型（GMM）。一个自然且关键的问题是：**给定一个聚类结果，如何评价它的好坏？**

这个问题之所以困难，在于聚类是**无监督学习**：没有"标准答案"可供对照。与分类任务可以计算准确率不同，聚类结果的质量必须从其他角度来衡量——要么与已知标签对比（外部评估），要么仅从数据本身的结构来判断（内部评估）。本节系统建立聚类评估的数学框架。

{% hint style="info" %}
📋 **本节学习目标**

完成本节学习后，读者应能够：

1. 区分外部评估与内部评估的适用场景和各自局限；
2. 推导并计算 Adjusted Rand Index（ARI）和 Normalized Mutual Information（NMI）两种外部评估指标（定义 5.26–5.27）；
3. 给出轮廓系数（Silhouette Coefficient）的完整定义和取值范围证明（定义 5.28，定理 5.11）；
4. 计算 Davies-Bouldin 指数和 Calinski-Harabasz 指数（定义 5.29–5.30）；
5. 使用 Gap 统计量自动确定最优簇数 $K$（定义 5.31）；
6. 在多种聚类算法和数据集上系统应用评估指标，分析指标间的一致性与分歧；
7. 根据任务场景选择合适的评估策略。
{% endhint %}

**前置知识**：§5.1（K-means，轮廓系数初步）；§5.5（GMM，BIC/AIC）；基础概率与信息论概念。本节方程编号从 (5.70) 起，定义从 5.26 起，定理从 5.11 起。

## 5.6.1 为什么需要聚类评估

### 5.6.1.1 无监督学习的"无答案"困境

聚类是典型的无监督学习任务——数据中没有预定义的类标签。这导致聚类结果的评估面临以下挑战：

1. **无标准答案**：同一数据集上，$K=3$ 和 $K=5$ 的聚类结果哪个更"正确"？没有绝对标准；
2. **算法依赖**：不同算法基于不同假设（球形簇、密度连通、层次结构），对同一数据可能给出截然不同的结果；
3. **主观性**：聚类的"好坏"部分取决于应用目标——市场细分的"好"聚类和生物分类的"好"聚类标准不同。

尽管如此，聚类评估仍然是必要的，原因包括：

- **算法选择**：在多种候选算法中选择最合适的一个；
- **参数调优**：为 $K$（K-means）、$\varepsilon$（DBSCAN）等参数选择最优值；
- **结果验证**：检验聚类结果是否反映了数据的真实结构，而非算法的人为产物。

### 5.6.1.2 评估指标的分类

| 类别 | 前提条件 | 评估思路 | 典型指标 |
|------|---------|---------|---------|
| **外部评估** | 有真实标签（ground truth） | 聚类结果与真实标签的一致性 | ARI、NMI、F-measure |
| **内部评估** | 无真实标签 | 聚类结果的紧凑性与分离度 | 轮廓系数、DBI、CH 指数 |
| **相对评估** | 比较多个聚类结果 | 不同参数/算法的结果排序 | Gap 统计量、稳定性分析 |

## 5.6.2 外部评估指标

外部评估需要一个**真实标签集**（ground truth labels），在基准测试和学术研究中广泛使用。

### 5.6.2.1 配对计数框架

设 $n$ 个样本的真实标签为 $\mathbf{y} = (y_1, \ldots, y_n)$，聚类结果标签为 $\hat{\mathbf{y}} = (\hat{y}_1, \ldots, \hat{y}_n)$。考虑所有 $\binom{n}{2}$ 对样本对 $(i, j)$，每对可按在两种分区中的"同/异"关系分为四类：

$$
\begin{aligned}
a &= |\{(i,j) : y_i = y_j \text{ 且 } \hat{y}_i = \hat{y}_j\}| & (\text{真同-预同}) \\
b &= |\{(i,j) : y_i = y_j \text{ 且 } \hat{y}_i \neq \hat{y}_j\}| & (\text{真同-预异}) \\
c &= |\{(i,j) : y_i \neq y_j \text{ 且 } \hat{y}_i = \hat{y}_j\}| & (\text{真异-预同}) \\
d &= |\{(i,j) : y_i \neq y_j \text{ 且 } \hat{y}_i \neq \hat{y}_j\}| & (\text{真异-预异})
\end{aligned} \tag{5.70}
$$

其中 $a + b + c + d = \binom{n}{2}$。

{% block type="definition" title="定义 5.26　Rand Index 与 Adjusted Rand Index（ARI）" %}
**Rand Index**（RI）衡量两种分区的一致性比例：

$$
\text{RI} = \frac{a + d}{\binom{n}{2}} \tag{5.71}
$$

RI 的取值范围为 $[0, 1]$，但即使在随机分区下，RI 也可能取得较高的值（尤其当簇数较多时）。为此，引入**校正形式**。

设列联表 $[n_{ij}]$ 为真实簇 $i$ 与预测簇 $j$ 的交叉计数，$n_{i\cdot} = \sum_j n_{ij}$，$n_{\cdot j} = \sum_i n_{ij}$。**Adjusted Rand Index** 通过减去随机期望来校正：

$$
\text{ARI} = \frac{\sum_{ij} \binom{n_{ij}}{2} - \frac{\sum_i \binom{n_{i\cdot}}{2} \cdot \sum_j \binom{n_{\cdot j}}{2}}{\binom{n}{2}}}{\frac{1}{2}\left[\sum_i \binom{n_{i\cdot}}{2} + \sum_j \binom{n_{\cdot j}}{2}\right] - \frac{\sum_i \binom{n_{i\cdot}}{2} \cdot \sum_j \binom{n_{\cdot j}}{2}}{\binom{n}{2}}} \tag{5.72}
$$

ARI 的性质：
- $\text{ARI} = 1$ 当且仅当两种分区完全一致；
- $\text{ARI} \approx 0$ 当分区为随机独立时（期望值为 0）；
- ARI 可以为负值（比随机更差）。
{% endblock %}

### 5.6.2.2 基于信息论的指标

{% block type="definition" title="定义 5.27　归一化互信息（Normalized Mutual Information, NMI）" %}
设真实标签和聚类标签分别定义离散随机变量 $Y$ 和 $\hat{Y}$，其**互信息**为：

$$
\text{MI}(Y, \hat{Y}) = \sum_{i,j} \frac{n_{ij}}{n} \ln \frac{n_{ij} / n}{(n_{i\cdot}/n)(n_{\cdot j}/n)} \tag{5.73}
$$

互信息度量两个分区共享的信息量，但其值域依赖于簇数。**归一化互信息**消除这一依赖：

$$
\text{NMI}(Y, \hat{Y}) = \frac{2 \cdot \text{MI}(Y, \hat{Y})}{H(Y) + H(\hat{Y})} \tag{5.74}
$$

其中 $H(Y) = -\sum_i \frac{n_{i\cdot}}{n} \ln \frac{n_{i\cdot}}{n}$ 是分区 $Y$ 的**熵**。

NMI 的性质：
- $\text{NMI} \in [0, 1]$；
- $\text{NMI} = 1$ 当且仅当两种分区包含完全相同的信息；
- $\text{NMI} = 0$ 当两种分区完全独立。
{% endblock %}

### 5.6.2.3 ARI 与 NMI 的对比

| 特性 | ARI | NMI |
|------|-----|-----|
| 理论基础 | 组合计数（配对一致性） | 信息论（互信息） |
| 取值范围 | $[-1, 1]$（可为负） | $[0, 1]$ |
| 随机基线 | 期望值 = 0（已校正） | 期望值 > 0（未完全校正） |
| 对簇数敏感 | 较不敏感 | 簇数增多时基线上升 |
| 推荐场景 | 通用对比，尤其是不同 $K$ 值之间 | 信息论视角，同 $K$ 下对比 |

## 5.6.3 内部评估指标

当没有真实标签时（实际应用的常见情形），只能从数据本身的结构——**簇内紧凑性**（intra-cluster compactness）和**簇间分离度**（inter-cluster separation）——来评价聚类质量。

### 5.6.3.1 轮廓系数

{% block type="definition" title="定义 5.28　轮廓系数（Silhouette Coefficient）" %}
设样本 $\mathbf{x}_i$ 被分配到簇 $C_{k}$。定义：

**簇内平均距离**（紧凑性）：

$$
a(i) = \frac{1}{|C_{k}| - 1} \sum_{j \in C_{k}, j \neq i} d(\mathbf{x}_i, \mathbf{x}_j) \tag{5.75}
$$

**最近邻簇平均距离**（分离度）：

$$
b(i) = \min_{l \neq k} \frac{1}{|C_l|} \sum_{j \in C_l} d(\mathbf{x}_i, \mathbf{x}_j) \tag{5.76}
$$

样本 $\mathbf{x}_i$ 的**轮廓系数**为：

$$
s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}} \tag{5.77}
$$

整个聚类的轮廓系数为所有样本的均值：

$$
\bar{s} = \frac{1}{n} \sum_{i=1}^n s(i) \tag{5.78}
$$
{% endblock %}

{% block type="theorem" title="定理 5.11　轮廓系数的取值范围" %}
对任意样本 $\mathbf{x}_i$，其轮廓系数满足 $s(i) \in [-1, 1]$。

**证明**：

由定义，$a(i) \geq 0$ 且 $b(i) \geq 0$。分三种情况讨论：

**情况 1**：$a(i) < b(i)$（样本距簇内更近）。此时 $\max\{a(i), b(i)\} = b(i)$，

$$
s(i) = \frac{b(i) - a(i)}{b(i)} = 1 - \frac{a(i)}{b(i)} \in [0, 1)
$$

当 $a(i) \to 0$ 时 $s(i) \to 1$：样本位于簇的"中心"。

**情况 2**：$a(i) > b(i)$（样本距其他簇更近——可能被错误分配）。此时 $\max\{a(i), b(i)\} = a(i)$，

$$
s(i) = \frac{b(i) - a(i)}{a(i)} = \frac{b(i)}{a(i)} - 1 \in (-1, 0)
$$

当 $b(i) \to 0$ 时 $s(i) \to -1$：样本完全应属于另一个簇。

**情况 3**：$a(i) = b(i)$，此时 $s(i) = 0$，样本位于两个簇的边界。

综合三种情况，$s(i) \in [-1, 1]$。$\square$
{% endblock %}

{% hint style="info" %}
**轮廓系数的解读指南**：
- $\bar{s} > 0.7$：聚类结构强且清晰；
- $0.5 < \bar{s} < 0.7$：合理的聚类结构；
- $0.25 < \bar{s} < 0.5$：结构较弱，可能有重叠；
- $\bar{s} < 0.25$：几乎没有实质性的聚类结构。
{% endhint %}

### 5.6.3.2 Davies-Bouldin 指数

{% block type="definition" title="定义 5.29　Davies-Bouldin 指数（DBI）" %}
设共有 $K$ 个簇。定义簇 $C_k$ 的**散度**（scatter）和簇对之间的**中心距离**：

$$
S_k = \frac{1}{|C_k|} \sum_{i \in C_k} \|\mathbf{x}_i - \boldsymbol{\mu}_k\| \tag{5.79}
$$

$$
M_{kl} = \|\boldsymbol{\mu}_k - \boldsymbol{\mu}_l\| \tag{5.80}
$$

**Davies-Bouldin 指数**为：

$$
\text{DBI} = \frac{1}{K} \sum_{k=1}^K \max_{l \neq k} \frac{S_k + S_l}{M_{kl}} \tag{5.81}
$$

DBI 越小越好（簇内紧凑、簇间分离）。$\text{DBI} \geq 0$，理论下界为 0（完美聚类）。
{% endblock %}

### 5.6.3.3 Calinski-Harabasz 指数

{% block type="definition" title="定义 5.30　Calinski-Harabasz 指数（CH / Variance Ratio Criterion）" %}
定义**簇间离散度矩阵的迹**和**簇内离散度矩阵的迹**：

$$
\text{SSB} = \sum_{k=1}^K |C_k| \, \|\boldsymbol{\mu}_k - \boldsymbol{\mu}\|^2 \tag{5.82}
$$

$$
\text{SSW} = \sum_{k=1}^K \sum_{i \in C_k} \|\mathbf{x}_i - \boldsymbol{\mu}_k\|^2 \tag{5.83}
$$

其中 $\boldsymbol{\mu}$ 为全局均值。**Calinski-Harabasz 指数**为：

$$
\text{CH} = \frac{\text{SSB} / (K-1)}{\text{SSW} / (n-K)} \tag{5.84}
$$

CH 越大越好。它本质上是聚类版的**方差分析 F 统计量**——簇间方差与簇内方差之比。
{% endblock %}

### 5.6.3.4 三种内部指标的对比

| 指标 | 优化方向 | 计算复杂度 | 优势 | 局限 |
|------|---------|----------|------|------|
| 轮廓系数 | 越大越好 | $O(n^2)$ | 可视化（轮廓图）；逐样本分析 | 对凸形簇有偏好 |
| DBI | 越小越好 | $O(nK)$ | 计算高效 | 依赖质心，不适合任意形状簇 |
| CH | 越大越好 | $O(nK)$ | 计算高效；类似 F 检验 | 偏好球形簇 |

## 5.6.4 相对评估与最优 K 选择

### 5.6.4.1 Gap 统计量

{% block type="definition" title="定义 5.31　Gap 统计量" %}
Gap 统计量通过将观测数据的聚类质量与**均匀随机数据**的聚类质量进行比较，来判断是否存在真实的簇结构。

对给定的簇数 $K$，定义：

$$
W_K = \sum_{k=1}^K \frac{1}{2|C_k|} \sum_{i,j \in C_k} \|\mathbf{x}_i - \mathbf{x}_j\|^2 \tag{5.85}
$$

生成 $B$ 组与原数据同范围的均匀随机参考数据 $\{\mathbf{X}^{(b)}\}_{b=1}^B$，对每组计算 $W_K^{(b)}$。**Gap 统计量**为：

$$
\text{Gap}(K) = \frac{1}{B} \sum_{b=1}^B \ln W_K^{(b)} - \ln W_K \tag{5.86}
$$

选择满足以下条件的**最小 $K$**：

$$
\text{Gap}(K) \geq \text{Gap}(K+1) - s_{K+1} \tag{5.87}
$$

其中 $s_{K+1}$ 是 $\ln W_{K+1}^{(b)}$ 的标准差乘以 $\sqrt{1 + 1/B}$。
{% endblock %}

### 5.6.4.2 多指标综合决策

单一指标往往无法给出可靠结论。实践中建议：

1. **多指标投票**：同时计算轮廓系数、DBI、CH 指数（以及 BIC，若使用 GMM），取多数一致的 $K$ 值；
2. **可视化辅助**：轮廓图提供逐簇的分配质量信息；降维后散点图提供直观印象；
3. **领域知识约束**：某些应用有自然的 $K$ 范围——客户细分通常 $K \in [3, 8]$，基因表达模式分析可能 $K > 10$。

## 5.6.5 指标间的比较与选择指南

{% block type="theorem" title="定理 5.12　ARI 在随机分区下的期望" %}
若两个分区 $Y$ 和 $\hat{Y}$ 在保持各自边际簇大小的条件下**随机独立**，则 ARI 的期望值为 0：

$$
\mathbb{E}[\text{ARI}] = 0 \tag{5.88}
$$

**证明思路**：

ARI 的定义形式为 $\frac{\text{Index} - \text{Expected Index}}{\text{Max Index} - \text{Expected Index}}$，其中 Expected Index 是在超几何模型（Hubert-Arabie 模型）下的期望值。由构造，当分区独立时，分子 $\text{Index} - \text{Expected Index}$ 的期望为 0。$\square$
{% endblock %}

**选择指南**：

| 场景 | 推荐指标 | 原因 |
|------|---------|------|
| 有真实标签，学术评测 | ARI | 校正了随机基线，对不同 $K$ 公平 |
| 有真实标签，信息论视角 | NMI | 捕获非线性依赖关系 |
| 无标签，选择最优 $K$ | 轮廓系数 + CH + DBI | 三指标投票，互补 |
| 无标签，GMM 模型 | BIC（§5.5） | 统计理论支撑的模型选择 |
| 无标签，检验"是否存在簇结构" | Gap 统计量 | 与零假设（无结构）对比 |

```python
import numpy as np
from sklearn.metrics import (adjusted_rand_score, normalized_mutual_info_score,
                             silhouette_score, davies_bouldin_score,
                             calinski_harabasz_score)

def evaluate_clustering(X, labels, y_true=None):
    """对聚类结果进行全面评估"""
    results = {}
    # 内部指标（始终可用）
    if len(set(labels)) > 1:  # 需要至少 2 个簇
        results['轮廓系数'] = silhouette_score(X, labels)
        results['DBI'] = davies_bouldin_score(X, labels)
        results['CH'] = calinski_harabasz_score(X, labels)
    # 外部指标（需要真实标签）
    if y_true is not None:
        results['ARI'] = adjusted_rand_score(y_true, labels)
        results['NMI'] = normalized_mutual_info_score(y_true, labels)
    return results
```

## 案例 5.6：多算法多数据集的系统评估

**问题背景**：本案例在 3 种不同结构的合成数据集上（球形簇、月牙形、不同密度的簇）运行 4 种聚类算法（K-means、DBSCAN、层次聚类、GMM），使用外部和内部指标进行系统评估。目标是展示不同指标的**一致性与分歧**——没有任何单一指标在所有场景下都是最佳选择。

```python
import numpy as np
from sklearn.datasets import make_blobs, make_moons
from sklearn.cluster import KMeans, DBSCAN, AgglomerativeClustering
from sklearn.mixture import GaussianMixture
from sklearn.metrics import (adjusted_rand_score, normalized_mutual_info_score,
                             silhouette_score, davies_bouldin_score,
                             calinski_harabasz_score)
from sklearn.preprocessing import StandardScaler

np.random.seed(42)

# ── 1. 生成三种数据集 ─────────────────────────────────────
datasets = {}
# 球形簇
X1, y1 = make_blobs(n_samples=500, centers=3, cluster_std=0.8, random_state=42)
datasets['球形簇'] = (X1, y1)
# 月牙形
X2, y2 = make_moons(n_samples=500, noise=0.05, random_state=42)
datasets['月牙形'] = (X2, y2)
# 不同密度
X3a = np.random.multivariate_normal([0, 0], [[0.3, 0], [0, 0.3]], 200)
X3b = np.random.multivariate_normal([5, 5], [[2.0, 0], [0, 2.0]], 300)
X3 = np.vstack([X3a, X3b])
y3 = np.array([0]*200 + [1]*300)
datasets['不同密度'] = (X3, y3)

# ── 2. 定义算法 ──────────────────────────────────────────
def get_algorithms(n_clusters):
    return {
        'K-means': KMeans(n_clusters=n_clusters, n_init=10, random_state=42),
        'DBSCAN': DBSCAN(eps=0.5, min_samples=5),
        'AgglomerativeClustering': AgglomerativeClustering(n_clusters=n_clusters),
        'GMM': GaussianMixture(n_components=n_clusters, n_init=5, random_state=42),
    }

# ── 3. 系统评估 ──────────────────────────────────────────
for ds_name, (X, y_true) in datasets.items():
    n_true = len(set(y_true))
    X_scaled = StandardScaler().fit_transform(X)
    print(f"\n{'='*60}")
    print(f"数据集：{ds_name}（真实簇数={n_true}）")
    print(f"{'='*60}")
    print(f"{'算法':<25} {'ARI':>6} {'NMI':>6} {'轮廓':>6} {'DBI':>6} {'CH':>8}")
    print('-' * 60)
    for algo_name, algo in get_algorithms(n_true).items():
        if algo_name == 'GMM':
            labels = algo.fit(X_scaled).predict(X_scaled)
        else:
            labels = algo.fit_predict(X_scaled)
        # 跳过所有噪声点的情况
        if len(set(labels)) <= 1 or (len(set(labels)) == 2 and -1 in labels
                                     and sum(labels != -1) < 10):
            print(f"{algo_name:<25} {'N/A':>6} {'N/A':>6} {'N/A':>6} {'N/A':>6} {'N/A':>8}")
            continue
        # 计算指标（内部指标需排除噪声点）
        mask = labels != -1
        ari = adjusted_rand_score(y_true[mask], labels[mask])
        nmi = normalized_mutual_info_score(y_true[mask], labels[mask])
        if sum(mask) > 1 and len(set(labels[mask])) > 1:
            sil = silhouette_score(X_scaled[mask], labels[mask])
            dbi = davies_bouldin_score(X_scaled[mask], labels[mask])
            ch = calinski_harabasz_score(X_scaled[mask], labels[mask])
        else:
            sil, dbi, ch = float('nan'), float('nan'), float('nan')
        print(f"{algo_name:<25} {ari:6.3f} {nmi:6.3f} {sil:6.3f} {dbi:6.3f} {ch:8.1f}")
```

**关键观察**：

1. **球形簇**：所有算法和所有指标一致表现优秀——K-means 和 GMM 的 ARI ≈ 1.0，轮廓系数高，DBI 低；
2. **月牙形数据**：K-means 和 GMM 的 ARI 接近 0（无法发现月牙形结构），但**轮廓系数可能不低**（因为 K-means 的簇是凸的，内部指标偏好凸形簇）；DBSCAN 的 ARI 高，但轮廓系数可能较低——**内部指标与外部指标的冲突**；
3. **不同密度**：DBSCAN 需要仔细调参才能处理不同密度的簇，K-means 表现稳健。

**启示**：在没有真实标签的实际应用中，**仅依赖内部指标可能误导决策**——内部指标偏好球形、等大的簇，对非凸结构不公平。此时需要结合可视化和领域知识。

## 本节小结

| 关键词 | 核心概念 |
|--------|---------|
| 外部评估 | 有真实标签时使用，衡量聚类与真实分区的一致性 |
| ARI（定义 5.26） | $\frac{\text{Index} - \text{E[Index]}}{\text{Max} - \text{E[Index]}}$，校正随机基线，期望值为 0 |
| NMI（定义 5.27） | $\frac{2 \cdot \text{MI}}{H(Y) + H(\hat{Y})}$，信息论视角，$\in [0,1]$ |
| 轮廓系数（定义 5.28） | $s(i) = \frac{b(i)-a(i)}{\max\{a(i),b(i)\}}$，逐样本评估，$\in [-1,1]$（定理 5.11） |
| DBI（定义 5.29） | $\frac{1}{K}\sum_k \max_{l\neq k}\frac{S_k+S_l}{M_{kl}}$，越小越好 |
| CH 指数（定义 5.30） | $\frac{\text{SSB}/(K-1)}{\text{SSW}/(n-K)}$，越大越好，聚类版 F 统计量 |
| Gap 统计量（定义 5.31） | 与随机参考比较，检验"是否存在簇结构" |
| ARI 随机期望（定理 5.12） | 随机独立分区下 $\mathbb{E}[\text{ARI}] = 0$ |
| 指标冲突 | 内部指标偏好凸形簇，可能与真实非凸结构矛盾 |

### B 级（数学推导）

**B1.** 证明轮廓系数 $s(i) = 1 - a(i)/b(i)$（当 $a(i) < b(i)$ 时）。当簇内只有 $\mathbf{x}_i$ 一个样本时（$|C_k| = 1$），如何定义 $a(i)$？此时 $s(i)$ 应取什么值？

**B2.** 推导 Calinski-Harabasz 指数的分解。证明 SST = SSB + SSW，其中 $\text{SST} = \sum_{i=1}^n \|\mathbf{x}_i - \boldsymbol{\mu}\|^2$ 为总离散度。（提示：在 $\mathbf{x}_i - \boldsymbol{\mu} = (\mathbf{x}_i - \boldsymbol{\mu}_k) + (\boldsymbol{\mu}_k - \boldsymbol{\mu})$ 中展开平方和并利用 $\sum_{i \in C_k}(\mathbf{x}_i - \boldsymbol{\mu}_k) = \mathbf{0}$。）

**B3.** 证明 Davies-Bouldin 指数 $\text{DBI} \geq 0$。在什么条件下 $\text{DBI} = 0$？这个下界在实际中是否可达？

**B4.** 对于 ARI 公式 (5.72)，令 $a = \sum_{ij}\binom{n_{ij}}{2}$，$b = \sum_i\binom{n_{i\cdot}}{2}$，$c = \sum_j\binom{n_{\cdot j}}{2}$。将 ARI 改写为更紧凑的形式 $\text{ARI} = \frac{a - bc/\binom{n}{2}}{\frac{1}{2}(b+c) - bc/\binom{n}{2}}$，并验证当两个分区完全一致时 ARI = 1。
