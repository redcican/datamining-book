# 5.5 基于模型的聚类方法

前四节的聚类方法——K-means、DBSCAN、层次聚类、网格聚类——都属于**启发式方法**：它们基于距离、密度或网格统计量来发现簇结构，但**没有为数据生成过程建立概率模型**。这意味着这些方法：(1) 只能给出"样本属于哪个簇"的**硬分配**结果，无法量化"属于该簇的概率有多大"；(2) 不能回答"这个聚类结果在多大程度上能解释数据"这一根本问题；(3) 难以系统地比较不同 $K$ 值的模型质量。

**基于模型的聚类**（Model-based Clustering）从根本上改变视角：假设数据由一个**概率混合模型**生成——每个簇对应混合模型的一个**成分分布**（component），聚类问题就等价于**推断每个样本来自哪个成分**。这一框架下，聚类结果不仅包含簇归属，还包含归属概率、簇的形状参数（均值、协方差）和模型质量指标（似然值），为聚类分析提供了完整的统计推断基础。

**高斯混合模型**（Gaussian Mixture Model, GMM）是最经典、最广泛使用的概率聚类模型，而 **EM 算法**（Expectation-Maximization）是求解 GMM 参数的标准方法。

## 5.5.1 从硬聚类到概率生成模型

### 5.5.1.1 K-means 的概率解读

回顾 K-means（§5.1）：每个样本被**硬分配**到距离最近的质心所在簇，整个过程没有涉及概率。但如果我们换一种方式来看——**假设每个簇是一个高斯分布**，那么 K-means 的赋值步可以重新解读为：

$$
\gamma(\mathbf{x}_i) = \arg\max_k \; \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_k, \sigma^2 \mathbf{I}) \tag{5.54}
$$

即将样本分配到**使其概率密度最大**的那个高斯成分。当所有成分共享相同的球形协方差 $\sigma^2 \mathbf{I}$，且成分权重相等时，上式等价于最近质心赋值。

但现实数据的簇往往不是等大小、等方差的球形——它们可能是**椭圆形**（不同方向的方差不同）、**大小悬殊**（某些簇的样本数远多于其他簇）、**彼此重叠**（簇边界模糊，样本可能同时属于多个簇）。这些情形超出了 K-means 的建模能力。

### 5.5.1.2 概率生成模型的优势

| 维度 | K-means（硬聚类） | GMM（软聚类） |
|------|-----------------|-------------|
| 输出 | 簇标签（0/1） | **归属概率**（$[0,1]$） |
| 簇形状 | 球形（各向同性） | **椭圆形**（任意协方差矩阵） |
| 簇大小 | 偏好等大 | **不同权重**（$\pi_k$ 不等） |
| 重叠处理 | 硬分割 | **概率量化不确定性** |
| 模型质量 | 仅 SSE（无统计意义） | **似然值**（可比较不同 $K$） |
| $K$ 选择 | Elbow/轮廓系数（启发式） | **BIC/AIC**（统计准则） |

GMM 将聚类从"几何划分"提升为"统计推断"，是连接无监督学习与概率统计的桥梁。

## 5.5.2 高斯混合模型的数学框架

{% block type="definition" title="定义 5.23　高斯混合模型（Gaussian Mixture Model, GMM）" %}
设数据集 $\mathcal{D} = \{\mathbf{x}_1, \ldots, \mathbf{x}_n\}$，$\mathbf{x}_i \in \mathbb{R}^d$。**高斯混合模型**假设每个数据点由以下概率过程生成：

1. 以概率 $\pi_k$ 选择第 $k$ 个成分（$k = 1, \ldots, K$），其中 $\pi_k \geq 0$，$\sum_{k=1}^K \pi_k = 1$；
2. 从第 $k$ 个成分的高斯分布 $\mathcal{N}(\boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k)$ 中采样 $\mathbf{x}$。

数据点 $\mathbf{x}$ 的**边际概率密度**为：

$$
p(\mathbf{x} \mid \boldsymbol{\Theta}) = \sum_{k=1}^K \pi_k \, \mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k) \tag{5.55}
$$

其中 $\mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}, \boldsymbol{\Sigma}) = \frac{1}{(2\pi)^{d/2}|\boldsymbol{\Sigma}|^{1/2}} \exp\!\left(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1}(\mathbf{x}-\boldsymbol{\mu})\right)$，参数集合 $\boldsymbol{\Theta} = \{(\pi_k, \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k)\}_{k=1}^K$。
{% endblock %}

{% block type="definition" title="定义 5.24　GMM 的对数似然函数" %}
给定独立同分布数据 $\mathcal{D}$，GMM 的**对数似然函数**为：

$$
\ell(\boldsymbol{\Theta}) = \sum_{i=1}^n \ln p(\mathbf{x}_i \mid \boldsymbol{\Theta}) = \sum_{i=1}^n \ln \left[\sum_{k=1}^K \pi_k \, \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k)\right] \tag{5.56}
$$

**极大似然估计**的目标是找到使 $\ell(\boldsymbol{\Theta})$ 最大的参数 $\boldsymbol{\Theta}^*$。
{% endblock %}

### 5.5.2.1 直接最优化的困难

为什么不能直接对 $\ell(\boldsymbol{\Theta})$ 求导、令梯度为零来求解？因为公式 (5.56) 中**对数内有求和**（$\ln \sum_k$），导致对 $\boldsymbol{\mu}_k$ 或 $\boldsymbol{\Sigma}_k$ 的偏导无法分离成封闭形式——各成分的参数耦合在一起，无法像单一高斯分布那样直接求解。

如果我们知道每个样本来自哪个成分（即**隐变量**已知），问题就简单了——只需对每个成分独立地做极大似然估计。EM 算法正是利用这一洞察，通过**交替估计隐变量和参数**来迭代逼近极大似然解。

## 5.5.3 EM 算法：期望最大化

### 5.5.3.1 隐变量与完整数据似然

引入**隐变量** $z_i \in \{1, \ldots, K\}$，表示样本 $\mathbf{x}_i$ 来自第 $z_i$ 个成分。若隐变量已知，**完整数据**的对数似然为：

$$
\ell_c(\boldsymbol{\Theta}) = \sum_{i=1}^n \left[\ln \pi_{z_i} + \ln \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_{z_i}, \boldsymbol{\Sigma}_{z_i})\right] \tag{5.57}
$$

由于对数直接作用于单个高斯密度（不再有 $\ln \sum$），$\ell_c$ 可以对各参数分别求导并得到封闭解。

### 5.5.3.2 E 步：计算后验概率（责任度）

由于隐变量 $z_i$ 未知，EM 算法在 E 步计算其**后验概率**——给定当前参数 $\boldsymbol{\Theta}^{(t)}$，样本 $\mathbf{x}_i$ 属于第 $k$ 个成分的概率：

$$
\gamma_{ik}^{(t)} \triangleq P(z_i = k \mid \mathbf{x}_i, \boldsymbol{\Theta}^{(t)}) = \frac{\pi_k^{(t)} \, \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_k^{(t)}, \boldsymbol{\Sigma}_k^{(t)})}{\sum_{j=1}^K \pi_j^{(t)} \, \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_j^{(t)}, \boldsymbol{\Sigma}_j^{(t)})} \tag{5.58}
$$

$\gamma_{ik}$ 称为**责任度**（responsibility）：第 $k$ 个成分对"解释"样本 $\mathbf{x}_i$ 应承担的"责任"比例。它是贝叶斯定理的直接应用。

### 5.5.3.3 M 步：更新参数

利用 E 步计算的责任度 $\gamma_{ik}^{(t)}$ 作为"软权重"，对每个成分独立地更新参数：

**有效样本数**：

$$
N_k = \sum_{i=1}^n \gamma_{ik} \tag{5.59}
$$

**混合权重更新**：

$$
\pi_k^{(t+1)} = \frac{N_k}{n} \tag{5.60}
$$

**均值更新**：

$$
\boldsymbol{\mu}_k^{(t+1)} = \frac{1}{N_k} \sum_{i=1}^n \gamma_{ik} \, \mathbf{x}_i \tag{5.61}
$$

**协方差更新**：

$$
\boldsymbol{\Sigma}_k^{(t+1)} = \frac{1}{N_k} \sum_{i=1}^n \gamma_{ik} (\mathbf{x}_i - \boldsymbol{\mu}_k^{(t+1)})(\mathbf{x}_i - \boldsymbol{\mu}_k^{(t+1)})^\top \tag{5.62}
$$

### 5.5.3.4 EM 算法完整流程

```text
算法：EM for GMM(D, K, max_iter, tol)
──────────────────────────────────────────
输入：数据集 D = {x_1,...,x_n}，成分数 K，最大迭代次数，收敛阈值
输出：参数 Θ = {(π_k, μ_k, Σ_k)}_{k=1}^K，责任度矩阵 γ

1. 初始化：用 K-means 获得初始 μ_k，Σ_k = I，π_k = 1/K
2. for t = 1 to max_iter:
3.     E 步：对每个 i,k 计算 γ_ik（公式 (5.58)）
4.     M 步：更新 N_k, π_k, μ_k, Σ_k（公式 (5.59)–(5.62)）
5.     计算对数似然 ℓ(Θ^(t))（公式 (5.56)）
6.     if |ℓ^(t) - ℓ^(t-1)| < tol: break
7. return Θ, γ
```

### 5.5.3.5 Python 实现

```python
import numpy as np
from scipy.stats import multivariate_normal

def gmm_em(X, K, max_iter=100, tol=1e-6):
    """高斯混合模型 EM 算法从零实现"""
    n, d = X.shape
    # ── 初始化（K-means 式）─────────────────────────────
    from sklearn.cluster import KMeans
    km = KMeans(n_clusters=K, n_init=5, random_state=42).fit(X)
    mu = km.cluster_centers_.copy()
    sigma = np.array([np.eye(d)] * K)
    pi = np.ones(K) / K
    gamma = np.zeros((n, K))
    log_likelihood_old = -np.inf
    for iteration in range(max_iter):
        # ── E 步：计算责任度 ──────────────────────────
        for k in range(K):
            gamma[:, k] = pi[k] * multivariate_normal.pdf(X, mu[k], sigma[k])
        gamma_sum = gamma.sum(axis=1, keepdims=True)
        gamma_sum = np.maximum(gamma_sum, 1e-300)  # 防止除零
        gamma /= gamma_sum
        # ── M 步：更新参数 ────────────────────────────
        N_k = gamma.sum(axis=0)
        for k in range(K):
            N = max(N_k[k], 1e-10)
            mu[k] = gamma[:, k] @ X / N
            diff = X - mu[k]
            sigma[k] = (gamma[:, k, None] * diff).T @ diff / N
            sigma[k] += 1e-6 * np.eye(d)  # 正则化防奇异
            pi[k] = N / n
        # ── 对数似然 ─────────────────────────────────
        log_likelihood = 0.0
        for k in range(K):
            log_likelihood += pi[k] * multivariate_normal.pdf(X, mu[k], sigma[k])
        log_likelihood = np.sum(np.log(np.maximum(log_likelihood, 1e-300)))
        if abs(log_likelihood - log_likelihood_old) < tol:
            break
        log_likelihood_old = log_likelihood
    return pi, mu, sigma, gamma, log_likelihood
```

## 5.5.4 EM 算法的收敛性

{% block type="theorem" title="定理 5.9　EM 算法的似然单调性（Monotonicity of EM）" %}
EM 算法的每次迭代都使观测数据的对数似然**不减**：

$$
\ell(\boldsymbol{\Theta}^{(t+1)}) \geq \ell(\boldsymbol{\Theta}^{(t)}) \tag{5.63}
$$

**证明思路**：

引入**证据下界**（Evidence Lower BOund, ELBO）。对任意关于隐变量 $z$ 的分布 $q(z)$，Jensen 不等式给出：

$$
\ell(\boldsymbol{\Theta}) = \sum_{i=1}^n \ln p(\mathbf{x}_i \mid \boldsymbol{\Theta}) \geq \sum_{i=1}^n \sum_{k=1}^K q(z_i = k) \ln \frac{p(\mathbf{x}_i, z_i = k \mid \boldsymbol{\Theta})}{q(z_i = k)} \triangleq \text{ELBO}(q, \boldsymbol{\Theta}) \tag{5.64}
$$

**E 步**选择 $q^{(t)}(z_i = k) = \gamma_{ik}^{(t)} = P(z_i = k \mid \mathbf{x}_i, \boldsymbol{\Theta}^{(t)})$，此时 ELBO 恰好等于 $\ell(\boldsymbol{\Theta}^{(t)})$（Jensen 不等式取等）。

**M 步**在固定 $q^{(t)}$ 下最大化 ELBO 关于 $\boldsymbol{\Theta}$ 的值，得到 $\boldsymbol{\Theta}^{(t+1)}$。由于 ELBO$(\boldsymbol{\Theta}^{(t+1)}) \geq$ ELBO$(\boldsymbol{\Theta}^{(t)}) = \ell(\boldsymbol{\Theta}^{(t)})$，且 ELBO 始终是 $\ell$ 的下界，有 $\ell(\boldsymbol{\Theta}^{(t+1)}) \geq$ ELBO$(\boldsymbol{\Theta}^{(t+1)}) \geq \ell(\boldsymbol{\Theta}^{(t)})$。$\square$

{% hint style="warning" %}
**局部最优警告**：定理 5.9 只保证 $\ell$ 单调不减，不保证收敛到全局最大值。与 K-means 类似，EM 对初始化敏感，实践中应使用多次随机重启（sklearn 中 `n_init=10`），或用 K-means 的结果作为初始化。
{% endhint %}
{% endblock %}

## 5.5.5 GMM 与 K-means 的深层联系

{% block type="theorem" title="定理 5.10　K-means 是 GMM 的极限情形" %}
当 GMM 的所有成分共享相同的球形协方差 $\boldsymbol{\Sigma}_k = \sigma^2 \mathbf{I}$，且 $\sigma \to 0$ 时，EM 算法退化为 K-means：

**E 步退化**：当 $\sigma \to 0$ 时，责任度 $\gamma_{ik}$ 趋向于 0-1 硬分配：

$$
\lim_{\sigma \to 0} \gamma_{ik} = \begin{cases} 1 & \text{若 } k = \arg\min_j \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2 \\ 0 & \text{否则} \end{cases} \tag{5.65}
$$

**M 步退化**：当 $\gamma_{ik} \in \{0, 1\}$ 时，均值更新公式 (5.61) 退化为 K-means 的质心更新公式 (5.5)：

$$
\boldsymbol{\mu}_k = \frac{\sum_{i: \gamma_{ik}=1} \mathbf{x}_i}{|\{i: \gamma_{ik}=1\}|} \tag{5.66}
$$

因此，**K-means 是 GMM 在各向同性、零方差极限下的特例**。
{% endblock %}

| 维度 | K-means | GMM |
|------|---------|-----|
| 分配方式 | 硬分配（$\gamma \in \{0,1\}$） | 软分配（$\gamma \in [0,1]$） |
| 簇形状 | 球形（$\sigma^2 \mathbf{I}$） | 椭圆形（任意 $\boldsymbol{\Sigma}_k$） |
| 簇权重 | 隐式等权 | 显式 $\pi_k$（可不等） |
| 目标函数 | SSE（公式 (5.2)） | 对数似然（公式 (5.56)） |
| 不确定性量化 | 无 | 归属概率 $\gamma_{ik}$ |
| $K$ 选择 | Elbow/轮廓系数 | BIC/AIC（统计准则） |
| 初始化 | K-means++ | K-means + EM |
| 计算代价 | $O(nKd \cdot T)$ | $O(nKd^2 \cdot T)$（协方差估计） |

## 5.5.6 模型选择：BIC 与 AIC

GMM 需要预先指定成分数 $K$。过少的 $K$ 无法捕获数据的多模态结构（欠拟合），过多的 $K$ 导致过拟合。BIC 和 AIC 提供了系统的模型选择准则。

{% block type="definition" title="定义 5.25　BIC 与 AIC 模型选择准则" %}
设模型的对数似然为 $\ell$，参数个数为 $p$，样本数为 $n$：

**贝叶斯信息准则**（Bayesian Information Criterion, BIC）：

$$
\text{BIC} = -2\ell + p \ln n \tag{5.67}
$$

**赤池信息准则**（Akaike Information Criterion, AIC）：

$$
\text{AIC} = -2\ell + 2p \tag{5.68}
$$

选择使 BIC（或 AIC）**最小**的 $K$。对于 $K$ 个成分、$d$ 维数据的完整 GMM，参数个数为：

$$
p = K\left(\frac{d(d+1)}{2} + d + 1\right) - 1 \tag{5.69}
$$

（每个成分：$d(d+1)/2$ 个协方差参数 + $d$ 个均值参数 + 1 个权重参数；减 1 因为权重和为 1 的约束。）
{% endblock %}

BIC 对模型复杂度的惩罚更重（$p \ln n$ vs $2p$），倾向于选择更简单的模型，在样本量较大时更可靠；AIC 更宽松，可能选择成分数稍多的模型。**实践中推荐 BIC**。

```python
from sklearn.mixture import GaussianMixture
import numpy as np

def select_K_by_bic(X, K_range=range(1, 11)):
    bics = []
    aics = []
    for k in K_range:
        gmm = GaussianMixture(n_components=k, n_init=5, random_state=42)
        gmm.fit(X)
        bics.append(gmm.bic(X))
        aics.append(gmm.aic(X))
    best_k_bic = list(K_range)[np.argmin(bics)]
    best_k_aic = list(K_range)[np.argmin(aics)]
    return best_k_bic, best_k_aic, bics, aics
```

## 5.5.7 协方差结构的选择

GMM 的灵活性在很大程度上来自协方差矩阵的建模方式。不同的约束条件导致不同数量的参数，影响模型容量和计算效率：

| 协方差类型 | 约束 | 每个成分参数数 | sklearn 参数 | 适用场景 |
|-----------|------|-------------|------------|---------|
| **完整**（full） | $\boldsymbol{\Sigma}_k$ 任意正定 | $d(d+1)/2$ | `covariance_type='full'` | 椭圆形簇，方向各异 |
| **对角**（diag） | $\boldsymbol{\Sigma}_k = \text{diag}(\sigma_{k1}^2, \ldots, \sigma_{kd}^2)$ | $d$ | `covariance_type='diag'` | 轴对齐椭圆簇 |
| **球形**（spherical） | $\boldsymbol{\Sigma}_k = \sigma_k^2 \mathbf{I}$ | $1$ | `covariance_type='spherical'` | 球形簇 |
| **绑定**（tied） | $\boldsymbol{\Sigma}_k = \boldsymbol{\Sigma}$（所有成分共享） | $d(d+1)/2$（共享） | `covariance_type='tied'` | 形状相同、位置不同的簇 |

**选择建议**：数据量充足时用 `full`（最灵活）；样本量小或维度高时用 `diag` 或 `tied`（减少参数防过拟合）；BIC 可用于自动选择协方差类型。

## 案例 5.5：Old Faithful 间歇泉的混合高斯建模

**问题背景**：Old Faithful 间歇泉数据集记录了 272 次喷发的持续时间和等待时间，是展示双峰分布和 GMM 软聚类的经典数据集。肉眼可见数据呈两团分布——短喷发/短等待和长喷发/长等待，但两团之间存在重叠区域，硬聚类无法反映这种模糊性。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.mixture import GaussianMixture
from sklearn.preprocessing import StandardScaler
from matplotlib.patches import Ellipse

# ── 1. 加载数据 ───────────────────────────────────────────
# Old Faithful 数据（内置于多个 R/Python 包中）
# 此处使用 sklearn 模拟等效数据
np.random.seed(42)
# 成分 1：短喷发
n1 = 160
X1 = np.random.multivariate_normal([2.0, 54], [[0.07, 0.4], [0.4, 34]], n1)
# 成分 2：长喷发
n2 = 112
X2 = np.random.multivariate_normal([4.3, 80], [[0.17, 0.9], [0.9, 37]], n2)
X = np.vstack([X1, X2])
np.random.shuffle(X)
feature_names = ['喷发持续时间（分钟）', '等待时间（分钟）']

# ── 2. BIC 选择 K ─────────────────────────────────────────
K_range = range(1, 7)
bics = []
for k in K_range:
    gmm = GaussianMixture(n_components=k, n_init=10, random_state=42)
    gmm.fit(X)
    bics.append(gmm.bic(X))
best_K = list(K_range)[np.argmin(bics)]
print(f"BIC 最优 K = {best_K}")

# ── 3. GMM 拟合（K=2）────────────────────────────────────
gmm = GaussianMixture(n_components=2, n_init=10, random_state=42)
gmm.fit(X)
labels = gmm.predict(X)
proba = gmm.predict_proba(X)

print(f"\n成分权重：{gmm.weights_.round(3)}")
for k in range(2):
    print(f"成分 {k+1}：μ = {gmm.means_[k].round(2)}，"
          f"Σ 对角线 = {np.diag(gmm.covariances_[k]).round(2)}")

# ── 4. 不确定性分析 ───────────────────────────────────────
# 找出最"不确定"的样本（两个成分的责任度最接近 0.5）
uncertainty = 1 - np.max(proba, axis=1)
top_uncertain = np.argsort(uncertainty)[-5:]
print(f"\n最不确定的 5 个样本（不确定度）：")
for idx in top_uncertain:
    print(f"  样本 {idx}: 喷发={X[idx,0]:.2f}min, 等待={X[idx,1]:.0f}min, "
          f"P(成分1)={proba[idx,0]:.3f}, P(成分2)={proba[idx,1]:.3f}")

# ── 5. 可视化 ─────────────────────────────────────────────
fig, axes = plt.subplots(1, 2, figsize=(14, 6))
# 左：软聚类散点图 + 等高线
colors = proba[:, 1]  # 用第2成分的责任度作为颜色渐变
sc = axes[0].scatter(X[:, 0], X[:, 1], c=colors, cmap='coolwarm',
                     s=30, alpha=0.7, edgecolors='white', linewidths=0.3)
plt.colorbar(sc, ax=axes[0], label='P(长喷发成分)')
# 绘制等高线
x_grid = np.linspace(X[:, 0].min()-0.5, X[:, 0].max()+0.5, 100)
y_grid = np.linspace(X[:, 1].min()-5, X[:, 1].max()+5, 100)
XX, YY = np.meshgrid(x_grid, y_grid)
grid_pts = np.column_stack([XX.ravel(), YY.ravel()])
Z = -gmm.score_samples(grid_pts).reshape(XX.shape)
axes[0].contour(XX, YY, Z, levels=10, colors='gray', linewidths=0.5, alpha=0.5)
axes[0].set_title('GMM 软聚类（颜色=归属概率）')
axes[0].set_xlabel(feature_names[0])
axes[0].set_ylabel(feature_names[1])
# 右：BIC 曲线
axes[1].bar(list(K_range), bics, color=['#dc2626' if k == best_K else '#94a3b8'
            for k in K_range])
axes[1].set_xlabel('成分数 K')
axes[1].set_ylabel('BIC')
axes[1].set_title(f'BIC 模型选择（最优 K={best_K}）')
plt.tight_layout()
plt.show()
```

**业务解读**：

- **两个成分清晰分离**：GMM 识别出短喷发（~2 min，等待~54 min）和长喷发（~4.3 min，等待~80 min）两种模式，与地质学家的观测一致；
- **重叠区域的不确定性**：喷发时间约 3.0–3.5 分钟的样本，两个成分的责任度接近 0.5，表明这些喷发介于"短"和"长"之间——K-means 会将它们硬划入一个簇，丢失这一重要信息；
- **BIC 一致**：$K=2$ 时 BIC 最小，与物理背景（两种喷发模式）一致；$K=3$ 的 BIC 明显上升，说明数据不支持更复杂的模型；
- **预测应用**：公园管理员可利用 GMM 预测"下次喷发等待时间"——根据当前喷发持续时间，用 $\gamma_{ik}$ 加权两个成分的条件均值，给出概率化的等待时间预测。

## 本节小结

| 关键词 | 核心概念 |
|--------|---------|
| 高斯混合模型（定义 5.23） | $p(\mathbf{x}) = \sum_k \pi_k \mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k)$，每个簇对应一个高斯成分 |
| 对数似然（定义 5.24） | $\ell = \sum_i \ln[\sum_k \pi_k \mathcal{N}_k]$，$\ln \sum$ 结构导致无封闭解 |
| EM 算法 E 步 | 计算责任度 $\gamma_{ik} = P(z_i=k \mid \mathbf{x}_i, \boldsymbol{\Theta})$（公式 (5.58)），贝叶斯定理 |
| EM 算法 M 步 | 加权极大似然更新 $\pi_k, \boldsymbol{\mu}_k, \boldsymbol{\Sigma}_k$（公式 (5.59)–(5.62)） |
| 似然单调性（定理 5.9） | $\ell(\boldsymbol{\Theta}^{(t+1)}) \geq \ell(\boldsymbol{\Theta}^{(t)})$，基于 ELBO 证明 |
| K-means 是 GMM 的极限（定理 5.10） | $\boldsymbol{\Sigma}_k = \sigma^2\mathbf{I}$，$\sigma \to 0$ 时软分配退化为硬分配 |
| BIC/AIC（定义 5.25） | $\text{BIC} = -2\ell + p\ln n$，惩罚模型复杂度，选最小值对应的 $K$ |
| 协方差类型 | full/diag/spherical/tied，控制模型灵活性与参数量的权衡 |
| 软聚类 | $\gamma_{ik} \in [0,1]$ 量化归属不确定性，区别于 K-means 的 0/1 硬分配 |

### B 级（数学推导）

**B1.** 推导 EM 算法 M 步中均值更新公式 (5.61)：对完整数据对数似然 $\ell_c$（公式 (5.57)），以 $\gamma_{ik}$ 替代硬分配，对 $\boldsymbol{\mu}_k$ 求偏导并令其为零，得到加权均值。

**B2.** 推导协方差更新公式 (5.62)：类似 B1，对 $\boldsymbol{\Sigma}_k$ 求偏导（利用矩阵求导公式 $\frac{\partial}{\partial \boldsymbol{\Sigma}} \ln|\boldsymbol{\Sigma}| = \boldsymbol{\Sigma}^{-1}$ 和 $\frac{\partial}{\partial \boldsymbol{\Sigma}} \mathbf{a}^\top \boldsymbol{\Sigma}^{-1} \mathbf{a} = -\boldsymbol{\Sigma}^{-1}\mathbf{a}\mathbf{a}^\top\boldsymbol{\Sigma}^{-1}$），得到加权协方差。

**B3.** 证明定理 5.10（K-means 是 GMM 的极限）的细节：当 $\boldsymbol{\Sigma}_k = \sigma^2 \mathbf{I}$ 且 $\sigma \to 0$ 时，(1) 证明 $\gamma_{ik}$ 退化为 0-1 指示函数（利用 softmax 在温度 $\to 0$ 时的极限行为）；(2) 证明此时对数似然（忽略常数项）正比于 K-means 目标函数 $J$ 的负值。

**B4.** GMM 参数个数公式 (5.69) 的推导：完整协方差矩阵 $\boldsymbol{\Sigma}_k$ 是 $d \times d$ 对称正定矩阵，其自由参数个数为 $d(d+1)/2$。加上均值 $\boldsymbol{\mu}_k$ 的 $d$ 个参数和权重 $\pi_k$ 的 1 个参数（减去 $\sum \pi_k = 1$ 的约束），推导总参数个数。
