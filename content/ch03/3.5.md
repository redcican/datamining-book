# 3.5 神经网络分类方法

1986年，Rumelhart、Hinton 与 Williams 在《自然》杂志发表了反向传播算法，一举解决了多层神经网络的训练难题，开启了"联结主义"的黄金时代。然而神经网络随后经历了两个"寒冬"——计算力不足、数据匮乏，以及支持向量机（§3.3）等算法的强势崛起，使它在 1990 至 2000 年代几乎销声匿迹。直到 2012 年，Krizhevsky、Sutskever 与 Hinton 凭借 AlexNet 在 ImageNet 图像竞赛中将 Top-5 错误率从 26% 直降至 15.3%，深度神经网络才彻底复活，并在此后十年重塑了整个人工智能版图。

神经网络的核心思想异常简洁：**用可学习的线性变换与固定的非线性激活函数交替叠加，逼近任意复杂的输入-输出映射。** 单层线性分类器（感知机）只能处理线性可分问题；增加一个隐藏层，就能拟合任意光滑决策边界（通用近似定理）；继续堆叠层数，每一层都在上一层提取出的"特征"之上再提取更抽象的"特征"，从像素到边缘、从边缘到纹理、从纹理到物体部件……这种层次化的表示学习，是神经网络超越所有浅层方法的根本原因。

{% hint style="info" %}
📋 **本节学习目标**

完成本节学习后，读者应能够：

1. 描述单个人工神经元的计算模型：加权输入 $z = \mathbf{w}^\top\mathbf{x}+b$，经激活函数 $a = \sigma(z)$ 输出激活值，并解释激活函数为何必须非线性；
2. 推导多层感知机的前向传播公式 $\mathbf{z}^{(l)} = \mathbf{W}^{(l)}\mathbf{a}^{(l-1)}+\mathbf{b}^{(l)}$，$\mathbf{a}^{(l)}=\sigma(\mathbf{z}^{(l)})$，以及 Softmax 输出层与交叉熵损失的组合形式；
3. 用链式法则完整推导反向传播的误差信号传递公式，手动计算一个 2-2-1 网络的前向与反向过程；
4. 陈述通用近似定理（Cybenko, 1989）并正确理解其含义与局限——定理保证表达能力，而非可学习性；

6. 通过手写数字案例，比较 MLP、KNN、决策树、SVM 在相同数据上的性能，分析不同架构的过拟合风险，理解 L2 正则化与 Dropout 的作用机制。
{% endhint %}

## 3.5.1 动机：从线性到非线性

§3.3 中的支持向量机（SVM）以核技巧将数据映射到高维特征空间，从而拟合非线性边界。但这一做法有两个根本限制：核函数是人工选定的，无法从数据中学习；当类别结构极端复杂（如自然语言、图像）时，没有任何手工设计的核能覆盖所有有意义的特征交叉。

神经网络提供了一条完全不同的路径——**端到端的特征学习**（end-to-end learning）。网络的每一个隐藏层都可以理解为一个自动学习的特征提取器：给定足够的数据和足够的层数，网络会自主发现最有利于分类的特征表示，无需人工介入。这一能力源于以下观察：

若仅将线性变换叠加（$\mathbf{W}^{(2)}\mathbf{W}^{(1)}\mathbf{x}$），无论叠加多少层，结果仍等价于单个线性变换 $\mathbf{W}\mathbf{x}$，毫无新增表达能力。只有在每层线性变换之后插入一个**非线性激活函数**（例如 $\sigma$、ReLU），才能打破这种线性塌缩，使深层网络真正超越浅层。

{% hint style="ideology" %}
🧠 **核心哲学：表示学习**

传统机器学习的范式是"特征工程 + 浅层分类器"：人类专家手工设计特征（SIFT、HOG、词袋模型），再输入 SVM 或逻辑回归。深度神经网络的范式是"原始输入 + 深层网络"：网络自动从原始数据（像素、字符、传感器信号）中学习层次化特征表示。这一转变不仅是方法论的革命，也是对"什么是智能"这一问题的深刻回答——智能不是预设的规则，而是从数据中涌现的模式。
{% endhint %}

## 3.5.2 单个人工神经元

神经网络的基本构件是单个**人工神经元**（artificial neuron），其灵感来自生物神经元的"树突整合 → 轴突发射"机制。

{% block type="definition" label="3.26" title="人工神经元" %}
给定输入向量 $\mathbf{x} \in \mathbb{R}^d$，权重向量 $\mathbf{w} \in \mathbb{R}^d$ 与偏置 $b \in \mathbb{R}$，人工神经元的**预激活**（pre-activation）与**激活值**（activation）分别为：

$$
z = \mathbf{w}^\top \mathbf{x} + b \tag{3.56}
$$

$$
a = \sigma(z) \tag{3.57}
$$

其中 $\sigma : \mathbb{R} \to \mathbb{R}$ 为非线性**激活函数**。神经元的参数为 $(\mathbf{w}, b)$，通过训练从数据中学习。
{% endblock %}

方程 (3.56) 执行的是线性组合——对输入特征加权求和，加上偏置项；方程 (3.57) 将线性结果压缩或非线性变换。图 3.5.1 展示了单个神经元的完整计算图。偏置 $b$ 等价于一个恒为 $+1$ 的虚拟输入，权重为 $b$，这使得神经元可以平移激活阈值。

</figure>

{% block type="definition" label="3.27" title="激活函数" %}
**激活函数** $\sigma : \mathbb{R} \to \mathbb{R}$ 为神经元引入非线性。常用选择包括：

- **Sigmoid**：$\sigma(z) = \dfrac{1}{1+e^{-z}}$，值域 $(0,1)$，历史上广泛用于输出层的二分类概率。
- **Tanh**：$\tanh(z) = \dfrac{e^z - e^{-z}}{e^z + e^{-z}}$，值域 $(-1,1)$，输出以 0 为中心。
- **ReLU**（线性整流单元）：$f(z) = \max(0, z)$，值域 $[0, +\infty)$，正半轴梯度恒为 1，计算极简。
- **Leaky ReLU**：$f(z) = \max(0.1z, z)$，负半轴保留微小梯度，避免"死亡 ReLU"问题。
- **Softmax**（用于多分类输出层）：$\hat{p}_k = \dfrac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}$，将实数向量映射为概率分布。
{% endblock %}

图 3.5.2 展示了四种激活函数的形状与导数。Sigmoid 和 Tanh 在远离零点的区域斜率趋近于零，导致反向传播时梯度信号逐层衰减——即**梯度消失**问题。ReLU 在正半轴保持梯度 = 1，有效缓解了这一问题，因此成为深度网络隐藏层的默认选择。

<img src="/figures/ch03/fig3_5_02_activation_functions.png" alt="常用激活函数及其导数对比" style="max-width:100%;border-radius:6px;" />
<figcaption><strong>图 3.5.2</strong>　常用激活函数及其导数对比。上行：四种激活函数的函数形状——Sigmoid 输出 $(0,1)$，Tanh 输出 $(-1,1)$，ReLU 在正半轴为线性、负半轴置零，Leaky ReLU 负半轴保留斜率 0.1。下行：对应导数——Sigmoid 与 Tanh 在饱和区（$|x|>2$）导数趋近于零（梯度消失区域以阴影标注），ReLU 在正半轴导数恒为 1（不消失），负半轴为 0（死亡区域）。深度网络的隐藏层默认使用 ReLU 或其变体，输出层使用 Softmax。</figcaption>
</figure>

{% hint style="caution" %}
⚠️ **梯度消失 vs 死亡 ReLU**

Sigmoid/Tanh 在饱和区（$|z|$ 很大时）导数接近零，信号经多层传播后几乎消失——这是 1980 年代训练深层网络失败的主因。ReLU 解决了梯度消失，但引入了新问题：若一个神经元的输入始终为负，则其输出恒为零，梯度也恒为零，参数永远不更新——称为"死亡神经元"（dead neuron）。Leaky ReLU、ELU、GELU 等变体均旨在兼顾两者的优点。
{% endhint %}

## 3.5.3 多层感知机与前向传播

将多个神经元组织为**层**，多层堆叠即构成多层感知机。

{% block type="definition" label="3.28" title="多层感知机（MLP）" %}
**多层感知机**（Multilayer Perceptron, MLP）是一个有向无环图，由 $L$ 层组成：输入层（第 0 层）、若干隐藏层（第 1 至 $L-1$ 层）和输出层（第 $L$ 层）。每层与相邻层**全连接**（fully connected）。

设第 $l$ 层有 $n_l$ 个神经元，则该层的参数矩阵 $\mathbf{W}^{(l)} \in \mathbb{R}^{n_l \times n_{l-1}}$ 与偏置向量 $\mathbf{b}^{(l)} \in \mathbb{R}^{n_l}$ 定义如下**前向传播**（forward propagation）规则：

$$
\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)} \tag{3.58}
$$

$$
\mathbf{a}^{(l)} = \sigma\!\left(\mathbf{z}^{(l)}\right), \quad l = 1, \ldots, L-1
$$

其中 $\mathbf{a}^{(0)} = \mathbf{x}$ 为输入样本。输出层使用 Softmax 激活：

$$
\hat{p}_k = \frac{e^{z_k^{(L)}}}{\displaystyle\sum_{j=1}^{K} e^{z_j^{(L)}}}, \quad k = 1, \ldots, K \tag{3.59}
$$

整个网络的参数集合为 $\Theta = \{\mathbf{W}^{(1)},\mathbf{b}^{(1)},\ldots,\mathbf{W}^{(L)},\mathbf{b}^{(L)}\}$。
{% endblock %}

图 3.5.1(b) 展示了一个 $[3, 5, 4, 3]$ 的 MLP：输入维度 $d=3$，两个隐藏层（宽度分别为 5 和 4），输出类别数 $K=3$。每条连接对应权重矩阵中的一个元素，网络的总参数量为 $\sum_{l=1}^{L} (n_{l-1}+1) \times n_l$。

**交叉熵损失函数**（Cross-Entropy Loss）是多分类 MLP 的标准目标函数，由最大似然原理推导：

$$
\mathcal{L}(\Theta) = -\frac{1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} y_{ik} \log \hat{p}_{ik} \tag{3.60}
$$

其中 $y_{ik} = \mathbf{1}[y_i = k]$ 为 one-hot 标签。交叉熵损失是真实标签分布与预测分布之间的 KL 散度（加常数），最小化它等价于最大化样本的对数似然。

{% hint style="mathbg" %}
📐 **Softmax + 交叉熵的数值技巧**

直接计算 $e^{z_k}$ 在 $z_k$ 很大时会数值溢出。实际实现中，令 $c = \max_j z_j$，计算：

$$
\hat{p}_k = \frac{e^{z_k - c}}{\sum_j e^{z_j - c}}
$$

由于指数函数的平移不变性，此变换不改变结果，却将最大指数拉至 $e^0 = 1$，消除溢出风险。Softmax 与交叉熵联合使用时，对 $z_k$ 的梯度化简为 $\hat{p}_k - y_k$，形式极为简洁，是标准实现的出发点。
{% endhint %}

## 3.5.4 反向传播：用链式法则计算梯度

给定损失 $\mathcal{L}$，训练目标是计算 $\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(l)}}$ 和 $\frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(l)}}$，以便梯度下降更新参数。**反向传播**（backpropagation）是高效计算这些偏导数的动态规划算法，其本质是对计算图从输出层向输入层应用链式法则。

{% block type="definition" label="3.29" title="反向传播" %}
定义第 $l$ 层的**误差信号**（error signal）为：

$$
\boldsymbol{\delta}^{(l)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(l)}} \tag{3.61}
$$

反向传播由以下三步递推完成：

**步骤 1. 输出层误差**（Softmax + 交叉熵的联合梯度）：

$$
\boldsymbol{\delta}^{(L)} = \hat{\mathbf{p}} - \mathbf{y} \tag{3.62}
$$

**步骤 2. 隐藏层误差**（ReLU 激活的链式传递）：

$$
\boldsymbol{\delta}^{(l)} = \left(\mathbf{W}^{(l+1)}\right)^\top \boldsymbol{\delta}^{(l+1)} \odot \sigma'\!\left(\mathbf{z}^{(l)}\right) \tag{3.63}
$$

其中 $\odot$ 为逐元素乘法，$\sigma'(\mathbf{z}^{(l)}) = \mathbf{1}[\mathbf{z}^{(l)} > \mathbf{0}]$（ReLU 导数）。

**步骤 3. 参数梯度**：

$$
\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(l)}} = \boldsymbol{\delta}^{(l)} \left(\mathbf{a}^{(l-1)}\right)^\top, \qquad \frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(l)}} = \boldsymbol{\delta}^{(l)} \tag{3.64}
$$
{% endblock %}

公式 (3.63) 揭示了反向传播的直觉：来自上层的误差信号 $\boldsymbol{\delta}^{(l+1)}$ 经权重矩阵的转置"反向"传回当前层，再用激活函数导数"门控"——激活值接近零的神经元，梯度被遮断，不参与本轮更新。

{% block type="example" label="3.8" title="2-2-1 网络前向与反向手算" %}
考虑一个仅有一个隐藏层（2 个神经元）的网络，对单样本 $\mathbf{x} = (0.5,\, 0.8)^\top$，标签 $y = 1$（二分类，Sigmoid 输出 + 二元交叉熵）。

初始化权重（简化以便手算）：

$$
\mathbf{W}^{(1)} = \begin{pmatrix}0.4 & 0.6\\0.3 & 0.7\end{pmatrix},\quad
\mathbf{b}^{(1)} = \begin{pmatrix}0.1\\0.2\end{pmatrix},\quad
\mathbf{W}^{(2)} = (0.5,\, 0.9),\quad b^{(2)} = 0.3
$$

**前向传播**：

$$
\mathbf{z}^{(1)} = \mathbf{W}^{(1)}\mathbf{x} + \mathbf{b}^{(1)}
= \begin{pmatrix}0.4\times0.5+0.6\times0.8+0.1\\0.3\times0.5+0.7\times0.8+0.2\end{pmatrix}
= \begin{pmatrix}0.78\\0.91\end{pmatrix}
$$

$$
\mathbf{a}^{(1)} = \text{ReLU}(\mathbf{z}^{(1)}) = \begin{pmatrix}0.78\\0.91\end{pmatrix}
\quad(\text{均为正，不变})
$$

$$
z^{(2)} = 0.5\times0.78 + 0.9\times0.91 + 0.3 = 0.39+0.819+0.3 = 1.509
$$

$$
\hat{p} = \sigma(1.509) = \frac{1}{1+e^{-1.509}} \approx 0.819, \quad
\mathcal{L} = -\log(0.819) \approx 0.200
$$

**反向传播**：

$$
\delta^{(2)} = \hat{p} - y = 0.819 - 1 = -0.181
$$

$$
\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(2)}} = \delta^{(2)} \cdot (\mathbf{a}^{(1)})^\top
= -0.181 \times (0.78,\ 0.91) = (-0.141,\ -0.165)
$$

$$
\boldsymbol{\delta}^{(1)} = (\mathbf{W}^{(2)})^\top \delta^{(2)} \odot \mathbf{1}[\mathbf{z}^{(1)} > 0]
= \begin{pmatrix}-0.181\times0.5\\-0.181\times0.9\end{pmatrix}
= \begin{pmatrix}-0.0905\\-0.1629\end{pmatrix}
$$

$$
\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(1)}} = \boldsymbol{\delta}^{(1)}\mathbf{x}^\top
= \begin{pmatrix}-0.0453 & -0.0724\\-0.0815 & -0.1303\end{pmatrix}
$$

取学习率 $\eta = 0.5$，则 $\mathbf{W}^{(2)}$ 更新为 $(0.571,\ 0.982)$，损失随之从 0.200 降低。
{% endblock %}

## 3.5.5 优化：小批量随机梯度下降

整个训练集上计算梯度（批量梯度下降）在大数据集上不可行；仅用单样本（随机梯度下降）则噪声极大。实践中采用**小批量 SGD**（mini-batch SGD）：每次从训练集中随机抽取 $B$ 个样本（一个 mini-batch），用其近似估计整体梯度。

{% block type="algorithm" label="3.6" title="含反向传播的小批量 SGD" %}
**输入**：训练集 $\{(\mathbf{x}_i, y_i)\}_{i=1}^n$，网络架构 $[n_0, n_1, \ldots, n_L]$，学习率 $\eta$，批大小 $B$，总轮次 $T$，L2 正则系数 $\lambda$

**输出**：训练后的参数 $\Theta = \{\mathbf{W}^{(l)}, \mathbf{b}^{(l)}\}_{l=1}^L$

1. **初始化**：用 He 初始化 $\mathbf{W}^{(l)} \sim \mathcal{N}(0,\ 2/n_{l-1})$，$\mathbf{b}^{(l)} \leftarrow \mathbf{0}$
2. **对每轮** $t = 1, \ldots, T$：
   1. 随机打乱训练集索引
   2. **对每个 mini-batch** $\mathcal{B} \subset \{1,\ldots,n\}$，$|\mathcal{B}|=B$：
      1. **前向传播**：对批内每个样本，依次计算 $\mathbf{z}^{(l)}, \mathbf{a}^{(l)}$，得 $\hat{\mathbf{p}}_i$
      2. 计算批内平均交叉熵损失 $\mathcal{L}_\mathcal{B}$
      3. **反向传播**：依公式 (3.62)–(3.64) 计算各层参数梯度
      4. **参数更新**（含 L2 正则化权重衰减，见下方公式 3.65）
3. **返回** $\Theta$

**步骤 2.2.4 参数更新公式**：

$$
\mathbf{W}^{(l)} \leftarrow \mathbf{W}^{(l)} - \eta\!\left(\frac{\partial \mathcal{L}_\mathcal{B}}{\partial \mathbf{W}^{(l)}} + \lambda \mathbf{W}^{(l)}\right) \tag{3.65}
$$

$$
\mathbf{b}^{(l)} \leftarrow \mathbf{b}^{(l)} - \eta\,\frac{\partial \mathcal{L}_\mathcal{B}}{\partial \mathbf{b}^{(l)}}
$$

**时间复杂度**：每轮 $O\!\left(n \cdot \sum_l n_{l-1} n_l\right)$，内存 $O\!\left(B \cdot \sum_l n_l\right)$。
{% endblock %}

{% hint style="engineering" %}
⚙️ **优化器的工程演进**

朴素 SGD 对学习率敏感，且在损失曲面的"峡谷"方向容易震荡。工程实践常用以下改进：

- **Momentum**：引入"速度"累积历史梯度方向，$\mathbf{v} \leftarrow \beta \mathbf{v} + (1-\beta)\nabla\mathcal{L}$，$\mathbf{W} \leftarrow \mathbf{W} - \eta\mathbf{v}$
- **Adam**（Adaptive Moment Estimation）：同时估计梯度一阶矩（动量）和二阶矩（自适应学习率），几乎成为深度学习默认优化器
- **学习率调度**：余弦退火（Cosine Annealing）、Warmup + 衰减等策略，避免训练后期学习率过大导致震荡

本节从零实现采用朴素 mini-batch SGD，帮助理解梯度更新的核心机制；实际项目应直接使用 PyTorch 或 TensorFlow 的 Adam 优化器。
{% endhint %}

## 3.5.6 表达能力：通用近似定理

神经网络能逼近任意函数吗？Cybenko（1989）给出了肯定回答。

{% block type="theorem" label="3.12" title="通用近似定理（Cybenko, 1989）" %}
设 $\sigma : \mathbb{R} \to \mathbb{R}$ 为连续的有界非常数激活函数（如 Sigmoid），$I_d = [0,1]^d$ 为 $d$ 维单位超立方体。对任意连续函数 $f : I_d \to \mathbb{R}$ 与任意 $\varepsilon > 0$，存在整数 $N$、权重向量 $\{\mathbf{w}_j\}$、阈值 $\{\theta_j\}$ 和系数 $\{\alpha_j\}$，使得

$$
\left|F(\mathbf{x}) - f(\mathbf{x})\right| < \varepsilon \quad \forall\, \mathbf{x} \in I_d
$$

其中 $F(\mathbf{x}) = \sum_{j=1}^{N} \alpha_j \sigma(\mathbf{w}_j^\top \mathbf{x} + \theta_j)$ 为单隐藏层神经网络。

进而，对 ReLU 激活函数（Hornik, 1991），结论对所有 $L^p$ 空间中的函数同样成立。
{% endblock %}

定理 3.12 是神经网络合法性的理论基石，但存在两个至关重要的**误解防范点**：

**第一，定理只保证存在性，不保证可学习性。** 能逼近任意函数的网络宽度 $N$ 可能极大；在有限数据和有限计算下，SGD 不一定能找到这个全局最优解。

**第二，定理是关于宽度的，不是关于深度的。** 单隐藏层（足够宽）在理论上足够；但实践表明，使用多隐藏层（深度）可以用**指数级更少**的参数表达同等复杂的函数，这是深度网络优于浅层网络的核心优势。

## 3.5.7 过拟合与正则化

由于神经网络参数量通常远超训练样本数，过拟合是首要风险。图 3.5.3 展示了训练/测试损失曲线的典型过拟合模式——训练损失持续下降，而测试损失在某轮次后开始上升。

<figure>
<img src="/figures/ch03/fig3_5_03_training_curves.png" alt="MLP训练过程与过拟合诊断" style="max-width:100%;border-radius:6px;" />
<figcaption><strong>图 3.5.3</strong>　MLP 训练动态与正则化效果（make_moons 数据集，$n=600$，noise=0.28）。左图（a）：三种配置的训练/测试损失曲线——大模型（128-64-32，无正则，蓝色）训练损失持续下降但测试损失约在第 300 轮后上升（过拟合区域以粉色阴影标注）；L2 正则化模型（$\lambda=0.01$，青色）两条曲线保持平行；小模型（16-8，橙色）容量不足，损失较高但不过拟合。中图（b）：对应准确率曲线，过拟合模型训练准确率持续上升而测试准确率停滞。右图（c）：决策边界对比——过拟合模型边界扭曲，L2 正则化边界更平滑，泛化性更佳。</figcaption>
</figure>

**L2 正则化**（权重衰减，Weight Decay）在损失函数中加入参数范数惩罚：

$$
\mathcal{L}_{\text{reg}} = \mathcal{L} + \frac{\lambda}{2}\sum_{l}\|\mathbf{W}^{(l)}\|_F^2 \tag{3.66}
$$

这等价于在每次参数更新时额外将权重乘以 $(1-\eta\lambda)$（见算法 3.6 步骤 2.2.4），抑制权重绝对值过大。

**Dropout**（Srivastava et al., 2014）是神经网络特有的正则化技术：训练阶段，每个神经元以概率 $p$（保留率）独立保留，以概率 $1-p$ 随机置零；推断阶段，所有神经元参与计算，输出乘以 $p$ 保持期望值不变：

$$
\tilde{a}_j^{(l)} = \frac{m_j}{p} \cdot a_j^{(l)}, \quad m_j \sim \text{Bernoulli}(p) \tag{3.67}
$$

Dropout 的直觉：每次训练相当于对一个随机子网络进行更新，最终的网络等价于大量子网络的集成（Ensemble），从而大幅降低过拟合风险。

{% hint style="caution" %}
⚠️ **批归一化与早停**

除 L2 正则化和 Dropout 外，还有两种实践中极为重要的技术：

- **批归一化**（Batch Normalization, Ioffe & Szegedy, 2015）：在每层激活之前，将该层输出归一化为均值 0、方差 1，显著加快收敛，允许更大学习率
- **早停**（Early Stopping）：监控验证集损失，在其不再下降时提前终止训练，是最简单有效的过拟合防护。sklearn 的 `MLPClassifier` 通过 `early_stopping=True` 启用
{% endhint %}

## 3.5.8 案例：手写数字 MLP 分类

{% block type="case" label="3.5" title="MLP 手写数字识别与跨算法对比" %}
**数据集**：`sklearn.datasets.load_digits`，$n=1797$ 个样本，$d=64$ 个特征（$8\times8$ 像素灰度值），$K=10$ 类（0–9 数字）。按 75%/25% 分层划分训练/测试集，特征使用 `StandardScaler` 标准化。

**架构搜索**：对 8 种隐藏层配置（从单层 16 神经元到三层 256-128-64）使用 5 折交叉验证选择最优架构（见图 3.5.4(a)）。

```python
import numpy as np
from sklearn.datasets import load_digits
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.preprocessing import StandardScaler
from sklearn.neural_network import MLPClassifier
from sklearn.metrics import accuracy_score, confusion_matrix
# --- 1. 数据准备 ---
data = load_digits()
X, y = data.data, data.target
X_tr, X_te, y_tr, y_te = train_test_split(
    X, y, test_size=0.25, random_state=42, stratify=y)
scaler = StandardScaler()
X_tr_s = scaler.fit_transform(X_tr)
X_te_s  = scaler.transform(X_te)
# --- 2. 架构搜索 ---
configs = [(64,), (128,), (64, 32), (128, 64), (128, 64, 32)]
best_cfg, best_cv = None, 0
for cfg in configs:
    clf = MLPClassifier(hidden_layer_sizes=cfg, activation="relu",
                        solver="adam", max_iter=500, random_state=42)
    cv = cross_val_score(clf, X_tr_s, y_tr, cv=5, scoring="accuracy").mean()
    if cv > best_cv:
        best_cv, best_cfg = cv, cfg
# --- 3. 最优模型评估 ---
best_clf = MLPClassifier(hidden_layer_sizes=best_cfg, activation="relu",
                         solver="adam", max_iter=500, random_state=42)
best_clf.fit(X_tr_s, y_tr)
y_pred = best_clf.predict(X_te_s)
print(f"最优架构: {best_cfg}  测试准确率: {accuracy_score(y_te, y_pred):.3f}")
```

**从零实现 MLPScratch**（纯 NumPy，含 ReLU、Softmax、交叉熵、mini-batch SGD）：

```python
class MLPScratch:
    def __init__(self, hidden_sizes=(128,), lr=0.01, batch_size=64,
                 n_epochs=200, lam=1e-4, random_state=42):
        self.hidden_sizes = hidden_sizes
        self.lr = lr
        self.batch_size = batch_size
        self.n_epochs = n_epochs
        self.lam = lam
        self.rng = np.random.default_rng(random_state)
    def _init_params(self, input_dim, n_classes):
        sizes = [input_dim] + list(self.hidden_sizes) + [n_classes]
        self.W, self.b = [], []
        for i in range(len(sizes) - 1):
            std = np.sqrt(2.0 / sizes[i])      # He 初始化
            self.W.append(self.rng.normal(0, std, (sizes[i+1], sizes[i])))
            self.b.append(np.zeros(sizes[i+1]))
    def _relu(self, z):
        return np.maximum(0, z)
    def _softmax(self, z):
        z_shifted = z - z.max(axis=1, keepdims=True)
        exp_z = np.exp(z_shifted)
        return exp_z / exp_z.sum(axis=1, keepdims=True)
    def _forward(self, X):
        a = X
        caches = [(None, a)]
        for i, (W, b) in enumerate(zip(self.W, self.b)):
            z = a @ W.T + b
            a = self._relu(z) if i < len(self.W) - 1 else self._softmax(z)
            caches.append((z, a))
        return caches
    def _backward(self, caches, Y_onehot):
        grads_W = [None] * len(self.W)
        grads_b = [None] * len(self.b)
        B = Y_onehot.shape[0]
        delta = (caches[-1][1] - Y_onehot) / B      # 输出层误差 (3.62)
        for l in range(len(self.W) - 1, -1, -1):
            a_prev = caches[l][1]
            grads_W[l] = delta.T @ a_prev + self.lam * self.W[l]  # (3.64)
            grads_b[l] = delta.sum(axis=0)
            if l > 0:
                delta = (delta @ self.W[l]) * (caches[l][0] > 0)  # (3.63)
        return grads_W, grads_b
    def fit(self, X, y):
        n_classes = len(np.unique(y))
        self._init_params(X.shape[1], n_classes)
        Y_onehot = np.eye(n_classes)[y]
        idx = np.arange(len(X))
        for _ in range(self.n_epochs):
            self.rng.shuffle(idx)
            for start in range(0, len(X), self.batch_size):
                batch = idx[start:start + self.batch_size]
                caches = self._forward(X[batch])
                gW, gb = self._backward(caches, Y_onehot[batch])
                for l in range(len(self.W)):
                    self.W[l] -= self.lr * gW[l]      # (3.65)
                    self.b[l] -= self.lr * gb[l]
        return self
    def predict(self, X):
        return self._forward(X)[-1][1].argmax(axis=1)
    def score(self, X, y):
        return float(np.mean(self.predict(X) == y))
# --- 数值验证 ---
mlp_scratch = MLPScratch(hidden_sizes=(128,), lr=0.02,
                         batch_size=64, n_epochs=300, lam=1e-3)
mlp_scratch.fit(X_tr_s, y_tr)
scratch_acc = mlp_scratch.score(X_te_s, y_te)
sklearn_acc = accuracy_score(y_te, y_pred)
print(f"MLPScratch 准确率: {scratch_acc:.3f}  sklearn 准确率: {sklearn_acc:.3f}")
print(f"差值: {abs(scratch_acc - sklearn_acc):.4f}  (应 < 0.005)")
```

本节介绍的 MLP 是深度学习的基础砖块，但现代应用已演化出专门化的架构：

- **卷积神经网络**（CNN）：利用平移等变性，在空间局部感受野上共享权重，专为图像设计（LeNet → AlexNet → ResNet → EfficientNet）

上述架构均可视为"MLP + 特定归纳偏置"的变体，掌握 MLP 是理解一切深度学习的逻辑起点。

## 小结

本节从单个人工神经元出发，建立了多层感知机（MLP）的完整数学框架：前向传播（公式 3.56–3.60）将输入层层变换为类别概率；反向传播（公式 3.61–3.64）用链式法则高效计算每个参数的梯度；小批量 SGD（算法 3.6）在随机子集上迭代更新参数，兼顾效率与噪声。通用近似定理（定理 3.12）保证了 MLP 的理论表达能力，而 L2 正则化与 Dropout（公式 3.66–3.67）则是对抗过拟合的工程防线。

在与 KNN（§3.4）、决策树（§3.1）、SVM（§3.3）的横向比较中，MLP 在大多数任务上处于第一梯队，其核心优势——**从数据中端到端地学习层次化特征**——使它在图像、语音、文本等高维复杂数据上无可取代。

本章（§3.1–3.5）系统介绍了五类判别式分类器的统一框架：从决策树的规则划分、朴素贝叶斯的概率生成，到 SVM 的最大间隔超平面，再到神经网络的非线性表示学习。下一章（第 4 章）将转向回归问题，在监督学习的连续输出场景中建立同样完整的理论与工程闭环。

## 习题

### A 级（概念理解）

**A1.** 若将 MLP 所有激活函数替换为恒等函数 $\sigma(z) = z$，则无论网络多深多宽，其表达能力等价于什么？用矩阵代数证明。

**A2.** Sigmoid 激活在大 $|z|$ 时梯度趋近于零。若一个 5 层网络所有层均使用 Sigmoid，且每层激活的导数约为 0.2，第 1 层的梯度幅度是第 5 层的多少倍？结合这个计算解释"梯度消失"的危害。

**A3.** 陈述通用近似定理并指出其两个最常见的误解：定理是否意味着（a）任意宽度的单层网络都能在有限步骤内被 SGD 训练到全局最优？（b）更深的网络一定比更宽的单层网络准确率更高？

### B 级（公式推导）

**B1.** 推导 Softmax + 多分类交叉熵组合后，对第 $L$ 层预激活 $z_k^{(L)}$ 的梯度 $\frac{\partial \mathcal{L}}{\partial z_k^{(L)}} = \hat{p}_k - y_k$。（提示：分 $k = y$ 和 $k \neq y$ 两种情况讨论。）

**B2.** 对于批大小为 $B$ 的 mini-batch，写出反向传播的矩阵形式，说明如何将批内 $B$ 个样本的梯度同时计算（矩阵化实现），而非逐样本循环，并分析向量化对时间复杂度的影响。

**B3.** L2 正则化将更新规则变为 $\mathbf{W} \leftarrow (1-\eta\lambda)\mathbf{W} - \eta \nabla_\mathbf{W}\mathcal{L}$。从贝叶斯角度看，L2 正则化对应对权重施加何种先验分布？对应的 MAP 估计为什么等价于 L2 正则化的 MLE？

### C 级（编程实现）

**C1.** 在 §3.5.8 的 `MLPScratch` 基础上，添加 Dropout 支持：训练时以保留率 $p=0.8$ 随机 mask 隐藏层激活，推断时乘以 $p$。比较加 Dropout 前后在 make_moons（$n=300$，noise=0.4）上的测试准确率。

**C2.** 实现学习率指数衰减：$\eta_t = \eta_0 \cdot r^{t/T}$（$r=0.5$），并与固定学习率在 digits 数据集上对比损失曲线（横轴：epoch，纵轴：交叉熵损失）。

**C3.** 用 `sklearn.neural_network.MLPClassifier` 对 `load_breast_cancer` 数据集进行网格搜索（隐藏层大小 $\in \{(32,), (64,), (64,32)\}$，L2 正则 $\alpha \in \{0.001, 0.01, 0.1\}$），报告最优超参数组合及对应的测试 AUC。

### D 级（分析与比较）

**D1.** 对于 §3.5.8 的混淆矩阵，找出 MLP 最容易混淆的两对数字，说明可能的视觉原因。设计一种特征工程（例如：垂直对称性、像素密度分布）来区分这两对，并验证效果。

**D2.** 在 make_circles 数据集（$n=500$，noise=0.1）上，比较 1-NN KNN、RBF-SVM 和两层 MLP（(32, 16) 架构）的决策边界图形。哪种方法产生了最平滑的边界？在噪声增大到 0.3 时各自性能如何变化？

### E 级（开放探讨）

**E1.** 反向传播的计算图表示使得 PyTorch 等框架能自动求导（autograd）。解释：(a) 动态计算图（define-by-run）与静态图（define-and-run）的区别；(b) 为何动态图在研究阶段更受欢迎，以及其在部署阶段的潜在劣势。

**E2.** 神经网络的权重初始化至关重要：若全部初始化为零，会发生"对称性"问题。用数学证明：若所有 $\mathbf{W}^{(l)} = \mathbf{0}$，$\mathbf{b}^{(l)} = \mathbf{0}$，则每层内所有神经元在任意轮 SGD 后始终保持完全相同的权重，网络实际上等价于每层只有一个神经元，从而完全失去宽度带来的表达能力。
