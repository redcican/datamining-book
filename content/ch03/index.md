# 第三章 分类算法

分类是数据挖掘中最核心的监督学习任务：给定带标签的训练数据，学习一个映射函数将新样本分配到预定义类别。本章系统介绍六大类分类算法和完整的评估框架：§3.1 从信息增益到基尼不纯度，对比 ID3、C4.5 和 CART 三代决策树；§3.2 以 Bayes 定理为基础，推导朴素贝叶斯分类器的三种变体（高斯、多项式、伯努利）；§3.3 从最大间隔的几何直觉出发，严格推导 SVM 的对偶问题与核技巧；§3.4 介绍 KNN 的 Cover-Hart 理论保证和 KD 树加速；§3.5 从感知机到多层神经网络，详解反向传播算法；§3.6 介绍 Bagging、随机森林、AdaBoost、梯度提升和 Stacking 五种集成策略；§3.7 建立混淆矩阵、ROC/PR 曲线和交叉验证的系统评估框架。

## 本章学习目标

1. 掌握决策树的三种分裂准则（信息增益、增益率、基尼不纯度）及剪枝策略；
2. 理解贝叶斯分类的数学基础，掌握朴素贝叶斯的三种变体与拉普拉斯平滑；
3. 理解 SVM 最大间隔原理，掌握软间隔、对偶问题与核函数的数学推导；
4. 掌握 KNN 算法及其理论保证（Cover-Hart 定理），理解 KD 树加速原理；
5. 理解多层感知机的前向传播与反向传播算法；
6. 区分 Bagging 与 Boosting 的集成策略，掌握随机森林和梯度提升的原理；
7. 能使用混淆矩阵、F1、ROC-AUC、PR-AUC 和交叉验证系统评估分类器性能。
