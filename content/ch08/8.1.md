# 8.1 时间序列数据特征

前面七章讨论的数据挖掘方法，无论是分类、回归、聚类还是异常检测，都隐含一个共同假设：**样本之间是独立同分布的**（i.i.d.）。然而，大量真实数据天然具有**时间维度**——股票价格按分钟跳动、气温按小时变化、心电信号以毫秒级采样。在这些数据中，前后观测之间存在强烈的**时序依赖性**：今天的气温与昨天的气温高度相关，一分钟前的股价很大程度上决定了当前的股价。忽略这种依赖性而直接应用传统方法，不仅浪费了关键信息，还可能导致严重的建模偏差。

时间序列数据挖掘（Time Series Data Mining）正是为了应对这一挑战而发展起来的学科分支。它横跨统计学、信号处理和机器学习三个领域：统计学提供了自回归模型和平稳性理论（Box 和 Jenkins，1970），信号处理贡献了频域分析和小波变换，机器学习则引入了深度循环网络和 Transformer 架构。本章将系统介绍时间序列数据挖掘的核心方法。

本节作为全章的起点，建立时间序列分析的**数学基础**。我们将回答三个根本问题：(1) 时间序列数据与表格数据在数学上的本质区别是什么？(2) 如何将一条时间序列分解为可解释的基本成分？(3) 如何判断一条时间序列是否适合用简单模型建模？

## 8.1.1 时间序列的形式化定义

### 8.1.1.1 从表格数据到时间序列

在前面各章中，数据集被表示为样本矩阵 $$\mathbf{X} \in \mathbb{R}^{n \times d}$$，每一行 $$\mathbf{x}_i$$ 是一个独立观测。时间序列数据与此有两个根本区别：

- **有序性**：样本按时间排列，$$\mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_T$$ 的顺序不可打乱——打乱顺序的时间序列完全丧失信息；
- **依赖性**：相邻样本不独立，$$\mathbf{x}_t$$ 与 $$\mathbf{x}_{t-1}, \mathbf{x}_{t-2}, \ldots$$ 之间存在统计依赖关系。

{% block type="definition" title="定义 8.1　时间序列（Time Series）" %}
**时间序列**是一组按时间顺序排列的随机变量的实现值：

$$
\{x_t\}_{t=1}^{T} = \{x_1, x_2, \ldots, x_T\} \tag{8.1}
$$

其中每个 $$x_t$$ 是定义在概率空间 $$(\Omega, \mathcal{F}, P)$$ 上的随机变量 $$X_t$$ 的一次实现。完整的**随机过程**（Stochastic Process）为：

$$
\{X_t\}_{t \in \mathcal{T}} \tag{8.2}
$$

其中 $$\mathcal{T}$$ 为时间索引集。当 $$\mathcal{T} = \{1, 2, 3, \ldots\}$$ 时为**离散时间**序列，当 $$\mathcal{T} = [0, +\infty)$$ 时为**连续时间**过程。

时间序列 $$\{x_t\}$$ 是随机过程 $$\{X_t\}$$ 的**一条样本路径**（sample path）——这一区分至关重要：我们观测到的数据是一条路径，而统计推断的对象是背后的随机过程。
{% endblock %}

这一概率视角看似抽象，但对理解时间序列分析的基本困难至关重要：在表格数据中，我们有 $$n$$ 个独立样本来估计分布；在时间序列中，我们通常只有**一条**样本路径（一次实验），却要推断整个随机过程的性质。这意味着时间序列分析必须借助**时间维度上的规律性**（如平稳性）来弥补"样本量"的不足。

### 8.1.1.2 时间序列的分类

{% block type="definition" title="定义 8.2　时间序列的分类" %}
时间序列可从多个维度分类：

**按变量维度**：
- **单变量时间序列**（Univariate）：$$x_t \in \mathbb{R}$$，每个时刻只有一个观测值（如每日最高气温）；
- **多变量时间序列**（Multivariate）：$$\mathbf{x}_t \in \mathbb{R}^d$$，每个时刻有 $$d$$ 个同步观测值（如同时记录的温度、湿度、风速）。

**按采样间隔**：
- **等间隔**（Regular）：$$t_{i+1} - t_i = \Delta t$$ 恒定（如每小时一次的气温记录）；
- **不等间隔**（Irregular）：采样间隔不固定（如患者就诊记录、金融交易时间戳）。

**按值域**：
- **连续值**（Continuous-valued）：$$x_t \in \mathbb{R}$$（如股价、温度）；
- **离散值/符号**（Discrete/Symbolic）：$$x_t \in \Sigma$$（如 DNA 碱基序列 $$\{A, T, C, G\}$$、自然语言文本）。
{% endblock %}

本章主要讨论**等间隔单变量连续值**时间序列——这是最基础也是应用最广的类型。多变量时间序列可视为多条单变量时间序列的向量化推广，其核心方法（自相关分析、分解、预测）大多直接适用。不等间隔时间序列需要特殊处理（插值或基于事件的建模），超出本章范围。

{% hint style="think" %}
🤔 **思考：时间序列与序列数据**

时间序列是序列数据（Sequential Data）的一种特例。序列数据的更广义形式包括：文本（词序列）、DNA 序列、事件序列等。时间序列的特殊之处在于它具有**等间隔的度量时间轴**——这使得频域分析（傅里叶变换）、时间窗口操作和自回归建模成为可能。后续章节中，§8.3 的相似性度量和 §8.5 的分类方法对一般序列数据同样适用。
{% endhint %}

## 8.1.2 时间序列的成分分解

### 8.1.2.1 趋势、季节性与残差

直观地观察一条真实的时间序列（如图 8.1.1），通常可以辨别出几种叠加的模式：长期的上升或下降趋势（如全球气温逐年升高）、周期性波动（如气温的年周期）、以及不可预测的随机波动。将这些模式形式化地分离出来，是时间序列分析的第一步。

<figure>
<img src="/figures/ch08/fig8_1_01_ts_components.png" alt="时间序列成分分解" />
<figcaption>
<strong>图 8.1.1</strong>　时间序列的三成分分解示例。<em>顶部</em>：原始时间序列（含趋势、季节性和随机波动）；<em>下方三行</em>：分离出的趋势成分（长期方向）、季节性成分（周期波动）和残差成分（随机噪声）。
</figcaption>
</figure>

{% block type="definition" title="定义 8.3　时间序列的经典分解" %}
时间序列 $$\{x_t\}$$ 可分解为三个基本成分：

**加法模型**（Additive Model）：

$$
x_t = T_t + S_t + R_t \tag{8.3}
$$

**乘法模型**（Multiplicative Model）：

$$
x_t = T_t \cdot S_t \cdot R_t \tag{8.4}
$$

其中：
- $$T_t$$（**趋势**，Trend）：反映时间序列的长期方向——持续上升、下降或保持不变的趋势；
- $$S_t$$（**季节性**，Seasonality）：以固定周期 $$m$$ 重复出现的规律性波动，满足 $$S_{t+m} = S_t$$ 且 $$\sum_{j=1}^{m} S_j = 0$$（加法模型）或 $$\prod_{j=1}^{m} S_j = 1$$（乘法模型）；
- $$R_t$$（**残差**，Residual）：去除趋势和季节性后的随机波动——如果分解充分，残差应近似白噪声。

当季节性波动的幅度不随时间变化时，使用加法模型；当波动幅度与趋势成正比时（如股票波动率随价格增加），使用乘法模型。乘法模型可通过对数变换转化为加法模型：$$\log x_t = \log T_t + \log S_t + \log R_t$$。
{% endblock %}

### 8.1.2.2 经典分解方法

经典分解（Classical Decomposition）使用移动平均提取趋势，再通过周期平均提取季节性：

**步骤 1（估计趋势）**：使用 $$m$$ 阶居中移动平均消除季节性波动：

$$
\hat{T}_t = \frac{1}{m} \sum_{j=-\lfloor m/2 \rfloor}^{\lfloor m/2 \rfloor} x_{t+j} \tag{8.5}
$$

当 $$m$$ 为偶数时（如月度数据的年周期 $$m = 12$$），使用 $$2 \times m$$ 移动平均以保持对称性。

**步骤 2（估计季节性）**：去趋势后按周期位置平均：

$$
\hat{S}_j = \frac{1}{N_j} \sum_{k: t_k \bmod m = j} (x_{t_k} - \hat{T}_{t_k}), \quad j = 1, \ldots, m \tag{8.6}
$$

其中 $$N_j$$ 为落在周期位置 $$j$$ 上的样本数，然后中心化使 $$\sum_j \hat{S}_j = 0$$。

**步骤 3（残差）**：$$\hat{R}_t = x_t - \hat{T}_t - \hat{S}_t$$。

{% hint style="warning" %}
⚠️ **易错点：经典分解的局限**

经典分解假设季节性模式 $$S_t$$ 在整个时间范围内固定不变——这在许多场景下不成立。例如，空调的普及使得夏季用电量的季节性模式逐年增强。更先进的 **STL 分解**（Seasonal and Trend decomposition using Loess，Cleveland 等，1990）允许季节性随时间缓慢变化，通过局部加权回归（Loess）迭代估计各成分，是当前推荐的标准方法。
{% endhint %}

分解的价值不仅在于可视化——它为后续建模提供了清晰的策略：趋势成分可用回归模型拟合（§8.6），季节性成分可用傅里叶基函数建模，残差成分用自回归模型捕捉短期依赖。先分解再分别建模，是时间序列分析中"分而治之"的经典策略。

## 8.1.3 平稳性

### 8.1.3.1 为什么需要平稳性

平稳性（Stationarity）是时间序列分析中最核心的概念——几乎所有经典的时间序列模型（AR、MA、ARMA、ARIMA）都建立在平稳性假设之上。直觉上，平稳性意味着时间序列的统计性质"不随时间改变"——它没有趋势、没有周期、均值和方差保持恒定。

为什么平稳性如此重要？回到前面提到的根本困难：我们只有**一条**样本路径，却要估计随机过程的统计性质。如果过程是平稳的，那么**时间平均可以替代集成平均**——我们可以用一条足够长的路径来可靠地估计均值、方差和自相关。这一性质被称为**遍历性**（Ergodicity），是统计推断的基石。

{% block type="definition" title="定义 8.4　严平稳（Strictly Stationary）" %}
随机过程 $$\{X_t\}$$ 是**严平稳**的，若对任意正整数 $$k$$、任意时刻 $$t_1, t_2, \ldots, t_k$$ 和任意时移 $$h$$，联合分布保持不变：

$$
F_{X_{t_1}, X_{t_2}, \ldots, X_{t_k}}(x_1, x_2, \ldots, x_k) = F_{X_{t_1+h}, X_{t_2+h}, \ldots, X_{t_k+h}}(x_1, x_2, \ldots, x_k) \tag{8.7}
$$

即所有有限维联合分布仅取决于时间间隔，而不取决于绝对时间位置。
{% endblock %}

严平稳的要求过于严格——它要求**全部**有限维分布不变，在实际中难以验证。因此，统计学中更常用一个较弱的条件。

{% block type="definition" title="定义 8.5　宽平稳（Wide-Sense Stationary / Weakly Stationary）" %}
随机过程 $$\{X_t\}$$ 是**宽平稳**的（也称弱平稳、二阶平稳），若满足以下三个条件：

**（1）均值恒定**：

$$
E[X_t] = \mu, \quad \forall t \tag{8.8}
$$

**（2）方差有限且恒定**：

$$
\text{Var}(X_t) = \sigma^2 < \infty, \quad \forall t \tag{8.9}
$$

**（3）自协方差仅依赖于时间差**：

$$
\text{Cov}(X_t, X_{t+h}) = \gamma(h), \quad \forall t \tag{8.10}
$$

其中 $$\gamma(h)$$ 称为**自协方差函数**（Autocovariance Function），仅为滞后阶数 $$h$$ 的函数，不依赖于时间 $$t$$。
{% endblock %}

宽平稳只约束一阶矩（均值）和二阶矩（自协方差），不对高阶矩或完整分布做要求。对于高斯过程（所有有限维分布都是多元正态），宽平稳等价于严平稳——因为正态分布完全由均值和协方差确定。

{% hint style="think" %}
🤔 **思考：非平稳 ≠ 不可分析**

大量真实时间序列是非平稳的——带有明显的趋势或季节性。这并不意味着无法分析，而是需要先通过**差分**或**分解**将其转化为（近似）平稳序列。ARIMA 模型（§8.6）中的"I"（Integrated）正是指这种差分操作：一阶差分 $$\nabla x_t = x_t - x_{t-1}$$ 消除线性趋势，二阶差分 $$\nabla^2 x_t$$ 消除二次趋势，季节差分 $$\nabla_m x_t = x_t - x_{t-m}$$ 消除周期为 $$m$$ 的季节性。
{% endhint %}

### 8.1.3.2 Wold 分解定理

平稳时间序列具有一个深刻的结构性质：任何宽平稳过程都可以分解为"可预测"与"不可预测"两部分。

{% block type="theorem" title="定理 8.1　Wold 分解定理（Wold Decomposition Theorem, 1938）" %}
任意零均值的宽平稳过程 $$\{X_t\}$$ 可以唯一地分解为：

$$
X_t = \sum_{j=0}^{\infty} \psi_j \varepsilon_{t-j} + V_t \tag{8.11}
$$

其中：
- $$\{\varepsilon_t\}$$ 是白噪声过程（均值为 0，方差为 $$\sigma_\varepsilon^2$$，不同时刻不相关），$$\psi_0 = 1$$，$$\sum_{j=0}^{\infty} \psi_j^2 < \infty$$；
- $$V_t$$ 是**确定性成分**——可以由 $$\{X_s : s < t\}$$ 完美预测的部分；
- 两个成分相互正交：$$\text{Cov}(\varepsilon_t, V_s) = 0, \; \forall t, s$$。
{% endblock %}

{% hint style="proof" %}
**证明思路**。定义 $$\mathcal{H}_t = \overline{\text{span}}\{X_s : s \leq t\}$$ 为由过去观测张成的 Hilbert 空间。令 $$\hat{X}_t = \text{Proj}_{\mathcal{H}_{t-1}}(X_t)$$ 为 $$X_t$$ 在 $$\mathcal{H}_{t-1}$$ 上的投影（即最优线性预测），定义新息（innovation）：

$$
\varepsilon_t = X_t - \hat{X}_t \tag{8.12}
$$

由投影的正交性，$$\varepsilon_t \perp \mathcal{H}_{t-1}$$，即 $$\text{Cov}(\varepsilon_t, X_s) = 0$$ 对所有 $$s < t$$。特别地，$$\text{Cov}(\varepsilon_t, \varepsilon_s) = 0$$ 对 $$t \neq s$$——新息序列是白噪声。

由于 $$\{X_t\}$$ 宽平稳，$$\text{Var}(\varepsilon_t) = \sigma_\varepsilon^2$$ 恒定。反复迭代：

$$
X_t = \varepsilon_t + \psi_1 \varepsilon_{t-1} + \psi_2 \varepsilon_{t-2} + \cdots + V_t \tag{8.13}
$$

其中 $$\psi_j$$ 系数由递归投影确定，$$V_t$$ 是当 $$j \to \infty$$ 时的极限残余（确定性成分）。$$\sum \psi_j^2 < \infty$$ 由 $$\text{Var}(X_t) < \infty$$ 保证。 $$\square$$
{% endhint %}

Wold 分解定理的意义在于：它为时间序列建模提供了**理论正当性**。ARMA 模型（§8.6）本质上就是对 Wold 分解中 $$\sum \psi_j \varepsilon_{t-j}$$ 部分的**有限参数化近似**——用 AR 和 MA 多项式的比值来逼近无穷级数 $$\{\psi_j\}$$。

## 8.1.4 自相关分析

### 8.1.4.1 自相关函数（ACF）

自相关函数是分析时间序列内部依赖结构的基本工具。它度量的是序列与自身滞后版本之间的相关性。

{% block type="definition" title="定义 8.6　自相关函数（Autocorrelation Function, ACF）" %}
宽平稳过程 $$\{X_t\}$$ 的**自协方差函数**和**自相关函数**分别定义为：

$$
\gamma(h) = \text{Cov}(X_t, X_{t+h}) = E[(X_t - \mu)(X_{t+h} - \mu)] \tag{8.14}
$$

$$
\rho(h) = \frac{\gamma(h)}{\gamma(0)} = \frac{\text{Cov}(X_t, X_{t+h})}{\text{Var}(X_t)} \tag{8.15}
$$

其中 $$\rho(h) \in [-1, 1]$$，$$\rho(0) = 1$$，$$\rho(h) = \rho(-h)$$（对称性）。

给定长度为 $$T$$ 的观测序列 $$\{x_1, \ldots, x_T\}$$，**样本自相关函数**为：

$$
\hat{\rho}(h) = \frac{\sum_{t=1}^{T-h}(x_t - \bar{x})(x_{t+h} - \bar{x})}{\sum_{t=1}^{T}(x_t - \bar{x})^2}, \quad h = 0, 1, \ldots \tag{8.16}
$$
{% endblock %}

ACF 图（Correlogram）是时间序列分析中最重要的诊断工具。其典型模式包括：

- **白噪声**：所有滞后的 ACF 接近 0——无时序结构；
- **短记忆过程**（如 AR(1)）：ACF 指数衰减——远处的滞后相关性迅速消失；
- **季节性**：ACF 在滞后 $$m, 2m, 3m, \ldots$$ 处出现周期性峰值。

### 8.1.4.2 偏自相关函数（PACF）

ACF 度量的是 $$X_t$$ 与 $$X_{t+h}$$ 的**总相关**——它包含了通过中间变量 $$X_{t+1}, \ldots, X_{t+h-1}$$ 传递的间接相关。偏自相关函数去除了这种间接效应，只保留**直接相关**。

{% block type="definition" title="定义 8.7　偏自相关函数（Partial Autocorrelation Function, PACF）" %}
宽平稳过程 $$\{X_t\}$$ 在滞后 $$h$$ 处的**偏自相关**定义为：

$$
\phi(h) = \text{Corr}(X_t - \hat{X}_t^{(h)}, \; X_{t+h} - \hat{X}_{t+h}^{(h)}) \tag{8.17}
$$

其中 $$\hat{X}_t^{(h)}$$ 和 $$\hat{X}_{t+h}^{(h)}$$ 分别是 $$X_t$$ 和 $$X_{t+h}$$ 对中间变量 $$\{X_{t+1}, \ldots, X_{t+h-1}\}$$ 的最优线性预测。

等价地，$$\phi(h)$$ 是回归模型

$$
X_t = \alpha_1 X_{t-1} + \alpha_2 X_{t-2} + \cdots + \alpha_h X_{t-h} + \varepsilon_t \tag{8.18}
$$

中最后一个系数 $$\alpha_h$$ 的值。
{% endblock %}

PACF 的关键应用在于**AR 模型阶数的确定**：对于 AR($$p$$) 过程，PACF 在滞后 $$p$$ 之后截断为 0（即 $$\phi(h) = 0, \; \forall h > p$$），而 ACF 则呈指数衰减。类似地，MA($$q$$) 过程的 ACF 在滞后 $$q$$ 之后截断，而 PACF 呈指数衰减。这一"对偶"关系是 Box-Jenkins 方法论中模型识别的基础，将在 §8.6 中详细讨论。

### 8.1.4.3 白噪声检验

{% block type="definition" title="定义 8.8　白噪声过程（White Noise）" %}
随机过程 $$\{\varepsilon_t\}$$ 是**白噪声**，若满足：

$$
E[\varepsilon_t] = 0, \quad \text{Var}(\varepsilon_t) = \sigma^2, \quad \text{Cov}(\varepsilon_t, \varepsilon_s) = 0 \;\; (t \neq s) \tag{8.19}
$$

白噪声是"完全不可预测"的序列——知道过去的值不能帮助预测未来。它是 Wold 分解（定理 8.1）中的基本构建块，也是模型充分性的检验标准：如果模型拟合后的残差不是白噪声，说明模型未能充分捕捉数据中的结构。
{% endblock %}

判断一个序列是否为白噪声的标准方法是 **Ljung-Box 检验**：

$$
Q(H) = T(T+2) \sum_{h=1}^{H} \frac{\hat{\rho}(h)^2}{T-h} \tag{8.20}
$$

在白噪声假设下，$$Q(H) \sim \chi^2(H)$$。若 $$Q(H)$$ 显著大于 $$\chi^2(H)$$ 的临界值（p 值 < 0.05），则拒绝白噪声假设——序列中存在可建模的时序结构。

## 8.1.5 平稳性检验

### 8.1.5.1 ADF 检验

在实际分析中，需要一个统计检验来判断序列是否平稳。最常用的是 **ADF 检验**（Augmented Dickey-Fuller Test）。

{% block type="theorem" title="定理 8.2　ADF 检验（Augmented Dickey-Fuller Test）" %}
考虑带漂移项和趋势项的自回归模型：

$$
\Delta x_t = \alpha + \beta t + \delta x_{t-1} + \sum_{j=1}^{p} \phi_j \Delta x_{t-j} + \varepsilon_t \tag{8.21}
$$

其中 $$\Delta x_t = x_t - x_{t-1}$$ 为一阶差分。

**零假设** $$H_0: \delta = 0$$（序列存在单位根，即非平稳）。

**备择假设** $$H_1: \delta < 0$$（序列是平稳的）。

在 $$H_0$$ 下，检验统计量 $$\hat{\delta} / \text{SE}(\hat{\delta})$$ 不服从标准 $$t$$ 分布，而是服从 **Dickey-Fuller 分布**——其临界值由 Dickey 和 Fuller（1979）通过蒙特卡洛模拟制表给出。当检验统计量小于临界值（p 值 < 显著性水平）时，拒绝 $$H_0$$，认为序列是平稳的。
{% endblock %}

{% hint style="warning" %}
⚠️ **易错点：ADF 检验的零假设方向**

ADF 检验的零假设是"**存在单位根**"（非平稳），备择假设是"平稳"。这与多数假设检验的习惯相反——不拒绝 $$H_0$$ 意味着我们**没有证据**认为序列是平稳的，但不代表序列一定非平稳（可能是检验功效不足）。当样本量小或序列接近单位根边界时，ADF 检验的功效较低，此时可辅以 KPSS 检验（零假设为"平稳"）进行交叉验证。
{% endhint %}

## 案例 8.1：气象时间序列的探索性分析

{% block type="case" title="案例 8.1　城市气温数据的时序特征分析" %}

### 业务背景

气象数据是时间序列分析的经典应用场景。城市气温记录同时包含长期趋势（气候变暖）、季节性（夏热冬冷）和随机波动（天气事件），是展示时间序列分析方法的理想数据。

### 数据集说明

使用合成的日均气温数据（模拟北方城市 5 年数据，1826 天），包含线性升温趋势、正弦年周期季节性和随机扰动。

### 完整代码

```python
# ============================================================
# 代码 8.1：时间序列探索性分析——成分分解与平稳性检验
# 数据集：合成城市日均气温（5年，1826天）
# 依赖库：numpy, pandas, matplotlib, statsmodels
# ============================================================
# --- 1. 生成合成气温数据 ---
import numpy as np
import pandas as pd
from statsmodels.tsa.seasonal import STL
from statsmodels.tsa.stattools import adfuller, acf, pacf
np.random.seed(42)
T = 1826  # 5年日数据
dates = pd.date_range("2020-01-01", periods=T, freq="D")
# 趋势：每年升高 0.03°C
trend = np.linspace(0, 0.15, T)
# 季节性：年周期正弦波（振幅15°C，基准温度12°C）
seasonal = 15 * np.sin(2 * np.pi * np.arange(T) / 365.25 - np.pi / 2)
# 随机波动
noise = np.random.normal(0, 3, T)
# 组合
temp = 12 + trend + seasonal + noise
ts = pd.Series(temp, index=dates, name="日均气温(°C)")
# --- 2. STL 分解 ---
stl = STL(ts, period=365, robust=True)
result = stl.fit()
trend_comp = result.trend
seasonal_comp = result.seasonal
residual_comp = result.resid
print("=== STL 分解统计 ===")
print(f"趋势范围:   [{trend_comp.min():.2f}, {trend_comp.max():.2f}]°C")
print(f"季节性振幅: {(seasonal_comp.max() - seasonal_comp.min()):.2f}°C")
print(f"残差标准差: {residual_comp.std():.2f}°C")
# --- 3. 平稳性检验 ---
print("\n=== ADF 检验 ===")
for name, series in [("原始序列", ts),
                      ("去趋势+去季节", residual_comp.dropna())]:
    stat, pval, lags, nobs, crit, _ = adfuller(series.dropna(), maxlag=30)
    print(f"\n{name}:")
    print(f"  ADF 统计量 = {stat:.4f}")
    print(f"  p 值       = {pval:.6f}")
    print(f"  使用滞后数 = {lags}")
    for level, val in crit.items():
        print(f"  临界值({level}) = {val:.4f}")
    print(f"  结论: {'平稳 ✓' if pval < 0.05 else '非平稳 ✗'}")
# --- 4. ACF/PACF 分析 ---
nlags = 60
acf_vals = acf(residual_comp.dropna(), nlags=nlags)
pacf_vals = pacf(residual_comp.dropna(), nlags=nlags, method="ywm")
print(f"\n=== 残差自相关分析 ===")
print(f"ACF(1)  = {acf_vals[1]:.4f}")
print(f"ACF(7)  = {acf_vals[7]:.4f}")
print(f"PACF(1) = {pacf_vals[1]:.4f}")
print(f"PACF(2) = {pacf_vals[2]:.4f}")
# --- 5. 差分对比 ---
diff1 = ts.diff().dropna()
stat_d, pval_d, _, _, _, _ = adfuller(diff1, maxlag=30)
print(f"\n一阶差分: ADF={stat_d:.4f}, p={pval_d:.6f} → {'平稳 ✓' if pval_d < 0.05 else '非平稳 ✗'}")
```

### 结果分析

1. **STL 分解**准确分离了三个成分：趋势成分呈线性上升（0.15°C/5年），季节性成分的年振幅约 30°C（与设定一致），残差标准差约 3°C（接近设定的噪声水平）。
2. **ADF 检验**：原始序列因包含趋势和季节性而不拒绝单位根假设（p > 0.05，非平稳）；去趋势去季节后的残差 p 值接近 0（强平稳），证实分解充分。
3. **ACF/PACF 分析**：残差的 ACF(1) 为正的小值（天气的短期持续性），PACF 在滞后 1–2 后迅速衰减——暗示残差可用 AR(1) 或 AR(2) 模型建模。
{% endblock %}

## 本节小结

| 关键词 | 核心概念 |
|--------|---------|
| 时间序列（定义 8.1） | 随机过程 $$\{X_t\}$$ 的一条样本路径，具有有序性和依赖性 |
| 分类体系（定义 8.2） | 单/多变量、等/不等间隔、连续/离散值 |
| 三成分分解（定义 8.3） | $$x_t = T_t + S_t + R_t$$（加法）或 $$x_t = T_t \cdot S_t \cdot R_t$$（乘法） |
| 严平稳（定义 8.4） | 所有有限维联合分布不随时间平移改变 |
| 宽平稳（定义 8.5） | 均值恒定、方差有限且恒定、自协方差仅依赖时间差 |
| Wold 分解（定理 8.1） | $$X_t = \sum \psi_j \varepsilon_{t-j} + V_t$$，平稳过程 = MA(∞) + 确定性 |
| ACF / PACF（定义 8.6–8.7） | 自相关与偏自相关——模型识别的核心工具 |
| ADF 检验（定理 8.2） | $$H_0$$: 单位根（非平稳），检验统计量服从 Dickey-Fuller 分布 |

本节建立了时间序列分析的数学基础：从形式化定义出发，通过成分分解理解数据的宏观结构，通过平稳性和自相关分析揭示数据的微观依赖。这些工具将贯穿本章始终——§8.2 的预处理需要保持平稳性，§8.3 的相似性度量依赖于时间对齐，§8.6 的 ARIMA 模型直接建立在 Wold 分解和 ACF/PACF 分析之上。
