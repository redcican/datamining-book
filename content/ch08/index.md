# 第八章 时序数据挖掘

前面各章的数据挖掘方法都隐含一个共同假设：样本独立同分布（i.i.d.）。然而，大量现实数据天然具有时间维度——股价、温度、心电信号等连续观测之间存在强时间依赖性。本章系统介绍时间序列数据挖掘的完整技术体系：§8.1 建立时间序列的形式化定义、平稳性理论和成分分解方法；§8.2 介绍缺失值处理、重采样、平滑和差分等预处理技术；§8.3 从锁步对齐的欧氏距离到弹性对齐的 DTW，再到 SAX 符号化距离；§8.4 介绍 Matrix Profile 的 Motif 发现和 Shapelet 提取；§8.5 将距离度量和模式发现应用于时间序列分类与聚类；§8.6 从 ARIMA 到 LSTM 和 Transformer，对比经典与深度学习预测方法；§8.7 讨论时间序列异常检测的特殊挑战——同一数值在不同时间点可能正常或异常。

## 本章学习目标

1. 掌握时间序列的形式化定义、严平稳与宽平稳的区别、ACF 和 PACF 的含义；
2. 理解时间序列预处理流程——缺失值处理、重采样、标准化与平滑；
3. 掌握欧氏距离、DTW 和 SAX 三种相似性度量方法的原理与适用场景；
4. 掌握 Matrix Profile 的 Motif 发现和 Shapelet 提取方法；
5. 能使用多种方法对时间序列进行分类与聚类；
6. 掌握 ARIMA 建模流程和 LSTM/Transformer 深度学习预测方法；
7. 理解时间序列异常检测中上下文异常和集体异常的特殊性。
