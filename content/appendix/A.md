### 数据集获取方式

```python
# scikit-learn 内置数据集（无需下载）
from sklearn.datasets import load_iris, load_breast_cancer, load_wine
data = load_iris()
X, y = data.data, data.target

# 通过 fetch 下载（首次运行需网络）
from sklearn.datasets import fetch_olivetti_faces, fetch_openml
faces = fetch_olivetti_faces()

# Kaggle CLI 下载（需注册并配置 API Token）
# pip install kaggle
# export KAGGLE_USERNAME=your_username
# export KAGGLE_KEY=your_key
# kaggle datasets download -d mlg-ulb/creditcardfraud

# UCI 数据集可直接通过 URL 下载
import pandas as pd
url = "https://archive.ics.uci.edu/ml/machine-learning-databases/..."
df = pd.read_csv(url)

# PyTorch 内置数据集
import torchvision
mnist = torchvision.datasets.MNIST(root="./data", download=True)
cifar = torchvision.datasets.CIFAR10(root="./data", download=True)
```

💡 本书代码仓库中的 `code/chXX/_load_*.py` 文件封装了每个案例的数据下载、缓存和预处理逻辑，可直接调用。首次运行需要网络连接，数据缓存于 `~/.datamining_data/`。
{% endhint %}

## A.3 数据集选择指南

### 按学习目标选择

| 学习目标 | 推荐数据集 | 理由 | 预期耗时 |
|---------|-----------|------|---------|
| 入门分类算法 | Iris, Breast Cancer | 样本少、特征少，几秒完成训练 | 1–2h |
| 理解数据预处理 | Adult Income, Titanic | 混合类型、缺失值、需要特征工程 | 3–5h |
| 处理类不平衡 | Credit Card Fraud | 0.172% 正样本，强迫思考评估指标 | 4–6h |
| 学习聚类方法 | Iris, Wine | 有真实标签可对比聚类结果 | 2–3h |
| 实践推荐系统 | MovieLens (100K) | 规模适中、社区活跃、基准丰富 | 4–6h |
| 时间序列分析 | 北京 PM2.5 | 5 年小时级数据，季节性明显 | 5–8h |
| 文本挖掘入门 | 20 Newsgroups (sklearn) | 英文文本分类经典 | 3–5h |
| 图挖掘入门 | Karate Club | 34 节点，手动可验证算法正确性 | 1–2h |
| 深度学习入门 | MNIST → Fashion-MNIST → CIFAR-10 | 渐进式难度提升 | 各 2–4h |
| 端到端项目 | Telecom Churn, AI4I | 接近真实业务场景，特征可解释 | 8–12h |

### 按数据规模选择

| 规模 | 样本量 | 推荐 | 适用 |
|------|--------|------|------|
| 玩具级 | < 1K | Iris, Karate Club | 算法验证、教学演示 |
| 小型 | 1K–10K | Wine, AI4I, Breast Cancer | 快速实验、方法对比 |
| 中型 | 10K–100K | Adult, PM2.5, MovieLens 100K | 完整流程练习 |
| 大型 | 100K–1M | Credit Card Fraud, KDD Cup | 效率和扩展性测试 |
| 超大型 | > 1M | MovieLens 25M, ImageNet | 分布式计算、工程优化 |

## A.4 数据使用的许可与伦理

### 常见许可类型

| 许可 | 允许商用 | 要求署名 | 允许修改 | 典型数据集 |
|------|---------|---------|---------|-----------|
| CC0（公共领域） | ✅ | — | ✅ | 部分 Kaggle 数据集 |
| CC BY 4.0 | ✅ | ✅ | ✅ | 多数 UCI 数据集 |
| CC BY-NC | ❌ | ✅ | ✅ | 部分学术数据集 |
| 自定义学术协议 | 视具体条款 | 通常要求 | 视具体条款 | MIMIC, ImageNet |

### 使用注意事项

| 维度 | 要点 |
|------|------|
| **引用要求** | 学术使用时引用原始论文或提供者。例如 UCI 数据集应引用 Dua & Graff (2019) |
| **隐私保护** | 注意数据是否包含可识别个人信息 (PII)。Credit Card Fraud 数据已做 PCA 匿名化 |
| **偏差意识** | Adult Income 包含性别/种族偏差——模型可能学到历史歧视，需在实验报告中讨论 |
| **再分发限制** | 部分数据集禁止再分发原始数据（如 MIMIC），不可上传至 GitHub |
| **时效性** | 注意数据采集时间。2010 年的 PM2.5 数据不代表 2025 年的空气质量 |
| **代表性** | 数据集可能只代表特定地区/人群。Titanic 数据不能推广到现代船舶安全 |

{% hint style="warning" %}
⚠️ **数据集不等于真实世界**

本书使用的数据集经过清洗和标准化，与真实业务数据存在显著差距。真实数据更脏、更不平衡、更多缺失、标签更嘈杂、特征更多更冗余。将课程实践的经验直接迁移到生产环境时，应有这一心理预期，并参照 §14.3 的数据质量评估方法论。
{% endhint %}
